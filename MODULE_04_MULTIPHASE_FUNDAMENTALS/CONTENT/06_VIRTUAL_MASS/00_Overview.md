# แรงมวลเสมือน (Virtual Mass Forces)

## 1. บทนำ (Introduction)

**แรงมวลเสมือน (Virtual Mass Force)** หรือ **แรงมวลเพิ่ม (Added Mass Force)** คือแรงต้านอุทกพลศาสตร์ที่เกิดขึ้นเมื่อวัตถุเคลื่อนที่ด้วยความเร่งผ่านของไหล วัตถุนั้นไม่เพียงแต่ต้องเร่งมวลของตัวเอง แต่ยังต้องเร่งปริมาตรของของไหลที่อยู่รายล้อมไปด้วย ทำให้วัตถุประพฤติตัวเสมือนว่ามี "มวลเพิ่มขึ้น"

### ความสำคัญในระบบหลายเฟส:
- จำเป็นอย่างยิ่งสำหรับระบบที่มีความต่างของความหนาแน่นสูง (เช่น ฟองแก๊สในของเหลว)
- มีผลต่อการทำนายความเร็วในสภาวะไม่คงตัว (Transient behavior)
- ส่งผลต่อพลวัตของฟองอากาศ (Bubble dynamics) และวิถีการเคลื่อนที่ของอนุภาค

---

## 2. กลไกทางกายภาพ (Physical Mechanism)

เมื่ออนุภาคเร่งความเร็ว ของไหลรอบๆ จะต้องถูกขับออกไปและเร่งความเร็วตามไปด้วย กระบวนการนี้ต้องใช้พลังงานและสร้างแรงต้านกลับมาที่อนุภาค

**แรงมวลเสมือนแปรผันตาม:**
- ความเร่งสัมพัทธ์ระหว่างเฟส (Relative acceleration)
- ความหนาแน่นของเฟสต่อเนื่อง (Continuous phase density)
- ปริมาตรของเฟสกระจาย (Dispersed phase volume)

---

## 3. สมการหลัก (Governing Equation)

ในแบบจำลอง Eulerian-Eulerian แรงมวลเสมือนต่อหน่วยปริมาตรสำหรับเฟส $i$ ในเฟส $j$ คือ:

$$\mathbf{F}_{vm} = C_{vm} \rho_j \alpha_i \left(\frac{D\mathbf{u}_j}{Dt} - \frac{D\mathbf{u}_i}{Dt}\right) \tag{6.1}$$

**นิยามตัวแปร:**
- $C_{vm}$: สัมประสิทธิ์มวลเสมือน (Virtual Mass Coefficient)
- $ho_j$: ความหนาแน่นของเฟสต่อเนื่อง
- $\alpha_i$: สัดส่วนปริมาตรของเฟสกระจาย
- $\frac{D\mathbf{u}}{Dt}$: อนุพันธ์รวมตามการเคลื่อนที่ (Material Derivative) ซึ่งแสดงถึงความเร่ง

---

## 4. สัมประสิทธิ์มวลเสมือน ($C_{vm}$)

ค่า $C_{vm}$ ขึ้นอยู่กับรูปร่างของอนุภาคและระบอบการไหล:

| รูปร่างอนุภาค | $C_{vm}$ | หมายเหตุ |
|--------------|----------|---------|
| **ทรงกลมสมบูรณ์** | 0.5 | ได้จากทฤษฎีศักย์การไหล (Potential Flow) |
| **ทรงกระบอก** | 1.0 | สำหรับการไหลตั้งฉากกับแกน |
| **ทรงรี (Ellipsoid)** | แปรผัน | ขึ้นอยู่กับอัตราส่วนกว้างยาว (Aspect Ratio) |

---

## 5. การนำไปใช้ใน OpenFOAM

ใน OpenFOAM, แรงมวลเสมือนมักถูกจัดการแบบ **Implicit** เพื่อความเสถียรในการคำนวณ โดยตั้งค่าใน `constant/phaseProperties`:

```openfoam
virtualMass
(
    (air in water)
    {
        type            constantCoefficient;
        Cvm             0.5;
    }
);
```

**ข้อควรระวัง:** การละเลยแรงมวลเสมือนในระบบ Gas-Liquid อาจทำให้เกิดความผิดพลาดในการทำนายความดันตก (Pressure drop) และความเร็วในการลอยตัวของฟองอากาศในช่วงเริ่มต้นของการไหล
