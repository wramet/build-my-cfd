# ระบบก๊าซ-ของแข็ง (Gas-Solid Systems)

## 1. บทนำ (Introduction)

ระบบก๊าซ-ของแข็งพบได้บ่อยในกระบวนการทางอุตสาหกรรม เช่น เตาเผาฟลูอิไดซ์เบด (Fluidized Beds) และการขนส่งผงทางท่อ พฤติกรรมหลักถูกกำหนดโดยการชนกันของอนุภาค (Particle collisions) และแรงต้านทานจากก๊าซ

---

## 2. การจำแนกตามขนาดอนุภาค (Particle Size Classification)

### 2.1 อนุภาคละเอียด ($d_p < 100 \mu m$)

อนุภาคละเอียดเป็นขอบเขตของอนุภาคขนาดเล็กที่สุดในระบบก๊าซ-ของแข็ง โดยทั่วไปมีลักษณะเฉพาะคือเส้นผ่านศูนย์กลางของอนุภาคน้อยกว่า 100 ไมครอนเมตร

> [!INFO] คุณลักษณะทางกายภาพของอนุภาคละเอียด
> - **จำนวนเรย์โนลด์ต่ำ**: จำนวนเรย์โนลด์ของอนุภาค $\text{Re}_p = \frac{\rho_g u_g d_p}{\mu_g}$ โดยทั่วไปต่ำกว่า 1 ซึ่งบ่งชี้การไหลในระบอบ Stokes
> - **เอฟเฟกต์ยึดเหนี่ยวที่แข็งแกร่ง**: แรง Van der Waals และการสร้างสะพานความชื้นสร้างความยึดเหนี่ยวระหว่างอนุภาค
> - **แรงไฟฟ้าสถิตที่สำคัญ**: ปฏิสัมพันธ์ไฟฟ้าสถิตมีอิทธิพลต่อพฤติกรรมการไหลและการตกตะกอน
> - **เอฟเฟกต์การเคลื่อนที่แบบ Brownian**: การเคลื่อนที่แบบสุ่มสามารถมีอิทธิพลต่อเส้นทางการเคลื่อนที่ของอนุภาคขนาดเล็กกว่าไมครอน

#### แบบจำลองที่แนะนำสำหรับอนุภาคละเอียด

```cpp
// แบบจำลองแรงลากสำหรับอนุภาคละเอียด
dragModel    WenYu;

// การจำลองความปั่นป่วน
turbulenceModel    dispersedPhase;

// แรงระหว่างอนุภาค
solidContactModel    JohnsonJackson;

// การกระจายขนาดอนุภาค
diameterModel    RosinRammler;
```

#### การพิสูจน์ความถูกต้องของแบบจำลอง

- **แบบจำลองแรงลาก Wen-Yu**: กำหนดขึ้นโดยเฉพาะสำหรับการไหลที่มีจำนวนเรย์โนลด์ต่ำ
  $$C_D = \frac{24}{\text{Re}_p}(1 + 0.15\text{Re}_p^{0.687}) \quad \text{สำหรับ } \text{Re}_p < 1000$$

- **ความปั่นป่วนของเฟสกระจาย**: พิจารณาการสร้างความปั่นป่วนโดยการเคลื่อนที่ของอนุภาค

- **แบบจำลองการสัมผัสของของแข็ง Johnson-Jackson**: รวมสัมประสิทธิ์แรงเสียดทานอนุภาค-ผนัง $\mu_w$ และสัมประสิทธิ์การกระดอนของอนุภาค-ผนัง $e_w$

- **การกระจาย Rosin-Rammler**: จำลองการกระจายขนาดอนุภาคที่สมจริง
  $$F(d) = 1 - \exp\left[-\left(\frac{d}{d_{63.2}}\right)^n\right]$$

### 2.2 อนุภาคหยาบ ($d_p > 1$ mm)

อนุภาคหยาบเป็นขอบเขตของอนุภาคขนาดใหญ่ที่สุดซึ่งเอฟเฟกต์เฉื่อยมีอำนาจเหนือกว่าแรงพื้นผิว

> [!INFO] คุณลักษณะทางกายภาพของอนุภาคหยาบ
> - **จำนวนเรย์โนลด์สูง**: จำนวนเรย์โนลด์ของอนุภาคโดยทั่วไปเกิน 1000 ซึ่งวางอนุภาคไว้ในระบอบแรงลากของ Newton
> - **เอฟเฟกต์เฉื่อยมีอำนาจเหนือกว่า**: โมเมนตัมของอนุภาค $\mathbf{p}_p = m_p \mathbf{u}_p$ ควบคุมเส้นทางการเคลื่อนที่
> - **เอฟเฟกต์ยึดเหนี่ยวเล็กน้อย**: แรงโน้มถ่วงและแรงเฉื่อยมากกว่าแรง Van der Waals และแรงไฟฟ้าสถิตอย่างมีนัยสำคัญ
> - **การตกตะกอนโดยโน้มถ่วงอย่างมีนัยสำคัญ**: ความเร็วปลาย $u_t = \sqrt{\frac{4gdp_p(\rho_p - \rho_g)}{3C_D\rho_g}}$ กลายเป็นค่าที่สำคัญ

#### แบบจำลองที่แนะนำสำหรับอนุภาคหยาบ

```cpp
// แบบจำลองแรงลากสำหรับอนุภาคหยาบ
dragModel    MorsiAlexander;

// การจำลองความปั่นป่วน
turbulenceModel    mixture;

// เอฟเฟกต์มวลเสมือน
virtualMassModel    negligible;

// ขนาดอนุภาค
diameterModel    constant;
```

#### การพิสูจน์ความถูกต้องของแบบจำลอง

- **แบบจำลองแรงลาก Morsi-Alexander**: ให้สหสัมพันธ์สัมประสิทธิ์แรงลากเป็นชิ้นส่วนที่ถูกต้องในช่วงจำนวนเรย์โนลด์ที่กว้าง

- **แบบจำลองความปั่นป่วนของส่วนผสม**: ถือว่าส่วนผสมก๊าซ-อนุภาคเป็นของไหลมีประสิทธิภาพเดี่ยวด้วยความหนาแน่นที่ดัดแปลง:
  $$\rho_{\text{mix}} = \alpha_g\rho_g + \alpha_s\rho_s$$

- **มวลเสมือนเล็กน้อย**: สัมประสิทธิ์มวลเพิ่ม $C_{VM} \ll 1$ สำหรับอัตราส่วนความหนาแน่นสูง $\frac{\rho_p}{\rho_g} \gg 1$

- **เส้นผ่านศูนย์กลางคงที่**: เหมาะสำหรับระบบ monodisperse ที่การแปรผันของขนาดน้อยที่สุด

## 3. การจำแนกตามความเข้มข้น (Regime Classification)

### 3.1 การไหลแบบเจือจาง (Dilute Flow, $\alpha_s < 0.1$)

อนุภาคอยู่ห่างกันมาก ปฏิสัมพันธ์ระหว่างอนุภาคน้อย

- **Drag:** Wen-Yu Model (ออกแบบมาสำหรับก๊าซ-ของแข็งเจือจาง)
- **Turbulence:** การเคลื่อนที่ของอนุภาคสามารถเพิ่มหรือลดความปั่นป่วนของก๊าซได้

### 3.2 การไหลแบบหนาแน่น (Dense Flow, $\alpha_s > 0.1$)

การชนกันของอนุภาคมีความสำคัญมาก ต้องใช้ **Kinetic Theory of Granular Flow (KTGF)** เพื่อคำนวณความเครียดของของแข็ง (Solid stress)

- **Drag:** Gidaspow Model (รวม Wen-Yu และ Ergun เข้าด้วยกัน)
- **Solid Pressure:** คำนวณจาก Granular Temperature ($\Theta$)

---

## 4. ขอบเขตการไหล (Flow Regimes)

### 4.1 เตียงบรรจุ (Packed Bed)

เตียงบรรจุเป็นขอบเขตการไหลก๊าซ-ของแข็งที่หนาแน่นที่สุดซึ่งอนุภาคก่อตัวเป็นโครงสร้างที่ซึมผ่านได้ค่อนข้างคงที่ซึ่งก๊าซไหลผ่าน

> [!TIP] คุณลักษณะทางกายภาพของเตียงบรรจุ
> - **ส่วนประกอบของแข็งสูงมาก**: ส่วนประกอบปริมาตรของแข็ง $\alpha_s > 0.6$ สร้างช่องว่างเล็กน้อยสำหรับการไหลของก๊าซ
> - **การเคลื่อนที่ของอนุภาคเล็กน้อย**: อนุภาคได้รับการสั่นสะเทือนเล็กน้อยหรือการจัดเรียงใหม่เท่านั้น
> - **ระบอกการไหล Darcy**: การลดแรงดันถูกควบคุมโดยกฎของ Darcy:
>   $$\nabla p = -\frac{\mu}{K}\mathbf{u} - \frac{\rho C_F}{\sqrt{K}}|\mathbf{u}|\mathbf{u}$$

#### กลยุทธ์การเลือกแบบจำลองสำหรับเตียงบรรจุ

**แบบจำลองแรงลาก**: การปรับเปลี่ยนสมการ Ergun ขยายสหสัมพันธ์ Ergun มาตรฐาน:
$$\Delta p = \frac{150(1-\epsilon)^2\mu L u}{\epsilon^3 d_p^2} + \frac{1.75(1-\epsilon)\rho L u^2}{\epsilon^3 d_p}$$

**การจำลองความซึมผ่าน**: สหสัมพันธ์ Kozeny-Carman เชื่อมโยงความซึมผ่านกับขนาดอนุภาคและความพรุน:
$$K = \frac{\epsilon^3 d_p^2}{180(1-\epsilon)^2}$$

**การจำลองความปั่นป่วน**: โดยทั่วไปถูกละเลยเนื่องจากจำนวนเรย์โนลด์ต่ำในช่องว่างรูพรุน การสมมติฐานการไหลแบบลามินาร์ถูกต้องสำหรับ $\text{Re}_p < 10$

**การจำลองการถ่ายเทความร้อน**: การนำความร้อนของเมทริกซ์ของแข็งมีอำนาจเหนือกว่าด้วยสภาพนำความร้อนมีประสิทธิภาพ:
$$\kappa_{\text{eff}} = \epsilon\kappa_g + (1-\epsilon)\kappa_s$$

#### ข้อควรพิจารณาทางตัวเลข

> [!WARNING] ข้อควรพิจารณาสำคัญ
> - ใช้ความละเอียดของ mesh ละเอียดเพื่อจับฟิสิกส์ระดับรูพรุน
> - ใช้เงื่อนไขขอบเขตความดันมากกว่าการระบุความเร็ว
> - พิจารณาเอฟเฟกต์การขยายตัวทางความร้อนสำหรับการใช้งานอุณหภูมิสูง
> - รวมแบบจำลองปฏิกิริยาเคมีสำหรับกระบวนการเร่งปฏิกิริยา

### 4.2 เตียงไหล (Fluidized Bed)

เตียงไหลเป็นขอบเขตกลางที่ความเร็วของก๊าซเพียงพอที่จะระงับอนุภาค สร้างพฤติกรรมแบบไหลเหมือนของไหลที่มีพลวัต

> [!TIP] คุณลักษณะทางกายภาพของเตียงไหล
> - **ส่วนประกอบของแข็งปานกลาง**: ส่วนประกอบปริมาตรของแข็ง $0.3 < \alpha_s < 0.6$
> - **พฤติกรรมอนุภาคแบบพลวัต**: อนุภาคเผชิญกับการเคลื่อนที่อย่างต่อเนื่อง การผสม และรูปแบบการไหลเวียน
> - **การเชื่อมโยงเฟสที่แข็งแกร่ง**: การแลกเปลี่ยนโมเมนตัมระหว่างเฟสก๊าซและของแข็ง

#### พารามิเตอร์วิกฤตสำหรับเตียงไหล

- **ความเร็วการไหลต่ำสุด**:
  $$u_{mf} = \frac{(\rho_p - \rho_g)gd_p^2}{150\mu_g}\left(\frac{\epsilon_{mf}^3}{1-\epsilon_{mf}}\right)$$
  โดย $\epsilon_{mf}$ คือส่วนประกอบช่องว่างที่การไหลต่ำสุด

- **การก่อตัวของฟอง**: เมื่อ $u_g > u_{mf}$ ฟองก๊าซจะขึ้นผ่านเตียง

- **อุณหภูมิของเม็ด**: การวัดพลังงานจลน์คลื่นของอนุภาค:
  $$\Theta_p = \frac{1}{3}\langle \mathbf{c}' \cdot \mathbf{c}' \rangle$$

#### กลยุทธ์การเลือกแบบจำลองสำหรับเตียงไหล

**แบบจำลองแรงลาก**:
- Wen-Yu สำหรับการไหลแบบสม่ำเสมอ
- Syamlal-O'Brien สำหรับระบบที่มีพลวัตฟองที่สำคัญ

**การจำลองความปั่นป่วน**: แบบจำลองที่เชื่อมโยงสองทิศทางซึ่งความปั่นป่วนของก๊าซส่งผลต่ออนุภาคและการเคลื่อนที่ของอนุภาคปรับเปลี่ยนความปั่นป่วนของก๊าซ

**การจำลองความเครียดของแข็ง**: สมการอุณหภูมิของเม็ด:
$$\frac{3}{2}\rho_s\alpha_s\frac{D\Theta_p}{Dt} = (-p_s\mathbf{I} + \boldsymbol{\tau}_s):\nabla\mathbf{u}_s - \nabla\cdot\mathbf{q}_\theta - \gamma_\Theta$$

**การจำลองการถ่ายเทความร้อน**: การขนส่งแบบ Convective ด้วยการเคลื่อนที่ของอนุภาค:
$$\rho_sc_{p,s}\alpha_s\frac{DT_s}{Dt} = \nabla\cdot(\kappa_s\nabla T_s) + h_{gs}(T_g - T_s) + Q_{reactions}$$

#### คุณสมบัติขั้นสูง

- ใช้เงื่อนไขขอบเขตอุณหภูมิของเม็ดสำหรับปฏิสัมพันธ์ผนังที่สมจริง
- รวมแบบจำลองการยึดเหนี่ยวสำหรับอนุภาคละเอียดในเตียงไหล
- พิจารณาการเชื่อมโยงปฏิกิริยาเคมีสำหรับการไหลแบบเร่งปฏิกิริยา
- จำลองพลวัตฟองโดยใช้แนวทางสมดุลประชากร

### 4.3 การขนส่งด้วยระบบสุญญากาศ (Pneumatic Transport)

การขนส่งด้วยระบบสุญญากาศเป็นขอบเขตเฟสเจือจางที่อนุภาคถูกขนส่งโดยกระแสก๊าซความเร็วสูง

> [!INFO] คุณลักษณะทางกายภาพของการขนส่งด้วยระบบสุญญากาศ
> - **ส่วนประกอบของแข็งต่ำ**: ส่วนประกอบปริมาตรของแข็ง $\alpha_s < 0.1$ บ่งชี้การโหลดเจือจาง
> - **ความเร็วของก๊าซสูง**: ความเร็วของก๊าซผิวโดยทั่วไป $u_g > 10-20$ m/s
> - **การไหลของเฟสเจือจาง**: อนุภาคทำหน้าที่เป็นหน่วยแยกด้วยไดนามิกของก๊าซที่มีอำนาจเหนือกว่า

#### พารามิเตอร์การออกแบบ

- **ความเร็ว Saltation**: ความเร็วของก๊าซต่ำสุดเพื่อป้องกันการตกตะกอนของอนุภาค:
  $$u_{salt} = 4.1\sqrt{gd_p\left(\frac{\rho_p}{\rho_g} - 1\right)}$$

- **อัตราส่วนการโหลด**: อัตราส่วนอัตราการไหลมวล $\eta = \frac{\dot{m}_p}{\dot{m}_g}$ โดยทั่วไปอยู่ในช่วง 1-50

- **การไล่ระดับความดัน**: ถูกควบคุมโดยสมการ Darcy-Weisbach ที่ปรับเปลี่ยน:
  $$\frac{\mathrm{d}p}{\mathrm{d}x} = \frac{f_g\rho_g u_g^2}{2D} + \frac{f_p\rho_p u_p^2}{2D}$$

#### กลยุทธ์การเลือกแบบจำลอง

| แบบจำลอง | เหมาะสมสำหรับ | สมมติฐานหลัก |
|-----------|-------------|-------------|
| **Morsi-Alexander** | การไหลที่มีจำนวนเรย์โนลด์สูง ($\text{Re}_p > 1000$) | สัมประสิทธิ์แรงลาก $C_D \approx 0.44$ |
| **$k$-$\epsilon$ / $k$-$\omega$ SST** | ความปั่นป่วนของก๊าซ | การปรับเปลี่ยนความปั่นป่วนที่เกิดจากอนุภาค |
| **Lagrangian DPM/DEM** | การติดตามอนุภาคแยก | การจำลององค์ประกอบแยก |

#### การใช้งานทางตัวเลข

```cpp
// พารามิเตอร์การชนผนัง
restitutionCoeff    0.8;
frictionCoeff       0.3;

// การปรับเปลี่ยนความปั่นป่วน
turbulenceDispersion    on;
couplingFactor          0.5;

// ตัวเลือกการติดตามอนุภาค
trackingScheme          Euler;
maxCo                  0.5;
```

#### แนวทางการตรวจสอบ

- เปรียบเทียบการคาดการณ์ Eulerian-Eulerian กับการติดตามอนุภาค Lagrangian
- ตรวจสอบการคาดการณ์การลดแรงดันกับสหสัมพันธ์เชิงประจักษ์
- ตรวจสอบโปรไฟล์ความเร็วของอนุภาคกับการวัดการทดลอง
- ตรวจสอบการกระจายความเข้มข้นโดยใช้ข้อมูล tomography

#### การพิจารณาการทำงาน

> [!WARNING] ข้อควรพิจารณาด้านการดำเนินงาน
> - ตรวจสอบการตกตะกอนของอนุภาคในส่วนแนวนอน
> - พิจารณาเอฟเฟกต์ไฟฟ้าสถิตสำหรับการขนส่งผงละเอียด
> - ใช้แบบจำลองการกัดกร่อนสำหรับการชนของอนุภาคความเร็วสูง
> - ออกแบบสำหรับการสลายตัวของอนุภาคในสายขนส่งยาว

## 5. แบบจำลองที่แนะนำ (Recommended Models)

### 5.1 แรงฉุด Gidaspow Model

เป็นโมเดลที่เสถียรที่สุดสำหรับระบบที่ความเข้มข้นเปลี่ยนแปลงกว้าง:
- หาก $\alpha_g > 0.8$: ใช้ Wen-Yu
- หาก $\alpha_g \leq 0.8$: ใช้ Ergun Equation

### 5.2 ทฤษฎีจลน์ (KTGF)

ใช้ในการคำนวณความหนืดประสิทธิผลของอนุภาค:
$$ \mu_s = \mu_{s,coll} + \mu_{s,kin} + \mu_{s,fric} \tag{5.1} $$

## 6. การนำไปใช้ใน OpenFOAM

ตัวอย่างการตั้งค่าใน `phaseProperties` สำหรับระบบ Fluidized Bed:

```openfoam
solid
{
    type            solidPhase;
    kineticTheoryModel Gidaspow;

    kineticTheoryCoeffs
    {
        restitutionCoeff    0.9;
        frictionCoeff       0.05;
    }
}

phaseInteraction
{
    dragModel       Gidaspow;
}
```

### พารามิเตอร์วิกฤต:

> [!TIP] ในระบบ Gas-Solid ค่า **Restitution Coefficient ($e$)** มีผลอย่างมากต่อความสูงของเตียง (Bed expansion) โดยทั่วไปมีค่าระหว่าง 0.8 - 0.95

## 7. การเปรียบเทียบแบบจำลอง (Model Comparison)

| แบบจำลอง Drag | เหมาะสำหรับ | ข้อดี | ข้อเสีย |
|-----------------|-------------|---------|---------|
| **Wen-Yu** | การไหลเจือจาง ($\alpha_g > 0.8$) | แม่นยำสำหรับฟองก๊าซกระจาย | ไม่เหมาะกับการไหลหนาแน่น |
| **Ergun** | เตียงบรรจุ ($\alpha_g < 0.6$) | จับฟิสิกส์รูพรุนได้ดี | ไม่เหมาะกับการไหลพลวัต |
| **Gidaspow** | ทุกความเข้มข้น | ผสมผสานทั้งสองแบบจำลอง | การเปลี่ยนผ่านอาจแหว่ง |
| **Syamlal-O'Brien** | เตียงไหล | คำนึงถึงรูปร่างฟองก๊าซ | ต้องการปรับสอดคล้อง |

## 8. แนวทางการตรวจสอบ (Validation Approach)

### 8.1 ขั้นตอนการตรวจสอบ

1. **การตรวจสอบโค้ด**: ให้แน่ใจว่าการนำไปใช้เชิงตัวเลขถูกต้อง
2. **การตรวจสอบแบบจำลอง**: เปรียบเทียบการพยากรณ์กับข้อมูลทดลอง
3. **การประมาณความไม่แน่นอน**: ประเมินความไวต่อพารามิเตอร์แบบจำลอง
4. **การตรวจสอบข้าม**: ทดสอบกับชุดข้อมูลหลายชุด

### 8.2 เมตริกการตรวจสอบ

| ประเภทเมตริก | ตัวชี้วัด | เกณฑ์การยอมรับ |
|---------------|-------------|-----------------|
| **ปริมาณโดยรวม** | ความสูงของเตียง, การลดแรงดัน | ±5% ของข้อมูลทดลอง |
| **ปรากฏการณ์ในพื้นที่** | โปรไฟล์ความเข้มข้น, ความเร็ว | ความคลาดเคลื่อน RMS < 10% |
| **มาตรการทางสถิติ** | ความคลาดเคลื่อน RMS | R² > 0.85 |
| **ความสอดคล้องทางฟิสิกส์** | กฎการอนุรักษ์ | การละเมิด < 1% |

## 9. ข้อควรพิจารณาด้านประสิทธิภาพ (Computational Considerations)

### 9.1 การปรับขนาดต้นทุนการคำนวณ

| ประเภทแบบจำลอง | ต้นทุนสัมพัทธ์ | การใช้หน่วยความจำ | ประสิทธิภาพขนาน |
|-------------------|------------------|-------------------|-------------------|
| Dilute Flow | 1.0x | ต่ำ | ยอดเยี่ยม |
| Dense Flow | 2.0-3.0x | ปานกลาง | ดี |
| Dense + KTGF | 3.0-5.0x | สูง | ปานกลาง |
| Coupled Heat Transfer | 1.5-2.0x | ปานกลาง | ดี |

### 9.2 กลยุทธ์การเพิ่มประสิทธิภาพ

**เทคนิคการปรับปรุงประสิทธิภาพ:**
- **การปรับปรุงเมชแบบปรับตัว** ในบริเวณที่มีการไล่ระดับสัดส่วนปริมาตรเฟสสูง
- **แนวทางหลายมาตราส่วน** รวม CFD กับตัวแก้สมดุลประชากร
- **แบบจำลองลำดับที่ลดลง** สำหรับการทำซ้ำการออกแบบอย่างรวดเร็ว
- **การเชื่อมโยงแบบผสม** ของแนวทางลากรังจิยันและออยเลอร์

## 10. สรุป (Summary)

การเลือกแบบจำลองที่เหมาะสมสำหรับระบบก๊าซ-ของแข็งขึ้นอยู่กับ:

1. **ขนาดอนุภาค**: ละเอียด (< 100 μm) หรือ หยาบ (> 1 mm)
2. **ความเข้มข้น**: เจือจาง ($\alpha_s < 0.1$) หรือ หนาแน่น ($\alpha_s > 0.1$)
3. **ขอบเขตการไหล**: เตียงบรรจุ, เตียงไหล, หรือการขนส่งด้วยระบบสุญญากาศ
4. **ทรัพยากรการคำนวณ**: ความสมดุลระหว่างความแม่นยำและเวลาการคำนวณ

การใช้กรอบการตัดสินใจแบบลำดับชั้นเหล่านี้จะช่วยให้สามารถเลือกแบบจำลองที่เหมาะสมที่สุดสำหรับแอปพลิเคชันเฉพาะของคุณได้อย่างมีประสิทธิภาพ
