# แรงยกในการไหลแบบหลายเฟส (Lift Forces in Multiphase Flow)

## 1. บทนำ (Introduction)

**แรงยก (Lift forces)** เป็นกลไกสำคัญของการถ่ายโอนโมเมนตัมระหว่างเฟส โดยมีทิศทางตั้งฉากกับความเร็วสัมพัทธ์ระหว่างเฟส แรงนี้เกิดขึ้นจากการกระจายความดันที่ไม่สมมาตรรอบๆ อนุภาคหรือฟองอากาศ

### สถานการณ์ที่แรงยกมีความสำคัญ:
- การไหลแบบเฉือน (Shear flows) เช่น การไหลในท่อ
- การไหลที่มีความหมุนวน (Vorticity)
- ระบบ Gas-Liquid ที่ฟองอากาศมีการเสียรูป

---

## 2. กลไกทางกายภาพ (Physical Mechanisms)

| กลไก | สาเหตุการเกิด | ผลลัพธ์ |
|------|--------------|---------|
| **Shear-Induced Lift** | ความชันของความเร็วในของไหลรอบข้าง | ผลักอนุภาคไปในทิศทางแนวขวาง |
| **Rotation-Induced Lift** | การหมุนของอนุภาค (Magnus Effect) | สร้างแรงยกเนื่องจากการไหลวนรอบอนุภาค |
| **Wall-Induced Lift** | การอยู่ใกล้ผนังแข็ง | แรงผลักออกจากผนังเนื่องจากความไม่สมมาตรของไหล |
| **Deformation-Induced Lift** | การเสียรูปของฟองอากาศ | เปลี่ยนทิศทางแรงยกตามรูปร่างของฟอง |

---

## 3. สมการหลัก (Governing Equation)

ในรูปแบบทั่วไป แรงยกต่อหน่วยปริมาตรคำนวณได้จาก:

$$\mathbf{F}_L = C_L \rho_c \alpha_d (\mathbf{u}_c - \mathbf{u}_d) \times (\nabla \times \mathbf{u}_c) \tag{5.1}$$ 

**นิยามตัวแปร:**
- $C_L$: สัมประสิทธิ์แรงยก (Lift Coefficient)
- $\rho_c$: ความหนาแน่นของเฟสต่อเนื่อง (Continuous phase)
- $\alpha_d$: สัดส่วนปริมาตรของเฟสกระจาย (Dispersed phase)
- $\mathbf{u}_c - \mathbf{u}_d$: ความเร็วสัมพัทธ์ระหว่างเฟส
- $\nabla \times \mathbf{u}_c$: ความหมุนวน (Vorticity) ของเฟสต่อเนื่อง

---

## 4. แบบจำลองที่สำคัญ (Key Models)

### 4.1 Tomiyama Lift Model
เป็นโมเดลมาตรฐานสำหรับฟองอากาศในของเหลว โดยพิจารณาผลของ **Eötvös Number ($Eo$)**:
- **ฟองขนาดเล็ก ($Eo < 10$):** $C_L$ เป็นบวก (เคลื่อนที่เข้าหาผนังในท่อแนวตั้งลอยขึ้น)
- **ฟองขนาดใหญ่ ($Eo > 10$):** $C_L$ เป็นลบ (เคลื่อนที่ออกจากผนังไปสู่ศูนย์กลางท่อ)

### 4.2 Saffman-Mei Model
เหมาะสำหรับอนุภาคแข็งขนาดเล็กในระบอบการไหลที่มีเลขเรย์โนลด์ต่ำ

## 5. การนำไปใช้ใน OpenFOAM

ใน OpenFOAM (เช่น `multiphaseEulerFoam`), แรงยกจะถูกกำหนดในไฟล์ `constant/phaseProperties`:

```openfoam
lift
(
    (air in water)
    {
        type            Tomiyama;
    }
);
```

การเลือกแบบจำลองแรงยกที่ถูกต้องมีความสำคัญอย่างยิ่งต่อการทำนาย **การกระจายตัวของเฟส (Phase distribution)** ในแนวขวาง เช่น การทำนายว่าฟองอากาศจะรวมตัวกันที่กลางท่อหรือเกาะขอบท่อ
