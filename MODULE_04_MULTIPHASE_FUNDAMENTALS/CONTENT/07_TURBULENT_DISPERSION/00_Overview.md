# การกระจายตัวเนื่องจากความปั่นป่วน (Turbulent Dispersion)

## 1. บทนำ (Introduction)

**การกระจายตัวเนื่องจากความปั่นป่วน (Turbulent Dispersion)** คือปรากฏการณ์ที่อนุภาค ฟองอากาศ หรือหยดของเหลว ถูกพัดพาให้ฟุ้งกระจายออกจากเส้นทางการไหลเฉลี่ย เนื่องจากความผันผวนของความเร็ว (Velocity fluctuations) ในเฟสต่อเนื่องที่มีความปั่นป่วน

### ความสำคัญทางกายภาพ:
หากไม่มีการจำลองเทอมนี้ การจำลองแบบ Eulerian-Eulerian มักจะทำนายการรวมกลุ่มของอนุภาคในบริเวณที่มีความเร็วต่ำมากเกินความจริง (Unrealistic particle accumulation) และไม่สามารถจับพฤติกรรมการผสม (Mixing) ได้ถูกต้อง

---

## 2. กลไกทางกายภาพ (Physical Mechanisms)

### 2.1 ความผันผวนของความเร็ว
ในกระแสไหลปั่นป่วน ความเร็วรวมประกอบด้วยความเร็วเฉลี่ย ($\overline{\mathbf{u}}$) และความเร็วผันผวน ($\mathbf{u}'$):
$$\mathbf{u}_c = \overline{\mathbf{u}}_c + \mathbf{u}'_c \tag{7.1}$$ 

### 2.2 จำนวนสโตกส์ (Stokes Number - $St$)
ระบุว่าอนุภาคตอบสนองต่อความปั่นป่วนได้ดีเพียงใด:
$$St = \frac{\tau_p}{\tau_t}$$ 
- **$St \ll 1$:** อนุภาคเคลื่อนที่ตามความผันผวนของของไหลได้ทัน (การกระจายตัวสูง)
- **$St \gg 1$:** อนุภาคมีมวลมากเกินกว่าจะถูกพัดพาโดยความปั่นป่วน (การกระจายตัวต่ำ)

---

## 3. แบบจำลองทางคณิตศาสตร์ (Mathematical Models)

แบบจำลองส่วนใหญ่ใช้หลักการ **Gradient Diffusion** โดยมองว่าแรงกระจายตัวเป็นสัดส่วนกับเกรเดียนต์ของสัดส่วนปริมาตร:

### 3.1 Lopez de Bertodano Model (นิยมใช้ที่สุด)
$$\mathbf{F}_{TD} = C_{TD} \rho_c k_c \nabla \alpha_d \tag{7.2}$$ 

### 3.2 Burns et al. Model
$$D_{TD} = \frac{\nu_{t,c}}{\sigma_{TD}}$$ 
โดยที่ $\sigma_{TD}$ คือ Turbulent Schmidt Number (โดยทั่วไป $\approx 0.9$)

---

## 4. การนำไปใช้ใน OpenFOAM

ใน OpenFOAM แรงกระจายตัวเนื่องจากความปั่นป่วนถูกกำหนดใน `constant/phaseProperties`:

```openfoam
turbulentDispersion
(
    (air in water)
    {
        type            Burns;
        sigma           0.9;
        Ctd             1.0;
    }
);
```

### พารามิเตอร์ที่สำคัญ:
- **$C_{TD}$:** สัมประสิทธิ์การกระจายตัว (Default = 1.0)
- **$\\sigma_{TD}$:** อัตราส่วนความแพร่โมเมนตัมต่อความแพร่มวล

การใช้แบบจำลองนี้มีความจำเป็นอย่างยิ่งในระบบ **Bubble Column** และ **Fluidized Bed** เพื่อให้ได้โปรไฟล์ความเข้มข้นของเฟสที่ถูกต้องสอดคล้องกับการทดลอง
