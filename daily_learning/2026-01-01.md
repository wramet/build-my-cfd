## 1. Theory: Core Equations & Physics

### 1.1 Conservation of Mass (Continuity Equation)

**สมการการอนุรักษ์มวล (Conservation of Mass)**

$$
\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{U}) = 0
$$

**คำอธิบายพจน์:**
- $\rho$ (rho) = **ความหนาแน่น (density)** - มวลต่อหน่วยปริมาตร [kg/m³]
- $\mathbf{U}$ = **สนามความเร็ว (velocity field)** - เวกเตอร์ความเร็ว [m/s]
- $\nabla \cdot$ = **ไดเวอร์เจนซ์ (divergence)** - ตัวดำเนินการเชิงอนุพันธ์
- $t$ = **เวลา (time)** [s]

สำหรับการไหลแบบ **ไม่อัดตัว (incompressible flow)**: $\nabla \cdot \mathbf{U} = 0$

---

### 1.2 Conservation of Momentum (Navier-Stokes Equation)

**สมการการอนุรักษ์โมเมนตัม (Conservation of Momentum)**

$$
\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot \boldsymbol{\tau} + \rho \mathbf{g}
$$

**คำอธิบายพจน์:**
- $p$ = **ความดัน (pressure)** [Pa]
- $\boldsymbol{\tau}$ (tau) = **เทนเซอร์ความเค้น (stress tensor)** - แรงเฉือน/ความหนืด
- $\mathbf{g}$ = **ความเร่งเนื่องจากแรงโน้มถ่วง (gravitational acceleration)** [m/s²]
- $\rho \mathbf{U} \mathbf{U}$ = **การพาแบบไม่สมมาตร (convective term)** - การเคลื่อนที่ของของไหล

สำหรับ **นิวตันเหลว (Newtonian fluid)** แบบไม่อัดตัว:

$$
\frac{\partial \mathbf{U}}{\partial t} + (\mathbf{U} \cdot \nabla) \mathbf{U} = -\frac{1}{\rho} \nabla p + \nu \nabla^2 \mathbf{U} + \mathbf{g}
$$

- $\nu$ (nu) = **ความหนืดเชิงจลน์ (kinematic viscosity)** [m²/s]
- $\nabla^2$ = **ลาปลาเชียน (Laplacian)** - การแพร่กระจาย

---

### 1.3 Conservation of Energy

**สมการการอนุรักษ์พลังงาน (Conservation of Energy)**

$$
\frac{\partial (\rho e)}{\partial t} + \nabla \cdot (\rho e \mathbf{U}) = -\nabla \cdot \mathbf{q} + \boldsymbol{\tau} : \nabla \mathbf{U} + S_h
$$

**คำอธิบายพจน์:**
- $e$ = **พลังงานภายในต่อหน่วยมวล (internal energy)** [J/kg]
- $\mathbf{q}$ = **ฟลักซ์ความร้อน (heat flux)** [W/m²]
- $S_h$ = **แหล่งกำเนิดความร้อน (heat source)** [W/m³]
- $\boldsymbol{\tau} : \nabla \mathbf{U}$ = **การกระจายของพลังงานเนื่องจากความหนืด (viscous dissipation)**

---

### 1.4 Equation of State

**สมการสถานะ (Equation of State)**

สำหรับ **แก๊สอุดมคติ (Ideal Gas)**:

$$
p = \rho R T
$$

**คำอธิบายพจน์:**
- $R$ = **ค่าคงที่ของแก๊สเฉพาะ (specific gas constant)** [J/(kg·K)]
- $T$ = **อุณหภูมิสัมบูรณ์ (absolute temperature)** [K]

สำหรับ **ของไหลไม่อัดตัว (Incompressible Fluid)**:

$$
\rho = \text{constant}
$$

### 1.6 Dimensionless Numbers

**จำนวนไร้หน่วยที่สำคัญ (Important Dimensionless Numbers)**

- **เลขเรย์โนลด์ (Reynolds Number)**: $Re = \frac{\rho U L}{\mu} = \frac{U L}{\nu}$
  - อัตราส่วนระหว่างแรงเฉือนและแรงเหนียว
  - สูง = การไหลแบบปั่นป่วน (turbulent), ต่ำ = การไหลแบบลามินาร์ (laminar)

- **เลขมัค (Mach Number)**: $Ma = \frac{U}{c}$
  - อัตราส่วนระหว่างความเร็วการไหลและความเร็วเสียง
  - $Ma < 0.3$: การไหลไม่อัดตัว (incompressible)

- **เลขพรันท์ล (Prandtl Number)**: $Pr = \frac{\nu}{\alpha} = \frac{c_p \mu}{k}$
  - อัตราส่วนระหว่างการแพร่ของโมเมนตัมและความร้อน

## 3. Code Walkthrough

### 3.1 UEqn.H

**ไฟล์ UEqn.H** ใช้สำหรับสร้างสมการโมเมนตัม (Momentum Equation) ใน OpenFOAM solvers หลายตัว เช่น `simpleFoam`, `pimpleFoam` โดยใช้ Finite Volume Method (FVM)

**โครงสร้างหลักของ UEqn.H:**

```cpp
// สร้างสมการโมเมนตัมแบบไม่เชิงเส้น (Implicit momentum equation)
tmp<fvVectorMatrix> UEqn
(
    // 1. พจน์เชิงอนุพันธ์เวลา (Unsteady term - ddt)
    fvm::ddt(U)
    
    // 2. พจน์การพา (Convection term - div)
    // ใช้ upwind scheme สำหรับเสถียรภาพเชิงตัวเลข
  + fvm::div(phi, U)
    
    // 3. พจน์การแพร่ (Diffusion term - laplacian)
    // แทนด้วยความหนืดเชิงจลน์ (kinematic viscosity)
  + turbulence->divDevReff(U)
    
    // 4. แหล่งกำเนิด/การไหลของแรง (Source terms)
 ==
    // แรงดันทำหน้าที่เป็น source term ในสมการโมเมนตัม
    fvOptions(U)
);

// 5. ผ่อนคลายสมการ (Relaxation) สำหรับ steady-state
UEqn.relax();

// 6. แทรก source terms เพิ่มเติมจาก fvOptions
fvOptions.constrain(UEqn);

// 7. แก้สมการแรงดัน (Pressure equation)
// โดยใช้ momentum predictor จาก UEqn
if (pimple.momentumPredictor())
{
    solve(UEqn == -fvc::grad(p));
}
```

**คำอธิบายพจน์สำคัญ:**

| พจน์ (Term) | ความหมาย | ฟังก์ชัน OpenFOAM |
|------------|----------|-------------------|
| `fvm::ddt(U)` | อนุพันธ์เวลาของความเร็ว | First/second-order implicit |
| `fvm::div(phi, U)` | การพาของโมเมนตัม | Upwind/linear/TVD |
| `turbulence->divDevReff(U)` | การแพร่ของความเร็ว + ความปั่นป่วน | Reynolds stress |
| `fvc::grad(p)` | ไล่ระดับความดัน | Explicit gradient |

**ความแตกต่างระหว่าง fvm และ fvc:**
- **fvm** (Finite Volume Method): สร้างเมทริกซ์สำหรับ implicit solver (ใช้กับ ddt, div, laplacian)
- **fvc** (Finite Volume Calculus): คำนวณ explicit (ใช้กับ grad, curl, div ของสเกลาร์)

**ตำแหน่งไฟล์อ้างอิง:**
- `$FOAM_APP/solvers/incompressible/simpleFoam/UEqn.H`
- `$FOAM_APP/solvers/incompressible/pimpleFoam/UEqn.H`

### 3.2 pEqn.H

**ไฟล์ pEqn.H** ใช้สำหรับสร้างสมการแรงดัน (Pressure Equation) ใน OpenFOAM solvers เช่น `simpleFoam`, `pimpleFoam` โดยใช้หลักการของ SIMPLE/PISO algorithm

**โครงสร้างหลักของ pEqn.H:**

```cpp
// 1. สร้างสมการแรงดันจากสมการโมเมนตัม
// ใช้ HU แทนการคำนวณ UEqn โดยตรงเพื่อประสิทธิภาพ
volScalarField rAU(1.0/UEqn.A());
volVectorField HbyA(constrainHbyA(rAU*UEqn.H(), U, p));

// 2. คำนวณ surfaceScalarField สำหรับ flux
surfaceScalarField phiHbyA
(
    "phiHbyA",
    fvc::flux(HbyA)
  + fvc::interpolate(rAU)*fvc::ddtCorr(U, phi)
);

// 3. แก้ไข boundary conditions สำหรับ pressure
adjustPhi(phiHbyA, U, p);

// 4. สร้างและแก้สมการแรงดัน (Pressure equation)
// สำหรับ incompressible flow: laplacian(p) = div(phi)
while (simple.correctNonOrthogonal())
{
    tmp<fvScalarMatrix> pEqn
    (
        fvm::laplacian(rAU, p) == fvc::div(phiHbyA)
    );

    pEqn.setReference(pRefCell, pRefValue);
    pEqn.solve();

    if (simple.finalNonOrthogonalIter())
        phi = phiHbyA - pEqn.flux();
}

// 5. แก้ไขความเร็วจากแรงดันใหม่ (Velocity correction)
U = HbyA - rAU*fvc::grad(p);
U.correctBoundaryConditions();
```

**คำอธิบายขั้นตอนสำคัญ:**

| ขั้นตอน (Step) | ความหมาย | วัตถุประสงค์ |
|---------------|----------|--------------|
| `rAU = 1.0/UEqn.A()` | ผกผันของเมทริกซ์ | ใช้ในการคำนวณ pressure-velocity coupling |
| `HbyA` | H/U matrix | เวกเตอร์ความเร็ว intermediate |
| `fvc::flux(HbyA)` | คำนวณ flux | การไหลผ่านหน้าเซลล์ |
| `fvm::laplacian(rAU, p)` | สมการแรงดัน | Poisson equation สำหรับแรงดัน |
| `U = HbyA - rAU*grad(p)` | แก้ไขความเร็ว | ปรับความเร็วให้สอดคล้องกับ continuity |

**ความแตกต่างระหว่าง SIMPLE และ PISO:**
- **SIMPLE** (Semi-Implicit Method for Pressure-Linked Equations): ใช้กับ steady-state, มี under-relaxation
- **PISO** (Pressure Implicit with Splitting of Operators): ใช้กับ transient, แก้สมการแรงดันหลายครั้งต่อ time step

**ตำแหน่งไฟล์อ้างอิง:**
- `$FOAM_APP/solvers/incompressible/simpleFoam/pEqn.H`
- `$FOAM_APP/solvers/incompressible/pimpleFoam/pEqn.H`
