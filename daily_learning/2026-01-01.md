## 1. Theory: Core Equations & Physics

### 1.1 Conservation of Mass (Continuity Equation)

**Vector Form:**
$$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{U}) = 0$$

**Term-by-Term Explanation (Thai):**
- $\frac{\partial \rho}{\partial t}$: **อัตราการเปลี่ยนแปลงของความหนาแน่นเทียบกับเวลา** (Rate of change of density with time)
- $\nabla \cdot (\rho \mathbf{U})$: **การไหลของมวลออกจากปริมาตนควบคุม** (Mass flux out of control volume)
- $= 0$: **กฎการอนุรักษ์มวล** (Mass conservation principle)

**Physical Meaning:**
สมการนี้แสดงว่ามวลไม่สามารถถูกสร้างหรือทำลายได้ การเปลี่ยนแปลงของมวลในปริมาตรหนึ่งเท่ากับมวลที่ไหลเข้าและออกจากปริมาตรนั้น

---

### 1.2 Conservation of Momentum (Navier-Stokes Equation)

**Vector Form:**
$$\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot \boldsymbol{\tau} + \rho \mathbf{g}$$

**Term-by-Term Explanation (Thai):**
- $\frac{\partial (\rho \mathbf{U})}{\partial t}$: **อัตราการเปลี่ยนแปลงของโมเมนตัมเชิงเวลา** (Unsteady term - rate of change of momentum)
- $\nabla \cdot (\rho \mathbf{U} \mathbf{U})$: **การพาย้ายโมเมนตัม (Convection)** - การเคลื่อนที่ของโมเมนตัมเนื่องจากการไหลของของไหล
- $-\nabla p$: **แรงดันความดัน (Pressure gradient force)** - แรงที่เกิดจากความแตกต่างของความดัน
- $\nabla \cdot \boldsymbol{\tau}$: **แรงเฉือน (Viscous forces)** - แรงจากความหนืดของของไหล
- $\rho \mathbf{g}$: **แรงโน้มถ่วง (Body force)** - แรงภายนอกที่กระทำต่อปริมาตรของของไหล

**Physical Meaning:**
สมการนี้เป็นการนำกฎข้อที่สองของนิวตันมาใช้กับของไหล: ผลรวมของแรงทั้งหมดที่กระทำต่อปริมาตนของไหลเท่ากับอัตราการเปลี่ยนแปลงของโมเมนตัม

---

### 1.3 Conservation of Energy

**Vector Form:**
$$\frac{\partial (\rho h)}{\partial t} + \nabla \cdot (\rho \mathbf{U} h) = \frac{Dp}{Dt} + \nabla \cdot (k \nabla T) + \boldsymbol{\tau} : \nabla \mathbf{U} + S_h$$

**Term-by-Term Explanation (Thai):**
- $\frac{\partial (\rho h)}{\partial t}$: **อัตราการเปลี่ยนแปลงของเอนทาลปีเชิงเวลา** (Rate of change of enthalpy)
- $\nabla \cdot (\rho \mathbf{U} h)$: **การพาย้ายความร้อน (Convective heat transfer)** - การเคลื่อนที่ของความร้อนเนื่องจากการไหล
- $\frac{Dp}{Dt}$: **งานจากความดัน (Pressure work)** - งานที่เกิดจากการเปลี่ยนแปลงความดัน
- $\nabla \cdot (k \nabla T)$: **การนำความร้อน (Heat conduction)** - การถ่ายเทความร้อนโดยการนำ
- $\boldsymbol{\tau} : \nabla \mathbf{U}$: **การกระจายของพลังงานเนื่องจากความหนืด (Viscous dissipation)** - พลังงานที่สูญเสียไปเป็นความร้อนจากแรงเฉือน
- $S_h$: **แหล่งกำเนิดความร้อน (Heat source)** - ความร้อนที่เพิ่มเข้ามาภายนอกระบบ

**Physical Meaning:**
กฎการอนุรักษ์พลังงานสำหรับของไหล ซึ่งพิจารณาทั้งการเปลี่ยนแปลงของเอนทาลปี งาน และการถ่ายเทความร้อน

---

### 1.4 Equation of State

**For Comible Flow:**
$$p = \rho R T$$

**For Incompressible Flow:**
$$\rho = \text{constant}$$

**Thai Explanation:**
- **สมการสถานะ (Equation of State)**: ความสัมพันธ์ระหว่างความดัน ความหนาแน่น และอุณหภูมิ
- สำหรับการไหลที่ไม่อัดตัว (Incompressible): ความหนาแน่นค่าคงที่
- สำหรับการไหลที่อัดตัวได้ (Comible): ใช้กฎของแก๊สอุดมคติ

---

### 1.5 Stress Tensor for Newtonian Fluid

$$\boldsymbol{\tau} = \mu \left[ \nabla \mathbf{U} + (\nabla \mathbf{U})^T \right] - \frac{2}{3} \mu (\nabla \cdot \mathbf{U}) \mathbf{I}$$

**Thai Explanation:**
- $\mu$: **ความหนืดพลศาสตร์ (Dynamic viscosity)** - ความต้านทานต่อการไหลของของไหล
- $\nabla \mathbf{U}$: **เทนเซอร์อัตราการบิดเบือน (Rate of strain tensor)** - อัตราการเปลี่ยนรูปของของไหล
- $\mathbf{I}$: **เทนเซอร์เอกลักษณ์ (Identity tensor)**

**Physical Meaning:**
แรงเฉือนในของไหลชนิดนิวตัน (Newtonian fluid) แปรผันโดยตรงกับอัตราการบิดเบือน

## 2. OpenFOAM Class Hierarchy & Implementation

### 2.1 Core Field Classes

#### fvMesh (Finite Volume Mesh)
**Purpose:** Represents the computational mesh for finite volume discretization

**Inheritance Diagram:**
```
          polyMesh
              ^
              |
            fvMesh
              |
    +---------+---------+
    |         |         |
 fvBoundaryMesh fvSchemes fvSolution
```

**Memory Layout:**
```
fvMesh
├── points_          : pointField (List of mesh vertices)
├── faces_           : faceField (List of faces)
├── cells_           : cellList (List of cells)
├── owner_           : labelList (Face owner cell IDs)
├── neighbour_       : labelList (Face neighbor cell IDs)
├── Sf_              : surfaceVectorField (Face area vectors)
├── magSf_           : surfaceScalarField (Face area magnitudes)
├── C_               : volVectorField (Cell centers)
└── V_               : volScalarField (Cell volumes)
```

**Reference Files:**
- `$FOAM_SRC/finiteVolume/finiteVolume/fvMesh.H`
- `$FOAM_SRC/finiteVolume/finiteVolume/fvMesh.C`

---

#### GeometricField (Template for Fields)
**Purpose:** Base class for all field types (volScalarField, volVectorField, etc.)

**Inheritance Diagram:**
```
       DimensionedField
              ^
              |
         GeometricField
              |
    +---------+---------+---------+
    |         |         |         |
volScalarField volVectorField surfaceScalarField surfaceVectorField
```

**Memory Layout:**
```
GeometricField<Type>
├── storedPoints_   : List of mesh points
├── storedFaces_    : List of faces
├── storedCells_    : List of cells
├── boundaryField_  : GeometricBoundaryField (Boundary conditions)
├── internalField_  : Field<Type> (Internal cell values)
├── mesh_           : reference to fvMesh
├── dimensions_     : dimensionSet (Units: [kg m s K mol A cd])
└── oldTime_        : PtrList<GeometricField> (Previous time steps)
```

**Reference Files:**
- `$FOAM_SRC/OpenFOAM/fields/GeometricField/GeometricField.C`
- `$FOAM_SRC/finiteVolume/fields/volFields/volScalarField/volScalarField.H`

---

### 2.2 Discretization Classes

#### fvMatrix (Finite Volume Matrix)
**Purpose:** Represents the discretized system of linear equations [A]x = b

**Inheritance Diagram:**
```
       lduMatrix
           ^
           |
       fvMatrix
```

**Memory Layout:**
```
fvMatrix<Type>
├── lduMatrix_       : Base matrix (Lower-Diagonal-Upper)
├── source_          : Field<Type> (RHS vector b)
├── psi_             : GeometricField<Type> (Solution field x)
├── dimensions_      : dimensionSet
├── boundaryDiag_    : Field<Type> (Boundary diagonal coeffs)
├── boundaryCoeffs_  : Field<Type> (Boundary source coeffs)
└── internalCoeffs_  : Field<Type> (Internal coeffs)
```

**Reference Files:**
- `$FOAM_SRC/finiteVolume/finiteVolume/fvMatrices/fvMatrix/fvMatrix.H`
- `$FOAM_SRC/finiteVolume/finiteVolume/fvMatrices/fvMatrix/fvMatrix.C`

---

#### fvm::div (Convection Term)
**Purpose:** Discretizes the convection term ∇·(ρUU)

**Inheritance:**
```
       tmp<fvMatrix<Type>>
              ^
              |
         fvm::div
```

**Memory Layout:**
```
fvm::div
├── scheme_         : divScheme<Type> (Gauss, upwind, etc.)
├── faceFlux_       : surfaceScalarField (Mass flux at faces)
└── psi_            : GeometricField<Type> (Field to discretize)
```

**Reference Files:**
- `$FOAM_SRC/finiteVolume/finiteVolume/fvSchemes/fvSchemes.C`
- `$FOAM_SRC/finiteVolume/finiteVolume/fvm/fvmDiv.C`

---

#### fvm::laplacian (Diffusion Term)
**Purpose:** Discretizes the diffusion term ∇·(μ∇U)

**Memory Layout:**
```
fvm::laplacian
├── scheme_         : laplacianScheme<Type>
├── gamma_          : surfaceScalarField (Diffusivity at faces)
└── psi_            : GeometricField<Type> (Field to discretize)
```

**Reference Files:**
- `$FOAM_SRC/finiteVolume/finiteVolume/fvm/fvmLaplacian.C`

---

### 2.3 Solver Classes

#### UEqn (Momentum Equation)
**Purpose:** Solves the momentum equation for velocity U

**Memory Layout:**
```
UEqn (fvMatrix<vector>)
├── Convection      : fvm::div(phi, U)
├── Diffusion       : fvm::laplacian(nu, U)
├── Source          : - fvc::grad(p)
└── Boundary        : Wall functions, inlets, outlets
```

**Reference Files:**
- `$FOAM_SRC/applications/solvers/incompressible/simpleFoam/UEqn.H`

---

#### pEqn (Pressure Equation)
**Purpose:** Solves the pressure equation to enforce continuity

**Memory Layout:**
```
pEqn (fvScalarMatrix)
├── Laplacian       : fvm::laplacian(rAU, p)
├── Source          : fvc::div(phi) (Mass imbalance)
└── Correction      : phi -= pEqn.flux()
```

**Reference Files:**
- `$FOAM_SRC/applications/solvers/incompressible/simpleFoam/pEqn.H`

---

### 2.4 Transport Models

#### incompressible::RASModel
**Purpose:** Base class for Reynolds-Averaged Simulation turbulence models

**Inheritance Diagram:**
```
       turbulenceModel
              ^
              |
       RASModel
              ^
              |
    +---------+---------+
    |         |         |
kEpsilon   kOmegaSST   SpalartAllmaras
```

**Memory Layout:**
```
RASModel
├── nu_              : volScalarField (Kinematic viscosity)
├── nut_             : volScalarField (Turbulent viscosity)
├── k_               : volScalarField (Turbulent kinetic energy)
├── epsilon_         : volScalarField (Dissipation rate)
├── U_               : volVectorField (Velocity field)
├── phi_             : surfaceScalarField (Flux field)
└── coeffDict_       : dictionary (Model coefficients)
```

**Reference Files:**
- `$FOAM_SRC/turbulenceModels/incompressible/RAS/RASModel.H`
- `$FOAM_SRC/turbulenceModels/incompressible/RAS/kEpsilon/kEpsilon.H`

---

### 2.5 Thermophysical Models

#### rhoThermo
**Purpose:** Calculates thermodynamic properties (density, enthalpy, temperature)

**Inheritance Diagram:**
```
       fluidThermo
              ^
              |
          rhoThermo
              ^
              |
    +---------+---------+
    |                   |
heRhoThermo        hRhoThermo
```

**Memory Layout:**
```
rhoThermo
├── rho_             : volScalarField (Density field)
├── mu_              : volScalarField (Dynamic viscosity)
├── alpha_           : volScalarField (Thermal diffusivity)
├── Cp_              : volScalarField (Specific heat capacity)
├── p_               : volScalarField (Pressure field)
├── T_               : volScalarField (Temperature field)
└── mixture_         : PtrList<species> (Species properties)
```

**Reference Files:**
- `$FOAM_SRC/thermophysicalModels/basic/rhoThermo/rhoThermo.H`
- `$FOAM_SRC/thermophysicalModels/basic/BasicThermo/BasicThermo.H`

**Purpose:** Dirichlet boundary condition (fixed value at boundary)

**Memory Layout:**
```
fixedValueFvPatchField<Type>
├── value_          : Field<Type> (Fixed boundary values)
└── patch_          : reference to fvPatch
```

**Reference Files:**
- `$FOAM_SRC/finiteVolume/fields/fvPatchFields/basic/fixedValue/fixedValueFvPatchField.H`

**Memory Layout:**

zeroGradientFvPatchField<Type>

└── patch_          : reference to fvPatch

**Reference Files:**
- `$FOAM_SRC/finiteVolume/fields/fvPatchFields/basic/zeroGradient/zeroGradientFvPatchField.H`

### 2.7 Time Integration Classes

#### Time
**Purpose:** Manages time stepping and run-time control

**Memory Layout:**
```
Time
├── startTime_       : scalar (Simulation start time)
├── endTime_         : scalar (Simulation end time)
├── deltaT_          : scalar (Current time step)
├── deltaTSave_      : scalar (Saved time step)
├── timeIndex_       : label (Current time step index)
├── writeTime_       : scalar (Next write time)
├── controlDict_     : dictionary (Run-time control)
└── times_           : instantList (Available time directories)
```

**Reference Files:**
- `$FOAM_SRC/OpenFOAM/db/Time/Time.H`
- `$FOAM_SRC/OpenFOAM/db/Time/Time.C`
