## 1. Theory: Core Equations & Physics

### 1.1 Conservation of Mass (Continuity Equation)

**สมการการอนุรักษ์มวล (Continuity Equation)**

$$
\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{U}) = 0
$$

**คำอธิบายพจน์:**
- $\rho$ (rho) = **ความหนาแน่น** (density) [kg/m³]
- $\mathbf{U}$ = **สนามความเร็ว** (velocity field) [m/s]
- $\nabla \cdot$ = **ตัวดำเนินการไดเวอร์เจนซ์** (divergence operator)
- $t$ = **เวลา** (time) [s]

สำหรับการไหลแบบ **ไม่อัดตัว (incompressible flow)**: $\nabla \cdot \mathbf{U} = 0$

---

### 1.2 Conservation of Momentum (Navier-Stokes Equation)

**สมการการอนุรักษ์โมเมนตัม (Momentum Equation)**

$$
\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot \boldsymbol{\tau} + \rho \mathbf{g}
$$

**คำอธิบายพจน์:**
- $p$ = **ความดัน** (pressure) [Pa]
- $\boldsymbol{\tau}$ (tau) = **เทนเซอร์ความเค้น** (stress tensor) [Pa]
- $\mathbf{g}$ = **ความเร่งเนื่องจากแรงโน้มถ่วง** (gravitational acceleration) [m/s²]
- $\nabla p$ = **การไล่ระดับความดัน** (pressure gradient)
- $\rho \mathbf{g}$ = **แรงลมตามแนวดิ่ง** (body force)

สำหรับ **ของไหลนิวตัน (Newtonian fluid)**:

$$
\boldsymbol{\tau} = \mu \left[ \nabla \mathbf{U} + (\nabla \mathbf{U})^T \right] - \frac{2}{3}\mu (\nabla \cdot \mathbf{U})\mathbf{I}
$$

- $\mu$ (mu) = **ความหนืด** (dynamic viscosity) [Pa·s]
- $\mathbf{I}$ = **เทนเซอร์เอกลักษณ์** (identity tensor)

---

### 1.3 Conservation of Energy

**สมการการอนุรักษ์พลังงาน (Energy Equation)**

$$
\frac{\partial (\rho e)}{\partial t} + \nabla \cdot (\rho e \mathbf{U}) = -\nabla \cdot \mathbf{q} + \boldsymbol{\tau} : \nabla \mathbf{U} + S_h
$$

**คำอธิบายพจน์:**
- $e$ = **พลังงานภายในต่อหน่วยมวล** (specific internal energy) [J/kg]
- $\mathbf{q}$ = **ฟลักซ์ความร้อน** (heat flux) [W/m²]
- $\boldsymbol{\tau} : \nabla \mathbf{U}$ = **การกระจายพลังงานเนื่องจากความเค้น** (viscous dissipation)
- $S_h$ = **แหล่งกำเนิดความร้อน** (heat source) [W/m³]

ตาม **กฎของฟูริเยร์ (Fourier's Law)**:

$$
\mathbf{q} = -k \nabla T
$$

- $k$ = **สัมประสิทธิ์การนำความร้อน** (thermal conductivity) [W/(m·K)]
- $T$ = **อุณหภูมิ** (temperature) [K]

---

### 1.4 Equation of State

**สมการสถานะ (Equation of State)**

สำหรับ **แก๊สอุดมคติ (Ideal Gas)**:

$$
p = \rho R T
$$

- $R$ = **ค่าคงที่ของแก๊สเฉพาะ** (specific gas constant) [J/(kg·K)]

สำหรับ **ของไหลที่ไม่อัดตัว (Incompressible Fluid)**:

$$
\rho = \text{constant}
$$

---

### 1.5 Transport Equations (General Form)

**สมการขนส่งทั่วไป (General Transport Equation)**

$$
\frac{\partial (\rho \phi)}{\partial t} + \nabla \cdot (\rho \phi \mathbf{U}) = \nabla \cdot (\Gamma_\phi \nabla \phi) + S_\phi
$$

**คำอธิบายพจน์:**
- $\phi$ (phi) = **ตัวแปรขนส่ง** (transported variable) - สามารถเป็นอุณหภูมิ, ความเข้มข้น, หรือปริมาณสเกลาร์อื่นๆ
- $\Gamma_\phi$ (Gamma) = **สัมประสิทธิ์การแพร่** (diffusion coefficient)
- $S_\phi$ = **เทอมแหล่งกำเนิด** (source term)

**ความหมาย:** การเปลี่ยนแปลงของปริมาณ $\phi$ ในปริมาตรควบคุม = การไหลเข้า/ออก - การแพร่ + แหล่งกำเนิด

## 2. OpenFOAM Class Hierarchy & Implementation

### 2.1 Core Field Classes

OpenFOAM uses a sophisticated field-based architecture to represent and solve governing equations.

```
GeometricField
├── DimensionedField
│   └── Scalar fields (volScalarField, surfaceScalarField)
├── volField (Internal field + boundary field)
│   ├── volScalarField (e.g., p, T, rho)
│   ├── volVectorField (e.g., U)
│   ├── volTensorField (e.g., tau, gradU)
│   └── volSymmTensorField (e.g., stress)
└── surfaceField (Face fluxes)
    ├── surfaceScalarField (e.g., phi)
    └── surfaceVectorField
```

**Key Files:**
- `$FOAM_SRC/finiteVolume/fields/volFields/volFields.H`
- `$FOAM_SRC/finiteVolume/fields/surfaceFields/surfaceFields.H`
- `$FOAM_SRC/finiteVolume/fields/fvPatchFields/fvPatchFields.H`

## 3. Code Walkthrough

### 3.1 UEqn.H

ไฟล์ `UEqn.H` ใช้สำหรับประกอบสมการโมเมนตัม (Momentum Equation) ใน OpenFOAM solver ต่างๆ เช่น `simpleFoam`, `pimpleFoam` โดยแบ่งเป็นสองส่วนหลักคือการสร้าง implicit term และ explicit term

**ตัวอย่างโค้ดจาก `applications/solvers/incompressible/simpleFoam/UEqn.H`:**

```cpp
// สร้าง temporary equation สำหรับ implicit terms
tmp<fvVectorMatrix> tUEqn
(
    fvm::div(phi, U)      // Convection term (อนุพันธ์เชิงพื้นที่ของการพาความร้อน)
  + fvm::laplacian(nu, U) // Diffusion term (เทอมการแพร่ของความหนืด)
  + fvc::div(phi, T)      // Explicit source term (ถ้ามี)
);

fvVectorMatrix& UEqn = tUEqn.ref();

// ใส่เทอมแหล่งกำเนิด (source terms) จากแรงปฏิกิริยาหรือแรงภายนอก
UEqn.relax();             // Under-relaxation สำหรับ steady-state

// แก้สมการโมเมนตัม (pressure-velocity coupling)
solve(UEqn == -fvc::grad(p)); // ความดันเป็น source term
```

**คำอธิบาย:**
- `fvm` (Finite Volume Method) = **Implicit discretization** ใช้สำหรับเทอมที่ต้องการความเสถียร เช่น diffusion
- `fvc` (Finite Volume Calculus) = **Explicit calculation** ใช้สำหรับเทอมที่คำนวณโดยตรง เช่น pressure gradient
- `phi` = **Flux field** ($\phi = \mathbf{U} \cdot \mathbf{S_f}$) ใช้ใน convection term
- `nu` = **Kinematic viscosity** ($\nu = \mu/\rho$) สำหรับ incompressible flow
- `relax()` = **Under-relaxation** ช่วยให้การลู่เข้ามีเสถียรภาพมากขึ้น

### 3.2 pEqn.H

ไฟล์ `pEqn.H` ใช้สำหรับแก้สมการความดัน (Pressure Equation) เพื่อบังคับให้ได้รักษาสมการต่อเนื่อง (continuity equation) สำหรับการไหลแบบไม่อัดตัว โดยใช้วิธี SIMPLE (Semi-Implicit Method for Pressure-Linked Equations) หรือ PISO (Pressure-Implicit with Splitting of Operators)

**ตัวอย่างโค้ดจาก `applications/solvers/incompressible/simpleFoam/pEqn.H`:**

```cpp
// สร้างสมการความดันจากสมการโมเมนตัม
// แยกส่วน implicit และ explicit ของสมการ
volScalarField rAU(1.0/UEqn.A());         // 逆ของเมทริกซ์สัมประสิทธิ์
volVectorField HbyA(constrainHbyA(rAU*UEqn.H(), U, p)); // คำนวณ HbyA

// แก้ไข flux ที่ขอบเขตเพื่อให้สอดคล้องกับเงื่อนไขขอบเขต
surfaceScalarField phiHbyA
(
    "phiHbyA",
    fvc::flux(HbyA)
  + fvc::interpolate(rAU)*fvc::ddtCorr(U, phi)
);

// ปรับปรุง pressure boundary conditions
adjustPhi(phiHbyA, U, p);

// สร้างสมการความดัน (Poisson equation)
// Non-orthogonal correction loop
while (simple.correctNonOrthogonal())
{
    fvScalarMatrix pEqn
    (
        fvm::laplacian(rAU, p) == fvc::div(phiHbyA)
    );

    pEqn.setReference(pRefCell, pRefValue); // ตั้งค่า reference pressure
    pEqn.solve();                            // แก้สมการความดัน

    if (simple.finalNonOrthogonalIter())
    {
        phi = phiHbyA - pEqn.flux(); // ปรับ flux ให้สอดคล้องกับ pressure
    }
}

// แก้ไขความเร็วโดยใช้ gradient ของความดันใหม่
#include "continuityErrs.H"  // ตรวจสอบความผิดพลาดของสมการต่อเนื่อง
U = HbyA - rAU*fvc::grad(p); // ปรับความเร็ว
U.correctBoundaryConditions(); // แก้ไขค่าที่ขอบเขต
```

**คำอธิบาย:**
- `rAU` = **Inverse of diagonal coefficients** จากเมทริกซ์สมการโมเมนตัม
- `HbyA` = **Explicit velocity term** คำนวณจาก $H/A$ โดยไม่รวม pressure gradient
- `phiHbyA` = **Flux field** ที่คำนวณจาก HbyA และ correction terms
- `laplacian(rAU, p)` = **Diffusion term** ของสมการ Poisson สำหรับความดัน
- `div(phiHbyA)` = **Source term** จาก mass flux imbalance
- `simple.correctNonOrthogonal()` = **Non-orthogonal correction loop** สำหรับ mesh ที่ไม่ตั้งฉาก
- `pEqn.solve()` = **Solve pressure Poisson equation** เพื่อหาค่าความดันใหม่
- `pEqn.flux()` = **Pressure flux** ใช้ปรับ flux ให้สอดคล้องกับ continuity
