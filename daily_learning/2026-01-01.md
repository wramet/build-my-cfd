## 1. Theory: Core Equations & Physics

### 1.1 Conservation of Mass (Continuity Equation)

**Vector Form:**
$$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{U}) = 0$$

**Term-by-Term Explanation (Thai):**
- $\frac{\partial \rho}{\partial t}$: **อัตราการเปลี่ยนแปลงของความหนาแน่นเทียบกับเวลา** (Rate of change of density with time)
- $\nabla \cdot (\rho \mathbf{U})$: **การไหลของมวลออกจากปริมาตนควบคุม** (Mass flux out of control volume)
- $= 0$: **กฎการอนุรักษ์มวล** (Mass conservation principle)

**Physical Meaning:**
สมการนี้แสดงว่ามวลไม่สามารถถูกสร้างหรือทำลายได้ การเปลี่ยนแปลงของมวลในปริมาตรหนึ่งเท่ากับมวลที่ไหลเข้าและออกจากปริมาตรนั้น

---

### 1.2 Conservation of Momentum (Navier-Stokes Equation)

**Vector Form:**
$$\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot \boldsymbol{\tau} + \rho \mathbf{g}$$

**Term-by-Term Explanation (Thai):**
- $\frac{\partial (\rho \mathbf{U})}{\partial t}$: **อัตราการเปลี่ยนแปลงของโมเมนตัมเชิงเวลา** (Unsteady term - rate of change of momentum)
- $\nabla \cdot (\rho \mathbf{U} \mathbf{U})$: **การพาย้ายโมเมนตัม (Convection)** - การเคลื่อนที่ของโมเมนตัมเนื่องจากการไหลของของไหล
- $-\nabla p$: **แรงดันความดัน (Pressure gradient force)** - แรงที่เกิดจากความแตกต่างของความดัน
- $\nabla \cdot \boldsymbol{\tau}$: **แรงเฉือน (Viscous forces)** - แรงจากความหนืดของของไหล
- $\rho \mathbf{g}$: **แรงโน้มถ่วง (Body force)** - แรงภายนอกที่กระทำต่อปริมาตรของของไหล

**Physical Meaning:**
สมการนี้เป็นการนำกฎข้อที่สองของนิวตันมาใช้กับของไหล: ผลรวมของแรงทั้งหมดที่กระทำต่อปริมาตนของไหลเท่ากับอัตราการเปลี่ยนแปลงของโมเมนตัม

---

### 1.3 Conservation of Energy

**Vector Form:**
$$\frac{\partial (\rho h)}{\partial t} + \nabla \cdot (\rho \mathbf{U} h) = \frac{Dp}{Dt} + \nabla \cdot (k \nabla T) + \boldsymbol{\tau} : \nabla \mathbf{U} + S_h$$

**Term-by-Term Explanation (Thai):**
- $\frac{\partial (\rho h)}{\partial t}$: **อัตราการเปลี่ยนแปลงของเอนทาลปีเชิงเวลา** (Rate of change of enthalpy)
- $\nabla \cdot (\rho \mathbf{U} h)$: **การพาย้ายความร้อน (Convective heat transfer)** - การเคลื่อนที่ของความร้อนเนื่องจากการไหล
- $\frac{Dp}{Dt}$: **งานจากความดัน (Pressure work)** - งานที่เกิดจากการเปลี่ยนแปลงความดัน
- $\nabla \cdot (k \nabla T)$: **การนำความร้อน (Heat conduction)** - การถ่ายเทความร้อนโดยการนำ
- $\boldsymbol{\tau} : \nabla \mathbf{U}$: **การกระจายของพลังงานเนื่องจากความหนืด (Viscous dissipation)** - พลังงานที่สูญเสียไปเป็นความร้อนจากแรงเฉือน
- $S_h$: **แหล่งกำเนิดความร้อน (Heat source)** - ความร้อนที่เพิ่มเข้ามาภายนอกระบบ

**Physical Meaning:**
กฎการอนุรักษ์พลังงานสำหรับของไหล ซึ่งพิจารณาทั้งการเปลี่ยนแปลงของเอนทาลปี งาน และการถ่ายเทความร้อน

---

### 1.4 Equation of State

**For Comible Flow:**
$$p = \rho R T$$

**For Incompressible Flow:**
$$\rho = \text{constant}$$

**Thai Explanation:**
- **สมการสถานะ (Equation of State)**: ความสัมพันธ์ระหว่างความดัน ความหนาแน่น และอุณหภูมิ
- สำหรับการไหลที่ไม่อัดตัว (Incompressible): ความหนาแน่นค่าคงที่
- สำหรับการไหลที่อัดตัวได้ (Comible): ใช้กฎของแก๊สอุดมคติ

---

### 1.5 Stress Tensor for Newtonian Fluid

$$\boldsymbol{\tau} = \mu \left[ \nabla \mathbf{U} + (\nabla \mathbf{U})^T \right] - \frac{2}{3} \mu (\nabla \cdot \mathbf{U}) \mathbf{I}$$

**Thai Explanation:**
- $\mu$: **ความหนืดพลศาสตร์ (Dynamic viscosity)** - ความต้านทานต่อการไหลของของไหล
- $\nabla \mathbf{U}$: **เทนเซอร์อัตราการบิดเบือน (Rate of strain tensor)** - อัตราการเปลี่ยนรูปของของไหล
- $\mathbf{I}$: **เทนเซอร์เอกลักษณ์ (Identity tensor)**

**Physical Meaning:**
แรงเฉือนในของไหลชนิดนิวตัน (Newtonian fluid) แปรผันโดยตรงกับอัตราการบิดเบือน

### 3.2 UEqn.H - Momentum Equation

**Purpose:** Constructs and solves the momentum equation for velocity U

**Call Stack:**
```
main()
  └──> UEqn.H
        ├──> fvm::div (convection term discretization)
        ├──> fvm::laplacian (diffusion term discretization)
        └──> fvMatrix::solve (linear system solver)
```

**Code:**
```cpp
// 1. Construct momentum equation
// สร้างสมการโมเมนตัม
// fvMatrix<vector>: Matrix representing discretized momentum equation
tmp<fvVectorMatrix> UEqn
(
    // Convection term: ∇·(φU) - upwind scheme
    // พจน์การพา: ∇·(φU) - ใช้รูปแบบ upwind
    fvm::div(phi, U)
    
    // Diffusion term: ∇·(ν∇U) - Gauss linear
    // พจน์การแพร่: ∇·(ν∇U) - ใช้รูปแบบ Gauss linear
  + fvm::laplacian(nu, U)
);

// 2. Add under-relaxation if needed
// เพิ่ม under-relaxation หากจำเป็น
// Under-relaxation stabilizes iterative solution
UEqn().relax();

// 3. Solve momentum equation (optional in SIMPLE)
// แก้สมการโมเมนตัม (ไม่บังคับใน SIMPLE algorithm)
// In SIMPLE: momentum is solved after pressure correction
// solve(UEqn() == -fvc::grad(p));
```

**Thai Explanation:**
- **fvm::div(phi, U)**: การเก็บตก (discretization) พจน์การพา (convection) แบบ implicit
  - ใช้รูปแบบ upwind เพื่อเสถียรภาพเชิงตัวเลข
  - φ คือ mass flux ที่ผิวเซลล์
- **fvm::laplacian(nu, U)**: การเก็บตกพจน์การแพร่ (diffusion) แบบ implicit
  - ν คือ kinematic viscosity
  - ใช้รูปแบบ Gauss linear สำหรับการคำนวณ gradient
- **UEqn().relax()**: ใช้ under-relaxation เพื่อช่วยให้ iterative solution ลู่เข้า
- **fvVectorMatrix**: เมทริกซ์สำหรับระบบสมการเชิงเส้น [A]x = b

**Mathematical Form:**
$$\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot (\nu \nabla \mathbf{U})$$

Discretized as:
$$[A]\mathbf{U} = \mathbf{H} - \nabla p$$

Where:
- $[A]$: Coefficient matrix (convection + diffusion)
- $\mathbf{U}$: Velocity vector
- $\mathbf{H}$: Source term (off-diagonal coefficients)
- $\nabla p$: Pressure gradient

### 3.3 pEqn.H - Pressure Equation

**Purpose:** Solves the pressure equation to enforce mass conservation (continuity)

**Call Stack:**
```
main()
  └──> pEqn.H
        ├──> UEqn.H (get momentum matrix)
        ├──> fvc::grad (pressure gradient)
        ├──> fvm::laplacian (pressure Poisson equation)
        ├──> fvScalarMatrix::solve (linear solver)
        └──> fvc::flux (correct flux field)
```

**Code:**
```cpp
// 1. Get momentum matrix from UEqn
// รับเมทริกซ์โมเมนตัมจาก UEqn
// This matrix contains convection and diffusion terms
tmp<fvVectorMatrix> UEqn = ...;  // From UEqn.H

// 2. Calculate velocity from momentum equation (H/A)
// คำนวณความเร็วจากสมการโมเมนตัม
// U = H/A - (1/A)*grad(p)
volScalarField rAU(1.0/UEqn().A());  // Reciprocal of diagonal
volVectorField HbyA(constrainHbyA(rAU*UEqn().H(), U, p));

// 3. Calculate surface flux without pressure gradient
// คำนวณ flux ที่ผิวโดยไม่รวม gradient ของความดัน
surfaceScalarField phiHbyA
(
    "phiHbyA",
    fvc::flux(HbyA)
);

// 4. Adjust flux for fixed boundary conditions
// ปรับ flux สำหรับเงื่อนไขขอบเขตแบบ fixed value
adjustPhi(phiHbyA, U, p);

// 5. Set non-orthogonal correction
// ตั้งค่าการแก้ไขแบบ non-orthogonal
// Non-orthogonal meshes require iterative correction
label nNonOrthCorr = 0;  // Number of non-orthogonal corrections

// 6. Construct pressure equation (Poisson equation)
// สร้างสมการความดัน (สมการ Poisson)
// ∇·(1/A ∇p) = ∇·(φ*/A)
// This enforces mass conservation: ∇·U = 0
for (int nonOrth = 0; nonOrth <= nNonOrthCorr; nonOrth++)
{
    // Pressure Poisson equation
    // สมการ Poisson สำหรับความดัน
    fvScalarMatrix pEqn
    (
        fvm::laplacian(rAU, p) == fvc::div(phiHbyA)
    );
    
    // 7. Set reference value for pressure
    // ตั้งค่าอ้างอิงสำหรับความดัน
    // Pressure is relative, so fix one cell value
    pEqn.setReference(pRefCell, pRefValue);
    
    // 8. Solve pressure equation
    // แก้สมการความดัน
    pEqn.solve();
    
    // 9. Correct flux field using pressure gradient
    // แก้ไข flux โดยใช้ gradient ของความดัน
    // φ = φ* - (1/A)*∇p
    if (nonOrth == nNonOrthCorr)
    {
        phi = phiHbyA - pEqn.flux();
    }
}

// 10. Continuity error check
// ตรวจสอบความผิดพลาดของสมการต่อเนื่อง
#include "continuityErrs.H"

// 11. Correct velocity field using pressure gradient
// แก้ไขสนามความเร็วโดยใช้ gradient ของความดัน
// U = U* - (rAU)*grad(p)
U -= rAU*fvc::grad(p);

// 12. Correct boundary conditions
// แก้ไขเงื่อนไขขอบเขต
U.correctBoundaryConditions();
```

**Thai Explanation:**
- **rAU = 1/A**: ค่าผกผันของสัมประสิทธิ์เชิงเส้น (inverse of diagonal coefficients)
- **HbyA = H/A**: ส่วนของความเร็วที่ไม่ขึ้นกับความดัน (explicit part)
- **phiHbyA**: flux ที่คำนวณจาก HbyA (ยังไม่รวม pressure gradient)
- **fvm::laplacian(rAU, p)**: พจน์ด้านซ้ายของสมการ Poisson สำหรับความดัน
- **fvc::div(phiHbyA)**: พจน์ด้านขวา (source term) จาก mass imbalance
- **pEqn.solve()**: แก้ระบบสมการเชิงเส้นเพื่อหาค่า p
- **pEqn.flux()**: คำนวณ flux จาก pressure gradient
- **phi = phiHbyA - pEqn.flux()**: แก้ไข flux ให้สอดคล้องกับ continuity equation
- **U -= rAU*fvc::grad(p)**: แก้ไขความเร็วโดยใช้ pressure gradient

**Mathematical Form:**

Starting from momentum equation:
$$\mathbf{U} = \frac{\mathbf{H}}{A} - \frac{1}{A}\nabla p$$

Take divergence and enforce continuity (∇·U = 0):
$$\nabla \cdot \left(\frac{1}{A}\nabla p\right) = \nabla \cdot \left(\frac{\mathbf{H}}{A}\right)$$

This is the **Pressure Poisson Equation**:
$$\nabla^2 p = \nabla \cdot \mathbf{U}^*$$

Where:
- $p$: Pressure field
- $A$: Diagonal coefficients from momentum matrix
- $\mathbf{H}$: Off-diagonal terms + source terms
- $\mathbf{U}^* = \mathbf{H}/A$: Intermediate velocity field

**SIMPLE Algorithm Steps:**
1. Guess pressure field $p^*$
2. Solve momentum equation: $\mathbf{U}^* = f(p^*)$
3. Solve pressure equation: $p^{new} = f(\mathbf{U}^*)$
4. Correct velocity: $\mathbf{U}^{new} = \mathbf{U}^* - \frac{1}{A}\nabla(p^{new} - p^*)$
5. Repeat until convergence

### 3.2 UEqn.H - Momentum Equation

**Purpose:** Constructs and solves the momentum equation for velocity U

**Call Stack:**
```
main()
  └──> UEqn.H
        ├──> fvm::div (convection term discretization)
        ├──> fvm::laplacian (diffusion term discretization)
        └──> fvMatrix::solve (linear system solver)
```

**Code:**
```cpp
// 1. Construct momentum equation
// สร้างสมการโมเมนตัม
// fvMatrix<vector>: Matrix representing discretized momentum equation
tmp<fvVectorMatrix> UEqn
(
    // Convection term: ∇·(φU) - upwind scheme
    // พจน์การพา: ∇·(φU) - ใช้รูปแบบ upwind
    fvm::div(phi, U)
    
    // Diffusion term: ∇·(ν∇U) - Gauss linear
    // พจน์การแพร่: ∇·(ν∇U) - ใช้รูปแบบ Gauss linear
  + fvm::laplacian(nu, U)
);

// 2. Add under-relaxation if needed
// เพิ่ม under-relaxation หากจำเป็น
// Under-relaxation stabilizes iterative solution
UEqn().relax();

// 3. Solve momentum equation (optional in SIMPLE)
// แก้สมการโมเมนตัม (ไม่บังคับใน SIMPLE algorithm)
// In SIMPLE: momentum is solved after pressure correction
// solve(UEqn() == -fvc::grad(p));
```

**Thai Explanation:**
- **fvm::div(phi, U)**: การเก็บตก (discretization) พจน์การพา (convection) แบบ implicit
  - ใช้รูปแบบ upwind เพื่อเสถียรภาพเชิงตัวเลข
  - φ คือ mass flux ที่ผิวเซลล์
- **fvm::laplacian(nu, U)**: การเก็บตกพจน์การแพร่ (diffusion) แบบ implicit
  - ν คือ kinematic viscosity
  - ใช้รูปแบบ Gauss linear สำหรับการคำนวณ gradient
- **UEqn().relax()**: ใช้ under-relaxation เพื่อช่วยให้ iterative solution ลู่เข้า
- **fvVectorMatrix**: เมทริกซ์สำหรับระบบสมการเชิงเส้น [A]x = b

**Mathematical Form:**
$$\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot (\nu \nabla \mathbf{U})$$

Discretized as:
$$[A]\mathbf{U} = \mathbf{H} - \nabla p$$

Where:
- $[A]$: Coefficient matrix (convection + diffusion)
- $\mathbf{U}$: Velocity vector
- $\mathbf{H}$: Source term (off-diagonal coefficients)
- $\nabla p$: Pressure gradient

**Purpose:** Solves the pressure equation to enforce mass conservation (continuity)

**Call Stack:**
```
main()

        ├──> fvc::grad (pressure gradient)
        ├──> fvm::laplacian (pressure Poisson equation)
        ├──> fvScalarMatrix::solve (linear solver)
        └──> fvc::flux (correct flux field)
```

│         │  │surfaceScalar │   │  phi (flux)            │
│         │  └──────────────┘   │                        │
│         └──────────┬──────────┘                        │
│                    │                                    │
│         ┌──────────▼──────────┐                        │
│         │     fvMatrix        │                        │
│         │  ┌──────────────┐   │                        │
│         │  │ lduMatrix    │   │  Coefficient matrix    │
│         │  │ source_      │   │  RHS vector            │
│         │  │ psi_         │   │  Solution field        │
│         │  └──────────────┘   │                        │
│         └────────────────────┘                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Thai Explanation:**
- **fvMesh**: เก็บข้อมูลโครงสร้างเมช (จุด, หน้า, เซลล์)
- **Time**: จัดการข้อมูลเวลาและการเขียนผลลัพธ์
- **GeometricField**: คลาสฐานสำหรับสนามทุกประเภท
- **fvMatrix**: เมทริกซ์สำหรับระบบสมการเชิงเส้น
