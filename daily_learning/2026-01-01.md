## Part 1: Governing Equations - Theory & Implementation

### 1.1 Continuity Equation (Mass Conservation)

**Mathematical Form:**
```
∂ρ/∂t + ∇·(ρU) = 0
```

สำหรับ incompressible flow (ρ = constant):
```
∇·U = 0
```

**OpenFOAM Implementation:**
```cpp
// Continuity ถูก enforce ผ่าน pressure equation
// ดูที่: $FOAM_SRC/finiteVolume/cfdTools/general/adjustPhi/adjustPhi.C
// สมการ continuity ถูกแก้ผ่าน pressure correction:
// ∇²p = ∇·(U*)/dt  สำหรับ PISO/SIMPLE
```

### 1.2 Momentum Equation (Navier-Stokes)

**Mathematical Form:**
```
∂(ρU)/∂t + ∇·(ρUU) = -∇p + ∇·(μ∇U) + ρg
```

**OpenFOAM Implementation:**
```cpp
// จาก simpleFoam/pimpleFoam solver
tmp<fvVectorMatrix> tUEqn
(
    fvm::ddt(U)              // Unsteady term
  + fvm::div(phi, U)         // Convection term
  + fvm::laplacian(nu, U)    // Diffusion term
 ==
    fvOptions(U)             // Source terms
);
```

### Task 2: Custom BC - rampedFixedValue

**Header:** `rampedFixedValueFvPatchField.H`
```cpp
template<class Type>
class rampedFixedValueFvPatchField
:
    public fixedValueFvPatchField<Type>
{
    Type startValue_;
    Type endValue_;
    scalar duration_;
    
public:
    TypeName("rampedFixedValue");
    virtual void updateCoeffs();
};
```

**Implementation:** `rampedFixedValueFvPatchField.C`
```cpp
template<class Type>
void rampedFixedValueFvPatchField<Type>::updateCoeffs()
{
    if (this->updated()) return;
    
    scalar t = this->db().time().value();
    scalar ramp = min(t / duration_, 1.0);
    Type value = startValue_ + ramp * (endValue_ - startValue_);
    
    Field<Type>::operator=(value);
    fixedValueFvPatchField<Type>::updateCoeffs();
}
```

**Usage in 0/U:**
```cpp
inlet
{
    type            rampedFixedValue;
    startValue      uniform (0 0 0);
    endValue        uniform (10 0 0);
    duration        5.0;
}
```

## Part 6: Concept Check Questions

### Q1: อธิบายความแตกต่างระหว่าง fvc และ fvm

**Answer:**
- `fvc` (finite volume calculus): Explicit operators, คำนวณค่าจาก field ที่มีอยู่
  - เช่น `fvc::grad(p)`, `fvc::div(phi)`
  - ใช้สำหรับ terms ที่ไม่ต้องการ implicit solving
  
- `fvm` (finite volume method): Implicit operators, สร้าง matrix สำหรับ solving
  - เช่น `fvm::ddt(U)`, `fvm::laplacian(nu, U)`
  - ใช้สำหรับ terms ที่ต้องการ implicit solving
  - สร้าง coefficients ใน matrix equation Ax = b

### Q2: GeometricField ประกอบด้วยส่วนใดบ้าง

**Answer:**
```
GeometricField<Type, PatchField, GeoMesh>
├── internalField_    (Field<Type>)        // Cell values
└── boundaryField_    (PtrList<PatchField>) // Boundary values
    ├── [0] PatchField for patch 0
    ├── [1] PatchField for patch 1
    └── ...
```

### Q3: ขั้นตอน PISO algorithm ทำอย่างไร

**Answer:**
1. Solve momentum predictor: `solve(UEqn == -grad(p))`
2. Loop `nCorrectors` times:
   a. Calculate `HbyA = UEqn.H() / UEqn.A()`
   b. Solve pressure: `laplacian(rAU, p) == div(phiHbyA)`
   c. Correct flux: `phi -= pEqn.flux()`
   d. Correct velocity: `U = HbyA - rAU * grad(p)`
3. Update turbulence model

### Q4: แตกต่างระหว่าง volScalarField และ surfaceScalarField

**Answer:**
- `volScalarField`: Values at cell centers (GeometricField<scalar, fvPatchField, volMesh>)
  - ใช้สำหรับ pressure, temperature, etc.
  - จัดเก็บที่ cell centers
  
- `surfaceScalarField`: Values at face centers (GeometricField<scalar, fvsPatchField, surfaceMesh>)
  - ใช้สำหรับ flux (phi = U·Sf)
  - จัดเก็บที่ face centers
  - จำเป็นสำหรับ finite volume method

### Q5: อธิบายการทำงานของ fvm::ddt(U)

**Answer:**
```cpp
// Euler scheme (first-order)
fvm::ddt(U) = (U^n - U^(n-1)) / dt

// backward scheme (second-order)
fvm::ddt(U) = (1.5*U^n - 2*U^(n-1) + 0.5*U^(n-2)) / dt

// Source: $FOAM_SRC/finiteVolume/finiteVolume/fvm/fvmDdt/fvmDdt.C
// สร้าง diagonal coefficients ใน matrix:
// diag_coeff = V_cell / dt
// source_term = V_cell * U^(n-1) / dt
```
