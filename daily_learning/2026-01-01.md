## 1. Theory: Core Equations & Physics

### 1.1 Conservation of Mass (Continuity Equation)

**สมการการอนุรักษ์มวล (Continuity Equation)**

$$
\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{U}) = 0
$$

**คำอธิบายพจน์:**
- $\rho$ (rho) = **ความหนาแน่น** (density) [kg/m³]
- $\mathbf{U}$ = **สนามความเร็ว** (velocity field) [m/s]
- $\nabla \cdot$ = **ตัวดำเนินการไดเวอร์เจนซ์** (divergence operator)
- $t$ = **เวลา** (time) [s]

สำหรับการไหลแบบ **ไม่อัดตัว (incompressible flow)**: $\nabla \cdot \mathbf{U} = 0$

---

### 1.2 Conservation of Momentum (Navier-Stokes Equation)

**สมการการอนุรักษ์โมเมนตัม (Momentum Equation)**

$$
\frac{\partial (\rho \mathbf{U})}{\partial t} + \nabla \cdot (\rho \mathbf{U} \mathbf{U}) = -\nabla p + \nabla \cdot \boldsymbol{\tau} + \rho \mathbf{g}
$$

**คำอธิบายพจน์:**
- $p$ = **ความดัน** (pressure) [Pa]
- $\boldsymbol{\tau}$ (tau) = **เทนเซอร์ความเค้น** (stress tensor) [Pa]
- $\mathbf{g}$ = **ความเร่งเนื่องจากแรงโน้มถ่วง** (gravitational acceleration) [m/s²]
- $\nabla p$ = **แรงตามแนวปกติของความดัน** (pressure gradient force)
- $\nabla \cdot \boldsymbol{\tau}$ = **แรงเหนียว** (viscous force)

**สำหรับของไหลนิวตัน (Newtonian Fluid):**

$$
\boldsymbol{\tau} = \mu \left[ \nabla \mathbf{U} + (\nabla \mathbf{U})^T \right] - \frac{2}{3}\mu (\nabla \cdot \mathbf{U})\mathbf{I}
$$

- $\mu$ (mu) = **ความหนืดพลศาสตร์** (dynamic viscosity) [Pa·s]
- $\mathbf{I}$ = **เทนเซอร์เอกลักษณ์** (identity tensor)

---

### 1.3 Conservation of Energy

**สมการการอนุรักษ์พลังงาน (Energy Equation)**

$$
\frac{\partial (\rho e)}{\partial t} + \nabla \cdot (\rho e \mathbf{U}) = -\nabla \cdot \mathbf{q} + \boldsymbol{\tau} : \nabla \mathbf{U} + S
$$

**คำอธิบายพจน์:**
- $e$ = **พลังงานภายในต่อหน่วยมวล** (specific internal energy) [J/kg]
- $\mathbf{q}$ = **ฟลักซ์ความร้อน** (heat flux) [W/m²]
- $S$ = **แหล่งกำเนิดพลังงาน** (energy source) [W/m³]
- $\boldsymbol{\tau} : \nabla \mathbf{U}$ = **การกระจายพลังงานเนื่องจากความเค้น** (viscous dissipation)

**กฎของฟูริเยร์ (Fourier's Law):**

$$
\mathbf{q} = -k \nabla T
$$

- $k$ = **สัมประสิทธิภาพการนำความร้อน** (thermal conductivity) [W/(m·K)]
- $T$ = **อุณหภูมิ** (temperature) [K]

---

### 1.4 Equation of State

**สมการสถานะ (Equation of State)**

สำหรับ **แก๊สอุดมคติ (Ideal Gas)**:

$$
p = \rho R T
$$

- $R$ = **ค่าคงที่ของแก๊สเฉพาะ** (specific gas constant) [J/(kg·K)]

สำหรับ **ของไหลที่ไม่อัดตัว (Incompressible Fluid)**:

$$
\rho = \text{constant}
$$

---

### 1.5 Transport Equations (General Form)

**รูปแบบทั่วไปของสมการเคลื่อนที่ (General Transport Equation)**

$$
\frac{\partial (\rho \phi)}{\partial t} + \nabla \cdot (\rho \phi \mathbf{U}) = \nabla \cdot (\Gamma_\phi \nabla \phi) + S_\phi
$$

**คำอธิบายพจน์:**
- $\phi$ (phi) = **ตัวแปรขนส่ง** (transported variable) - สามารถเป็นสเกลาร์หรือเวกเตอร์
- $\Gamma_\phi$ (Gamma) = **สัมประสิทธิภาพการแพร่** (diffusion coefficient)
- $S_\phi$ = **เทอมแหล่งกำเนิด/เก็บกิน** (source/sink term)

**องค์ประกอบของสมการ:**
1. **เทอมไม่คงที่ (Unsteady term)**: $\frac{\partial (\rho \phi)}{\partial t}$
2. **เทอมเนื่องจากการนำ (Convection term)**: $\nabla \cdot (\rho \phi \mathbf{U})$
3. **เทอมเนื่องจากการแพร่ (Diffusion term)**: $\nabla \cdot (\Gamma_\phi \nabla \phi)$
4. **เทอมแหล่งกำเนิด (Source term)**: $S_\phi$

## 3. Code Walkthrough

### 3.1 UEqn.H

ไฟล์ `UEqn.H` ใช้สำหรับสร้างสมการโมเมนตัมใน OpenFOAM solver ต่างๆ (เช่น `simpleFoam`, `pimpleFoam`)

**โครงสร้างพื้นฐาน:**

```cpp
// สร้างสมการเมทริกซ์สำหรับสมการโมเมนตัม
tmp<fvVectorMatrix> UEqn
(
    fvm::div(phi, U)              // เทอมการนำ (Convection)
  + fvm::laplacian(nu, U)         // เทอมการแพร่ (Diffusion/Viscous)
  - fvc::div(nu*dev(T(fvc::grad(U)))) // เทอมความเค้นสำหรับของไหลที่ไม่อัดตัว
);

// แก้สมการโมเมนตัม
UEqn.solve();

// ปรับปรุงแรงดันจากสมการโมเมนตัม
if (simple.momentumPredictor())
{
    U = UEqn.H()/A(U);
    U.correctBoundaryConditions();
}
```

**คำอธิบาย:**
- `fvm::div(phi, U)` - เทอมการนำ (convection term) ใช้ implicit discretization
- `fvm::laplacian(nu, U)` - เทอมความหนืด (viscous term) ใช้ implicit discretization
- `fvc::div(...)` - เทอมการกระจายความเค้น (stress divergence) ใช้ explicit discretization
- `UEqn.solve()` - แก้ระบบสมการเชิงเส้น
- `UEqn.H()` - ดึงส่วน H (explicit part) ของเมทริกซ์
- `A(U)` - ดึงส่วน A (diagonal coefficients) ของเมทริกซ์

**ตัวอย่างเพิ่มเติมสำหรับแรงโน้มถ่วง:**

```cpp
// เพิ่มเทอมแรงโน้มถ่วง
tmp<fvVectorMatrix> UEqn
(
    fvm::div(phi, U)
  + fvm::laplacian(nu, U)
 ==
    fvOptions(U)  // แหล่งกำเนิดเพิ่มเติม (เช่น แรงต้าน)
);

UEqn.relax();  // การผ่อนคลายสมการ (under-relaxation)
```

### 3.2 pEqn.H

ไฟล์ `pEqn.H` ใช้สำหรับสร้างและแก้สมการความดัน (pressure equation) ใน OpenFOAM solver ต่างๆ (เช่น `simpleFoam`, `pimpleFoam`)

**โครงสร้างพื้นฐาน:**

```cpp
// สร้างสมการความดันจากสมการโมเมนตัม
volScalarField rAU(1.0/UEqn.A());
volVectorField HbyA(constrainHbyA(rAU*UEqn.H(), U, p));
surfaceScalarField phiHbyA("phiHbyA", fvc::flux(HbyA));

// ปรับปรุงเงื่อนไขขอบเขตสำหรับ flux
adjustPhi(phiHbyA, U, p);

// สร้างสมการความดัน
tmp<fvScalarMatrix> pEqn
(
    fvm::laplacian(rAU, p) == fvc::div(phiHbyA)
);

// แก้สมการความดัน
pEqn.solve();

// แก้ไข flux และความเร็วจากความดันใหม่
phi = phiHbyA - pEqn.flux();
U = HbyA - rAU*fvc::grad(p);
U.correctBoundaryConditions();
```

**คำอธิบาย:**
- `rAU` = ค่าผกผันของสัมประสิทธิ์เชิงแนวทแยง (inverse diagonal coefficients)
- `HbyA` = สนามความเร็วชั่วคราวจากสมการโมเมนตัม (intermediate velocity)
- `phiHbyA` = flux ชั่วคราวที่ผิวขอบเขต
- `fvm::laplacian(rAU, p)` = เทอมการแพร่ของความดัน (implicit)
- `fvc::div(phiHbyA)` = เทอมไดเวอร์เจนซ์ของ flux (explicit)
- `pEqn.solve()` = แก้สมการความดัน
- `pEqn.flux()` = ดึง flux จากสมการความดันที่แก้แล้ว

**ตัวอย่างสำหรับ SIMPLE algorithm:**

```cpp
// การผ่อนคลายสมการความดัน (under-relaxation)
pEqn.relax();

// แก้ไข flux ภายในโดเมน
phi = phiHbyA + pEqn.flux()/rAU;

// แก้ไขสนามความเร็ว
U = HbyA - rAU*fvc::grad(p);
U.correctBoundaryConditions();

// ทำให้ flux สอดคล้องกับเงื่อนไขการอนุรักษ์มวล
#include "continuityErrs.H"
```
