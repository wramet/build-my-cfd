# Governing Equations
## CFD Engine Development - 2026-01-01

---

## Learning Objectives

After this lesson, you will be able to:
- **Derive** the complete set of governing equations for two-phase refrigerant flow, including the critical expansion source term in the continuity equation ($\nabla \cdot U = \dot{m}(1/\rho_v - 1/\rho_l)$)
- **Implement** the Navier-Stokes equations with phase change mass transfer using the Lee model for evaporation ($\dot{m} = r_e \alpha_l \rho_l (T - T_{sat}) / T_{sat}$)
- **Design** a pressure-velocity coupling algorithm (SIMPLE/PISO) that properly handles the volumetric expansion term to prevent solver divergence
- **Integrate** CoolProp property tables with bilinear interpolation for real-time R410A/R32 thermodynamic properties during simulation
- **Validate** the implementation against analytical test cases (e.g., 1D evaporation in a heated tube) before applying to complex 3D geometries
