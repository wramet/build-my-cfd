## 📝 Format ของแต่ละวัน

แต่ละไฟล์รายวันประกอบด้วย:

1. **🎯 Learning Objectives** — เป้าหมายวันนี้
2. **💡 Key Concepts** — แนวคิดสำคัญ (สรุป)
3. **📖 Deep Dive** — เนื้อหาเชิงลึก (copy มาจาก source)
4. **🔧 Hands-on Practice** — ลองทำ
5. **🧠 Concept Check** — ทดสอบความเข้าใจ
6. **📚 Further Reading** — อ่านเพิ่มเติม (links)
7. **✅ Checklist** — สิ่งที่ต้องทำวันนี้
