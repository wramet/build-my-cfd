## 27.1 Mini-Project Structure and Requirements

### Introduction to Sparse Matrix Representations in CFD

### 27.1.1 Problem Definition: 2D Steady-State Heat Conduction

We'll solve the 2D heat equation:

$$
-\nabla \cdot (k \nabla T) = q
$$

For constant thermal conductivity $k$ and uniform grid spacing:

$$
-\left[\frac{T_{i-1,j} - 2T_{i,j} + T_{i+1,j}}{\Delta x^2} + \frac{T_{i,j-1} - 2T_{i,j} + T_{i,j+1}}{\Delta y^2}\right] = \frac{q}{k}
$$

This results in a sparse linear system $\mathbf{A}\mathbf{T} = \mathbf{b}$ where:

$$
\mathbf{A} =
\begin{bmatrix}
4 & -1 & 0 & -1 & 0 & \cdots \\
-1 & 4 & -1 & 0 & -1 & \cdots \\
0 & -1 & 4 & -1 & 0 & \cdots \\
-1 & 0 & -1 & 4 & -1 & \cdots \\
0 & -1 & 0 & -1 & 4 & \cdots \\
\vdots & \vdots & \vdots & \vdots & \vdots & \ddots
\end{bmatrix}
$$

### Mathematical Derivation of Gauss-Seidel Method

The Gauss-Seidel method is an iterative solver that uses the LDU decomposition implicitly. Starting from:

$$
A\mathbf{x} = \mathbf{b}
$$

We write:

$$
(L + D + U)\mathbf{x} = \mathbf{b}
$$

Rearranging for the update:

$$
L\mathbf{x}^{(k+1)} + D\mathbf{x}^{(k+1)} + U\mathbf{x}^{(k)} = \mathbf{b}
$$

Solving for $\mathbf{x}^{(k+1)}$:

$$
\mathbf{x}^{(k+1)} = D^{-1}(\mathbf{b} - L\mathbf{x}^{(k+1)} - U\mathbf{x}^{(k)})
$$

In component form:

$$
x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j=1}^{i-1} a_{ij}x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij}x_j^{(k)}\right)
$$

### Convergence Analysis

The convergence rate of Gauss-Seidel depends on the spectral radius of the iteration matrix:

$$
T_G = -(D + L)^{-1}U
$$

The method converges if $\rho(T_G) < 1$, where $\rho$ denotes the spectral radius.

For symmetric positive definite matrices, Gauss-Seidel converges for any initial guess.

### Jacobi Method Comparison

The Jacobi method uses the previous iteration's values for all updates:

$$
x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j \neq i} a_{ij}x_j^{(k)}\right)
$$

While simpler to implement, Jacobi typically converges slower than Gauss-Seidel for the same problem.

### Memory Requirements Analysis

For a sparse matrix with $nnz$ non-zero entries, the storage requirements are:

- **CSR format**: $(nnz + n + 1) \times$ sizeof(int) + $nnz \times$ sizeof(double)
- **CSC format**: Same as CSR
- **COO format**: $3 \times nnz \times$ sizeof(int/double)

For our 10,000-cell grid example:
- CSR: ~50,000 entries = ~400 KB (using doubles)
- Dense: 10,000 × 10,000 = 800 MB (1000x more memory!)

### 27.2.2 2D Mesh Generation Implementation

Creating structured 2D mesh connectivity:

```cpp
// File: lduAddressing.C
#include "lduAddressing.H"
#include <iostream>

namespace Foam {

autoPtr<lduAddressing> lduAddressing::create2DMesh
(
    const label nx,
    const label ny,
    const scalar delta
)
{
    label nCells = nx * ny;
    label nFaces = 2 * nx * ny - nx - ny;  // Internal faces only

    auto addressing = autoPtr<lduAddressing>(new lduAddressing(nCells, nFaces));

    // Fill connectivity arrays
    label facei = 0;

    // Horizontal faces (connecting cells in y-direction)
    for (label j = 0; j < ny; ++j)
    {
        for (label i = 0; i < nx; ++i)
        {
            if (j < ny - 1)  // Not on top boundary
            {
                label own = j * nx + i;           // Current cell
                label nei = (j + 1) * nx + i;    // Cell above

                addressing->owner_[facei] = own;
                addressing->neighbour_[facei] = nei;
                ++facei;
            }
        }
    }

    // Vertical faces (connecting cells in x-direction)
    for (label j = 0; j < ny; ++j)
    {
        for (label i = 0; i < nx; ++i)
        {
            if (i < nx - 1)  // Not on right boundary
            {
                label own = j * nx + i;           // Current cell
                label nei = j * nx + (i + 1);    // Cell to right

                addressing->owner_[facei] = own;
                addressing->neighbour_[facei] = nei;
                ++facei;
            }
        }
    }

    // Validate face count
    if (facei != nFaces)
    {
        std::cerr << "Error: Face count mismatch. Expected: " << nFaces
                  << ", Generated: " << facei << std::endl;
        return nullptr;
    }

    return addressing;
}

lduAddressing::lduAddressing(const label nCells, const label nFaces)
:
    nCells_(nCells),
    nFaces_(nFaces),
    owner_(nFaces),
    neighbour_(nFaces),
    lowerAddr_(nFaces),
    upperAddr_(nFaces)
{
    // Initialize addressing arrays
    for (label facei = 0; facei < nFaces; ++facei)
    {
        lowerAddr_[facei] = facei;  // Face index for lower triangle
        upperAddr_[facei] = facei;  // Face index for upper triangle
    }
}

void lduAddressing::printConnectivity() const
{
    std::cout << "=== Mesh Connectivity (lduAddressing) ===\n";
    std::cout << "Cells: " << nCells_ << ", Faces: " << nFaces_ << "\n\n";

    std::cout << "Face  | Owner | Neighbour\n";
    std::cout << "------|-------|----------\n";

    for (label facei = 0; facei < nFaces_; ++facei)
    {
        std::cout << facei << "     | " << owner_[facei]
                  << "     | " << neighbour_[facei] << "\n";
    }
}

void lduAddressing::validateConnectivity() const
{
    std::cout << "=== Validating Mesh Connectivity ===\n";

    // Check all owner/neighbour pairs are valid
    for (label facei = 0; facei < nFaces_; ++facei)
    {
        if (owner_[facei] < 0 || owner_[facei] >= nCells_)
        {
            std::cerr << "Error: Invalid owner " << owner_[facei]
                      << " for face " << facei << "\n";
        }

        if (neighbour_[facei] < 0 || neighbour_[facei] >= nCells_)
        {
            std::cerr << "Error: Invalid neighbour " << neighbour_[facei]
                      << " for face " << facei << "\n";
        }

        if (owner_[facei] == neighbour_[facei])
        {
            std::cerr << "Error: Face " << facei << " connects cell to itself\n";
        }
    }

    // Check for duplicate faces
    for (label facei = 0; facei < nFaces_; ++facei)
    {
        for (label facej = facei + 1; facej < nFaces_; ++facej)
        {
            // Check if faces connect same cells (in either order)
            bool sameConnection =
                (owner_[facei] == owner_[facej] && neighbour_[facei] == neighbour_[facej]) ||
                (owner_[facei] == neighbour_[facej] && neighbour_[facei] == owner_[facej]);

            if (sameConnection)
            {
                std::cerr << "Error: Duplicate connection found: Face " << facei
                          << " and " << facej << "\n";
            }
        }
    }

    std::cout << "Connectivity validation complete.\n";
}

} // namespace Foam
```

```cpp
// Constructor: Zero matrix
template<class Type>
lduMatrix<Type>::lduMatrix(const lduAddressing& addr)
:
    diagonal_(addr.nCells(), Type()),
    lower_(addr.nFaces(), Type()),
    upper_(addr.nFaces(), Type()),
    addressing_(addr)

    zero();

// Constructor: Initialize with value
template<class Type>
lduMatrix<Type>::lduMatrix(const lduAddressing& addr, const Type& value)
:
    diagonal_(addr.nCells(), value),
    lower_(addr.nFaces(), Type()),  // Off-diagonal starts at zero
    upper_(addr.nFaces(), Type()),
    addressing_(addr)
{
    // Phase 2: Zero off-diagonals
    for (label i = 0; i < addr.nFaces(); ++i)
    {
        lower_[i] = Type();
        upper_[i] = Type();
    }
}

// Matrix assembly - Phase 2 compliant indexing
template<class Type>
void lduMatrix<Type>::add(const label row, const label col, const Type& value)
{
    if (row == col)
    {
        // Diagonal element
        diagonal_[row] += value;
    }
    else if (col > row)
    {
        // Upper triangular element
        label facei = col - row - 1;  // Simplified for structured grid
        if (facei < addressing_.nFaces())
        {
            upper_[facei] += value;
        }
    }
    else
    {
        // Lower triangular element
        label facei = row - col - 1;  // Simplified for structured grid
        if (facei < addressing_.nFaces())
        {
            lower_[facei] += value;
        }
    }
}

template<class Type>
void lduMatrix<Type>::addDiag(const label i, const Type& value)
{
    diagonal_[i] += value;
}

template<class Type>
void lduMatrix<Type>::addLower(const label facei, const Type& value)
{
    if (facei >= 0 && facei < addressing_.nFaces())
    {
        lower_[facei] += value;
    }
}

template<class Type>
void lduMatrix<Type>::addUpper(const label facei, const Type& value)
{
    if (facei >= 0 && facei < addressing_.nFaces())
    {
        upper_[facei] += value;
    }
}

void lduMatrix<Type>::operator=(const lduMatrix<Type>& rhs)
{
    diagonal_ = rhs.diagonal_;
    lower_ = rhs.lower_;
    upper_ = rhs.upper_;
}

template<class Type>
void lduMatrix<Type>::operator+=(const lduMatrix<Type>& rhs)
{
    diagonal_ += rhs.diagonal_;
    lower_ += rhs.lower_;
    upper_ += rhs.upper_;
}

template<class Type>
void lduMatrix<Type>::validate() const
{
    // Check matrix properties
    Type sumDiag = Type();
    Type sumLower = Type();
    Type sumUpper = Type();

    for (label i = 0; i < size(); ++i)
    {
        sumDiag += diagonal_[i];
    }

    for (label i = 0; i < nFaces(); ++i)
    {
        sumLower += lower_[i];
        sumUpper += upper_[i];
    }

    std::cout << "Matrix validation:\n";
    std::cout << "  Diagonal sum: " << sumDiag << "\n";
    std::cout << "  Lower sum: " << sumLower << "\n";
    std::cout << "  Upper sum: " << sumUpper << "\n";

    // For symmetric matrices, lower and upper should be equal
    if (std::abs(sumLower - sumUpper) > 1e-10)
    {
        std::cerr << "⚠️ Warning: Matrix may not be symmetric\n";
    }
}

template<class Type>
scalar lduMatrix<Type>::norm() const
{
    scalar maxNorm = 0.0;

    for (label i = 0; i < size(); ++i)
    {
        scalar rowNorm = std::abs(diagonal_[i]);

        // Add contribution from off-diagonal elements
        for (label j = 0; j < nFaces(); ++j)
        {
            rowNorm += std::abs(lower_[j]) + std::abs(upper_[j]);
        }

        if (rowNorm > maxNorm)
            maxNorm = rowNorm;
    }

    return maxNorm;
}

template<class Type>
label lduMatrix<Type>::nnz() const
{
    label count = 0;

    // Count diagonal (all non-zero)
    count += size();

    // Count lower and upper triangular
    for (label i = 0; i < nFaces(); ++i)
    {
        if (std::abs(lower_[i]) > 1e-15) count++;
        if (std::abs(upper_[i]) > 1e-15) count++;
    }

    return count;
}

// Factory method: Create Laplacian matrix
template<class Type>
autoPtr<lduMatrix<Type>> lduMatrix<Type>::createLaplacian
(
    const lduAddressing& addr,
    const scalar delta
)
{
    auto matrix = autoPtr<lduMatrix<Type>>(new lduMatrix<Type>(addr));

    // For each face, contribute to Laplacian stencil
    const labelList& owner = addr.owner();
    const labelList& neighbour = addr.neighbour();

    scalar coeff = 1.0 / (delta * delta);

    for (label facei = 0; facei < addr.nFaces(); ++facei)
    {
        label own = owner[facei];
        label nei = neighbour[facei];

        // Add off-diagonal contribution (negative)
        matrix->add(own, nei, -coeff);
        matrix->add(nei, own, -coeff);

        // Add to diagonal (positive)
        matrix->addDiag(own, coeff);
        matrix->addDiag(nei, coeff);
    }

    return matrix;
}
```

```cpp
// Main Amul operation - Sparse matrix-vector multiplication
template<class Type>
void lduMatrixOps<Type>::Amul
(
    Field<Type>& result,
    const Field<Type>& source,
    const scalar alpha,
    const scalar beta
) const
{
    const label size = matrix_.size();
    const label nFaces = matrix_.nFaces();

    // Phase 2: Proper boundary checks
    assert(result.size() == size);
    assert(source.size() == size);

    if (beta == 0.0)
    {
        // Result = alpha * A * source
        for (label i = 0; i < size; ++i)
        {
            result[i] = alpha * matrix_.diag()[i] * source[i];

            // Add lower triangular contributions
            for (label j = 0; j < nFaces; ++j)
            {
                // This is simplified - in real implementation,
                // we'd need proper addressing lookup
                result[i] += alpha * matrix_.lower()[j] * source[j];
                result[i] += alpha * matrix_.upper()[j] * source[j];
            }
        }
    }
    else
    {
        // Result = beta * result + alpha * A * source
        for (label i = 0; i < size; ++i)
        {
            result[i] = beta * result[i] + alpha * matrix_.diag()[i] * source[i];

            // Add off-diagonal contributions
            for (label j = 0; j < nFaces; ++j)
            {
                result[i] += alpha * matrix_.lower()[j] * source[j];
                result[i] += alpha * matrix_.upper()[j] * source[j];
            }
        }
    }
}

// Hermitian transpose multiplication (A^T operation)
template<class Type>
void lduMatrixOps<Type>::HAmul
(
    Field<Type>& result,
    const Field<Type>& source,
    const scalar alpha,
    const scalar beta
) const
{
    const label size = matrix_.size();
    const label nFaces = matrix_.nFaces();

    assert(result.size() == size);
    assert(source.size() == size);

    if (beta == 0.0)
    {
        // Result = alpha * A^T * source
        for (label i = 0; i < size; ++i)
        {
            result[i] = alpha * matrix_.diag()[i] * source[i];

            // Add contributions from transpose (swap lower/upper roles)
            for (label j = 0; j < nFaces; ++j)
            {
                result[i] += alpha * matrix_.upper()[j] * source[j];
                result[i] += alpha * matrix_.lower()[j] * source[j];
            }
        }
    }
    else
    {
        // Result = beta * result + alpha * A^T * source
        for (label i = 0; i < size; ++i)
        {
            result[i] = beta * result[i] + alpha * matrix_.diag()[i] * source[i];

            for (label j = 0; j < nFaces; ++j)
            {
                result[i] += alpha * matrix_.upper()[j] * source[j];
                result[i] += alpha * matrix_.lower()[j] * source[j];
            }
        }
    }
}

// Forward/backward substitution (for triangular solves)
template<class Type>
void lduMatrixOps<Type>::LUsolve
(
    Field<Type>& x,
    const Field<Type>& b
) const
{
    const label size = matrix_.size();
    const label nFaces = matrix_.nFaces();

    assert(x.size() == size);
    assert(b.size() == size);

    // Forward substitution: L * y = b (assuming diagonal is 1 for L)
    Field<Type> y(size);

    for (label i = 0; i < size; ++i)
    {
        y[i] = b[i];

        // Subtract contributions from previous rows
        for (label j = 0; j < i; ++j)
        {
            // Simplified - need proper addressing for sparse L
            y[i] -= matrix_.lower()[i * nFaces + j] * y[j];
        }
    }

    // Backward substitution: U * x = y (assuming diagonal is 1 for U)
    for (label i = size - 1; i >= 0; --i)
    {
        x[i] = y[i];

        // Subtract contributions from later rows
        for (label j = i + 1; j < size; ++j)
        {
            // Simplified - need proper addressing for sparse U
            x[i] -= matrix_.upper()[i * nFaces + j] * x[j];
        }
    }
}

// Vector utility operations
template<class Type>
scalar lduMatrixOps<Type>::dotProduct
(
    const Field<Type>& a,
    const Field<Type>& b
) const
{
    assert(a.size() == b.size());

    scalar result = 0.0;
    for (label i = 0; i < a.size(); ++i)
    {
        result += a[i] * b[i];
    }

    return result;
}

template<class Type>
scalar lduMatrixOps<Type>::norm(const Field<Type>& field) const
{
    scalar sumSquares = 0.0;
    for (label i = 0; i < field.size(); ++i)
    {
        sumSquares += field[i] * field[i];
    }

    return std::sqrt(sumSquares);
}
```

```cpp
#include "laplacianAssembly.H"
#include <iostream>
#include <cmath>

namespace Foam {
namespace laplacianAssembly {

// Constructor - Complete problem setup
template<class Type>
HeatConductionProblem<Type>::HeatConductionProblem
(
    const label nx,
    const label ny,
    const scalar delta,
    const scalar k,
    const scalar source
)
:
    delta_(delta),
    k_(k),
    source_(source)
{
    // Create mesh addressing
    addressing_ = lduAddressing::create2DMesh(nx, ny, delta);
    if (!addressing_)
    {
        std::cerr << "❌ Failed to create mesh addressing\n";
        return;
    }

    // Create matrix and operations
    matrix_ = autoPtr<lduMatrix<Type>>(new lduMatrix<Type>(*addressing_));
    ops_ = autoPtr<lduMatrixOps<Type>>(new lduMatrixOps<Type>(*matrix_));

    // Set default boundary conditions (zero gradient)
    for (int i = 0; i < 4; ++i)
    {
        bc_[i] = BoundaryCondition<Type>(BCType::ZERO_GRADIENT, Type());
    }
}

// Boundary condition setup methods
template<class Type>
void HeatConductionProblem<Type>::setLeftBC(BCType type, Type value)
{
    bc_[0] = BoundaryCondition<Type>(type, value);
}

template<class Type>
void HeatConductionProblem<Type>::setRightBC(BCType type, Type value)
{

}

template<class Type>
void HeatConductionProblem<Type>::setBottomBC(BCType type, Type value)
{
    bc_[2] = BoundaryCondition<Type>(type, value);
}

template<class Type>
void HeatConductionProblem<Type>::setTopBC(BCType type, Type value)
{
    bc_[3] = BoundaryCondition<Type>(type, value);
}

// Complete matrix assembly with boundary conditions
template<class Type>
void HeatConductionProblem<Type>::assembleMatrix()
{
    // Reset matrix to zero
    matrix_->zero();

    const labelList& owner = addressing_->owner();
    const labelList& neighbour = addressing_->neighbour();

    // Discretization coefficient
    scalar coeff = k_ / (delta_ * delta_);

    // Face-based loop - core of FVM assembly
    for (label facei = 0; facei < addressing_->nFaces(); ++facei)
    {
        label own = owner[facei];
        label nei = neighbour[facei];

        // Off-diagonal contributions (negative for Laplacian)
        matrix_->add(own, nei, -coeff);
        matrix_->add(nei, own, -coeff);

        // Diagonal contributions (positive)
        matrix_->addDiag(own, coeff);
        matrix_->addDiag(nei, coeff);
    }

    // Apply boundary conditions
    label nx = addressing_->nCells() / addressing_->nCells();

    // Left boundary (x = 0)
    for (label j = 0; j < addressing_->nCells() / nx; ++j)
    {
        label cell = j * nx;  // Leftmost cell in row

        if (bc_[0].type == BCType::DIRICHLET)
        {
            // For Dirichlet BC: modify diagonal
            matrix_->addDiag(cell, coeff);
        }
        else if (bc_[0].type == BCType::NEUMANN)
        {
            // For Neumann BC: modify face coefficient
            // Find face to the right of boundary cell
            for (label facei = 0; facei < addressing_->nFaces(); ++facei)
            {
                if (owner[facei] == cell)
                {
                    matrix_->addDiag(cell, coeff);
                    break;
                }
            }
        }
    }

    // Right boundary (x = nx*delta)
    for (label j = 0; j < addressing_->nCells() / nx; ++j)
    {
        label cell = j * nx + (nx - 1);  // Rightmost cell in row

        if (bc_[1].type == BCType::DIRICHLET)
        {
            matrix_->addDiag(cell, coeff);
        }
        else if (bc_[1].type == BCType::NEUMANN)
        {
            for (label facei = 0; facei < addressing_->nFaces(); ++facei)
            {
                if (owner[facei] == cell)
                {
                    matrix_->addDiag(cell, coeff);
                    break;
                }
            }
        }
    }

    // Bottom boundary (y = 0)
    for (label i = 0; i < nx; ++i)
    {
        label cell = i;  // Bottom row

        if (bc_[2].type == BCType::DIRICHLET)
        {
            matrix_->addDiag(cell, coeff);
        }
        else if (bc_[2].type == BCType::NEUMANN)
        {
            for (label facei = 0; facei < addressing_->nFaces(); ++facei)
            {
                if (owner[facei] == cell)
                {
                    matrix_->addDiag(cell, coeff);
                    break;
                }
            }
        }
    }

    // Top boundary (y = ny*delta)
    for (label i = 0; i < nx; ++i)
    {
        label cell = (addressing_->nCells() / nx - 1) * nx + i;  // Top row

        if (bc_[3].type == BCType::DIRICHLET)
        {
            matrix_->addDiag(cell, coeff);
        }
        else if (bc_[3].type == BCType::NEUMANN)
        {
            for (label facei = 0; facei < addressing_->nFaces(); ++facei)
            {
                if (owner[facei] == cell)
                {
                    matrix_->addDiag(cell, coeff);
                    break;
                }
            }
        }
    }

    // Add source term to right-hand side (not to matrix)
    // This will be handled in the solver
}

// Residual calculation for convergence monitoring
template<class Type>
scalar HeatConductionProblem<Type>::computeResidual
(
    const Field<Type>& T,
    const Field<Type>& b
) const
{
    Field<Type> residual(size());

    // Compute residual: b - A*T
    ops_->Amul(residual, T, -1.0, 1.0);

    // Calculate L2 norm of residual
    return ops_->norm(residual);
}

template<class Type>
scalar HeatConductionProblem<Type>::maxResidual
(
    const Field<Type>& T,
    const Field<Type>& b
) const
{
    Field<Type> residual(size());
    ops_->Amul(residual, T, -1.0, 1.0);

    // Find maximum absolute residual
    scalar maxRes = 0.0;
    for (label i = 0; i < residual.size(); ++i)
    {
        scalar absRes = std::abs(residual[i]);
        if (absRes > maxRes) maxRes = absRes;
    }

    return maxRes;
}

// Create analytical solution for verification
template<class Type>
Field<Type> HeatConductionProblem<Type>::createAnalyticalSolution() const
{
    Field<Type> analytical(size());

    // For Laplace equation with zero source: T = 0 is trivial solution
    // For demonstration, create a simple analytical solution
    label nx = addressing_->nCells() / addressing_->nCells();

    for (label j = 0; j < addressing_->nCells() / nx; ++j)
    {
        for (label i = 0; i < nx; ++i)
```

```cpp
// File: /Users/woramet/Documents/Build My CFD/.claude/worktrees/upbeat-banach/daily_learning/Phase_02_DataStructures_Memory/ldu_matrix.H
#ifndef LDUMATRIX_H
#define LDUMATRIX_H

#include <iostream>
#include <vector>
#include <cmath>
#include <immintrin.h>
#include "aligned_field.H"

template<class Type>
class LDUMatrix {
private:
    // Sparse matrix storage
    std::vector<std::vector<Type>> rows_;  // Row storage
    std::vector<label> diagIndices_;      // Diagonal positions in each row
    std::vector<label> upperStart_;        // Start of upper triangular elements
    std::vector<label> lowerStart_;        // Start of lower triangular elements

    // Matrix properties
    label size_;
    label nnz_;  // Number of non-zeros

    // Convergence tracking
    Type tolerance_;
    label maxIterations_;

public:
    // Constructor pattern: ClassName(size, value, "name")
    LDUMatrix(label size, Type fillValue = Type(), const char* name = "ldu")
        : size_(size), nnz_(0), tolerance_(1e-10), maxIterations_(1000) {

        // Initialize storage
        rows_.resize(size);
        diagIndices_.resize(size);
        upperStart_.resize(size);
        lowerStart_.resize(size);

        // Fill with initial values
        for (label i = 0; i < size; ++i) {
            rows_[i].push_back(fillValue);  // Diagonal
            diagIndices_[i] = 0;
            upperStart_[i] = 1;
            lowerStart_[i] = 0;
        }

        nnz_ = size;  // Initially only diagonal
    }

    // Matrix assembly methods
    void addDiagonal(label i, const Type& value) {
        if (i >= size_) throw std::out_of_range("Row index out of bounds");
        rows_[i][diagIndices_[i]] = value;
    }

    void addUpper(label i, label j, const Type& value) {
        if (i >= size_ || j <= i) throw std::invalid_argument("Invalid upper triangular index");

        // Find insertion position
        auto pos = rows_[i].begin() + diagIndices_[i] + 1;
        while (pos < rows_[i].end() && j < (i + (pos - rows_[i].begin() - diagIndices_[i]))) {
            ++pos;
        }

        rows_[i].insert(pos, value);
        // Update indices for rows after this one
        for (label k = i + 1; k < size_; ++k) {
            upperStart_[k]++;
        }
        nnz_++;
    }

    void addLower(label i, label j, const Type& value) {
        if (i >= size_ || j >= i) throw std::invalid_argument("Invalid lower triangular index");

        // Insert at beginning of row
        rows_[i].insert(rows_[i].begin() + diagIndices_[i], value);
        // Update diagonal and upper indices
        diagIndices_[i]++;
        for (label k = i + 1; k < size_; ++k) {
            diagIndices_[k]++;
            upperStart_[k]++;
        }
        nnz_++;
    }

    // Access methods
    Type& operator[](label i) {
        if (i >= size_) throw std::out_of_range("Row index out of bounds");
        return rows_[i][diagIndices_[i]];  // Return diagonal element
    }

    const Type& operator[](label i) const {
        if (i >= size_) throw std::out_of_range("Row index out of bounds");
        return rows_[i][diagIndices_[i]];
    }

    // Matrix properties
    label size() const { return size_; }
    label nnz() const { return nnz_; }

    // Set convergence criteria
    void setTolerance(Type tol) { tolerance_ = tol; }
    void setMaxIterations(label maxIter) { maxIterations_ = maxIter; }

    // 1D Heat Equation Assembler
    void assembleHeatEquation(const Type& alpha, const Type& dx) {
        if (size_ < 3) throw std::invalid_argument("Matrix size too small for heat equation");

        Type coeff = alpha / (dx * dx);

        // Boundary conditions (Dirichlet)
        addDiagonal(0, 1.0);
        addDiagonal(size_ - 1, 1.0);

        // Interior points
        for (label i = 1; i < size_ - 1; ++i) {
            addDiagonal(i, -2.0 * coeff);
            addUpper(i, i + 1, coeff);
            addLower(i, i - 1, coeff);
        }
    }

    // Matrix-Vector Multiplication
    AlignedField<Type> multiply(const AlignedField<Type>& x) const {
        if (x.size() != size_) throw std::invalid_argument("Vector size mismatch");

        AlignedField<Type> result(size_, Type());

        for (label i = 0; i < size_; ++i) {
            Type sum = Type();

            // Lower triangular part
            for (label j = 0; j < diagIndices_[i]; ++j) {
                label col = i - (diagIndices_[i] - j);
                sum += rows_[i][j] * x[col];
            }

            // Diagonal
            sum += rows_[i][diagIndices_[i]] * x[i];

            // Upper triangular part
            for (label j = diagIndices_[i] + 1; j < rows_[i].size(); ++j) {
                label col = i + (j - diagIndices_[i]);
                sum += rows_[i][j] * x[col];
            }

            result[i] = sum;
        }

        return result;
    }

    // Dense matrix conversion for comparison
    std::vector<std::vector<Type>> toDense() const {
        std::vector<std::vector<Type>> dense(size_, std::vector<Type>(size_, Type()));

        for (label i = 0; i < size_; ++i) {
            // Fill lower triangular
            for (label j = 0; j < diagIndices_[i]; ++j) {
                label col = i - (diagIndices_[i] - j);
                dense[i][col] = rows_[i][j];
            }

            // Fill diagonal
            dense[i][i] = rows_[i][diagIndices_[i]];

            // Fill upper triangular
            for (label j = diagIndices_[i] + 1; j < rows_[i].size(); ++j) {
                label col = i + (j - diagIndices_[i]);
                dense[i][col] = rows_[i][j];
            }
        }

        return dense;
    }

    // Print matrix (for debugging)
    void print() const {
        std::cout << "LDU Matrix (" << size_ << "x" << size_ << ", nnz=" << nnz_ << "):\n";
        for (label i = 0; i < size_; ++i) {
            for (label j = 0; j < size_; ++j) {
                Type value = Type();

                // Find value at position (i,j)
                if (j < i) {
                    // Lower triangular
                    for (label k = 0; k < diagIndices_[i]; ++k) {
                        if (i - (diagIndices_[i] - k) == j) {
                            value = rows_[i][k];
                            break;
                        }
                    }
                } else if (j == i) {

                } else {
                    // Upper triangular
                    for (label k = diagIndices_[i] + 1; k < rows_[i].size(); ++k) {

                            value = rows_[i][k];
                            break;
                        }
                    }
                }

                std::cout << std::setw(10) << value << " ";
            }
            std::cout << "\n";
        }
    }
};
```

```cpp
#ifndef GAUSS_SEIDEL_H
#define GAUSS_SEIDEL_H

#include "ldu_matrix.H"
#include "aligned_field.H"
#include <iostream>

template<class Type>
class GaussSeidelSolver {
private:
    LDUMatrix<Type>& matrix_;
    Type tolerance_;
    label maxIterations_;

public:
    GaussSeidelSolver(LDUMatrix<Type>& mat, Type tol = 1e-10, label maxIter = 1000)
        : matrix_(mat), tolerance_(tol), maxIterations_(maxIter) {
        matrix_.setTolerance(tol);
        matrix_.setMaxIterations(maxIter);
    }

    // Solve Ax = b using Gauss-Seidel method
    bool solve(AlignedField<Type>& x, const AlignedField<Type>& b) {
        if (x.size() != matrix_.size() || b.size() != matrix_.size()) {
            throw std::invalid_argument("Vector size mismatch");
        }

        AlignedField<Type> x_old(x.size(), Type());
        Type residualNorm = Type();
        label iteration = 0;

        for (iteration = 0; iteration < maxIterations_; ++iteration) {
            // Store old solution
            for (label i = 0; i < x.size(); ++i) {
                x_old[i] = x[i];
            }

            // Gauss-Seidel update
            for (label i = 0; i < matrix_.size(); ++i) {
                Type sum = b[i];

                // Lower triangular part (using updated values)
                for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                    label col = i - (matrix_.diagIndices()[i] - j);
                    sum -= matrix_.rows()[i][j] * x[col];
                }

                // Diagonal
                sum -= matrix_.rows()[i][matrix_.diagIndices()[i]] * x[i];

                // Upper triangular part (using old values)
                for (label j = matrix_.diagIndices()[i] + 1; j < matrix_.rows()[i].size(); ++j) {
                    label col = i + (j - matrix_.diagIndices()[i]);
                    sum -= matrix_.rows()[i][j] * x_old[col];
                }

                // Update solution
                if (std::abs(matrix_.rows()[i][matrix_.diagIndices()[i]]) < 1e-12) {
                    std::cerr << "Warning: Zero diagonal at row " << i << std::endl;
                    x[i] = Type();  // Set to zero
                } else {
                    x[i] = sum / matrix_.rows()[i][matrix_.diagIndices()[i]];
                }
            }

            // Check convergence
            residualNorm = computeResidual(x, b);
            if (residualNorm < tolerance_) {
                break;
            }
        }

        std::cout << "Gauss-Seidel converged in " << iteration << " iterations\n";
        std::cout << "Final residual norm: " << residualNorm << "\n";

        return iteration < maxIterations_;
    }

private:
    Type computeResidual(const AlignedField<Type>& x, const AlignedField<Type>& b) {
        AlignedField<Type> residual = matrix_.multiply(x);
        Type norm = Type();

        for (label i = 0; i < residual.size(); ++i) {
            residual[i] = b[i] - residual[i];
            norm += residual[i] * residual[i];
        }

        return std::sqrt(norm);
    }
};
```

```cpp
#ifndef DENSE_SOLVER_H
#define DENSE_SOLVER_H

#include <vector>
#include <algorithm>

template<class Type>
class DenseSolver {
public:
    // Gaussian elimination with partial pivoting
    static bool solve(std::vector<std::vector<Type>>& A, std::vector<Type>& b) {
        const int n = A.size();

        // Forward elimination
        for (int i = 0; i < n; ++i) {
            // Find pivot
            int pivot_row = i;
            Type pivot_val = std::abs(A[i][i]);

            for (int j = i + 1; j < n; ++j) {
                if (std::abs(A[j][i]) > pivot_val) {
                    pivot_row = j;
                    pivot_val = std::abs(A[j][i]);
                }
            }

            // Swap rows if needed
            if (pivot_row != i) {
                std::swap(A[i], A[pivot_row]);
                std::swap(b[i], b[pivot_row]);
            }

            // Check for singular matrix
            if (std::abs(A[i][i]) < 1e-12) {
                return false;
            }

            // Eliminate column
            for (int j = i + 1; j < n; ++j) {
                Type factor = A[j][i] / A[i][i];
                for (int k = i; k < n; ++k) {
                    A[j][k] -= factor * A[i][k];
                }
                b[j] -= factor * b[i];
            }
        }

        // Back substitution
        for (int i = n - 1; i >= 0; --i) {
            for (int j = i + 1; j < n; ++j) {
                b[i] -= A[i][j] * b[j];
            }
            b[i] /= A[i][i];
        }

        return true;
    }

    // Matrix inverse
    static bool inverse(std::vector<std::vector<Type>>& A) {
        const int n = A.size();
        std::vector<std::vector<Type>> I(n, std::vector<Type>(n, Type()));

        // Create identity matrix
        for (int i = 0; i < n; ++i) {
            I[i][i] = 1.0;
        }

        // Solve for each column
        for (int j = 0; j < n; ++j) {
            if (!solve(A, I[j])) {
                return false;
            }
        }

        // Transpose to get inverse
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < n; ++j) {
                A[i][j] = I[j][i];
            }
        }

        return true;
    }

    // Matrix multiplication
    static std::vector<std::vector<Type>> multiply(
        const std::vector<std::vector<Type>>& A,
        const std::vector<std::vector<Type>>& B) {

        const int n = A.size();
        const int m = B[0].size();
        const int p = B.size();

        std::vector<std::vector<Type>> C(n, std::vector<Type>(m, Type()));

        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < m; ++j) {
                for (int k = 0; k < p; ++k) {
                    C[i][j] += A[i][k] * B[k][j];
                }
            }
        }

        return C;
    }
```

```cpp
template<class Type>
class SORSolver {
private:
    LDUMatrix<Type>& matrix_;
    Type omega_;  // Relaxation parameter (0 < omega < 2)
    Type tolerance_;
    label maxIterations_;

public:
    SORSolver(LDUMatrix<Type>& mat, Type omega = 1.5, Type tol = 1e-10, label maxIter = 1000)
        : matrix_(mat), omega_(omega), tolerance_(tol), maxIterations_(maxIter) {
        if (omega <= 0 || omega >= 2) {
            throw std::invalid_argument("Omega must be between 0 and 2");
        }
    }

    bool solve(AlignedField<Type>& x, const AlignedField<Type>& b) {
        AlignedField<Type> x_old(x.size(), Type());
        Type residualNorm = Type();
        label iteration = 0;

        for (iteration = 0; iteration < maxIterations_; ++iteration) {
            // Store old solution
            for (label i = 0; i < x.size(); ++i) {
                x_old[i] = x[i];
            }

            // SOR update
            for (label i = 0; i < matrix_.size(); ++i) {
                Type sum = b[i];
                Type x_new = x[i];

                // Lower triangular part
                for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                    label col = i - (matrix_.diagIndices()[i] - j);
                    sum -= matrix_.rows()[i][j] * x[col];
                }

                // Diagonal (partial update)
                sum -= matrix_.rows()[i][matrix_.diagIndices()[i]] * x_new;

                // Upper triangular part
                for (label j = matrix_.diagIndices()[i] + 1; j < matrix_.rows()[i].size(); ++j) {
                    label col = i + (j - matrix_.diagIndices()[i]);
                    sum -= matrix_.rows()[i][j] * x_old[col];
                }

                // Apply relaxation
                Type diag_val = matrix_.rows()[i][matrix_.diagIndices()[i]];
                if (std::abs(diag_val) < 1e-12) {
                    x[i] = Type();
                } else {
                    x[i] = x_old[i] + omega_ * (sum / diag_val - x_old[i]);
                }
            }

            // Check convergence
            residualNorm = computeResidual(x, b);
            if (residualNorm < tolerance_) {
                break;
            }
        }

        std::cout << "SOR converged in " << iteration << " iterations\n";
        std::cout << "Final residual norm: " << residualNorm << "\n";

        return iteration < maxIterations_;
    }

private:
    Type computeResidual(const AlignedField<Type>& x, const AlignedField<Type>& b) {
        AlignedField<Type> residual = matrix_.multiply(x);
        Type norm = Type();

        for (label i = 0; i < residual.size(); ++i) {
            residual[i] = b[i] - residual[i];
            norm += residual[i] * residual[i];
        }

        return std::sqrt(norm);
    }
};
```

```cpp
template<class Type>
class IncompleteCholesky {
private:
    LDUMatrix<Type>& matrix_;

public:
    IncompleteCholesky(LDUMatrix<Type>& mat) : matrix_(mat) {}

    // Compute M = L*L^T approximation
    void compute() {
        for (label i = 0; i < matrix_.size(); ++i) {
            Type diag = 0.0;

            // Compute diagonal element
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label k = i - (matrix_.diagIndices()[i] - j);
                Type l_ik = matrix_.rows()[i][j];

                for (label p = 0; p < j; ++p) {
                    label kp = i - (matrix_.diagIndices()[i] - p);
                    diag += l_ik * matrix_.rows()[k][p] * matrix_.rows()[k][p];
                }
            }

            // Update diagonal
            matrix_.addDiagonal(i, std::sqrt(matrix_.operator[](i) - diag));

            // Compute off-diagonal elements
            for (label j = matrix_.diagIndices()[i] + 1; j < matrix_.rows()[i].size(); ++j) {
                label col = i + (j - matrix_.diagIndices()[i]);

                Type sum = 0.0;
                for (label p = 0; p < matrix_.diagIndices()[i]; ++p) {
                    label kp = i - (matrix_.diagIndices()[i] - p);
                    label lp = col - (matrix_.diagIndices()[col] - p);
                    sum += matrix_.rows()[i][p] * matrix_.rows()[col][p];
                }

                matrix_.rows()[i][j] = (matrix_.rows()[i][j] - sum) / matrix_.operator[](i);
            }
        }
    }
};

// Preconditioned Gauss-Seidel
template<class Type>
class PreconditionedGaussSeidel {
private:
    LDUMatrix<Type>& matrix_;
    IncompleteCholesky<Type> preconditioner_;
    Type tolerance_;
    label maxIterations_;

public:
    PreconditionedGaussSeidel(LDUMatrix<Type>& mat, Type tol = 1e-10, label maxIter = 1000)
        : matrix_(mat), preconditioner_(mat), tolerance_(tol), maxIterations_(maxIter) {
        preconditioner_.compute();
    }

    bool solve(AlignedField<Type>& x, const AlignedField<Type>& b) {
        AlignedField<Type> r(b.size());  // Residual

        for (label iter = 0; iter < maxIterations_; ++iter) {
            // Apply preconditioner: solve M*z = r
            AlignedField<Type> z = solvePreconditioner(r);

            // Update solution
            for (label i = 0; i < x.size(); ++i) {
                x[i] += z[i];
            }

            // Update residual
            r = b - matrix_.multiply(x);

            // Check convergence
            Type residualNorm = computeNorm(r);
            if (residualNorm < tolerance_) {
                std::cout << "Preconditioned GS converged in " << iter << " iterations\n";
                return true;
            }
        }

        std::cout << "Preconditioned GS did not converge\n";
        return false;
    }

private:
    AlignedField<Type> solvePreconditioner(const AlignedField<Type>& b) {
        AlignedField<Type> z(b.size());

        // Forward substitution: L*y = b
        for (label i = 0; i < matrix_.size(); ++i) {
            Type sum = b[i];
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label col = i - (matrix_.diagIndices()[i] - j);
                sum -= matrix_.rows()[i][j] * z[col];
            }
            z[i] = sum / matrix_.operator[](i);
        }

        // Back substitution: L^T*x = y
        for (label i = matrix_.size() - 1; i >= 0; --i) {
            Type sum = z[i];
            for (label j = matrix_.diagIndices()[i] + 1; j < matrix_.rows()[i].size(); ++j) {
                label col = i + (j - matrix_.diagIndices()[i]);
                sum -= matrix_.rows()[i][j] * z[col];
            }
            z[i] = sum / matrix_.operator[](i);
        }

        return z;
    }

    Type computeNorm(const AlignedField<Type>& v) {
        Type norm = 0.0;
        for (label i = 0; i < v.size(); ++i) {
            norm += v[i] * v[i];
        }
        return std::sqrt(norm);
    }
};
```

```cpp
            label col = i - (matrix_.diagIndices()[i] - j);
            sum -= matrix_.rows()[i][j] * x[col];
        }

        // Diagonal
        sum -= matrix_.rows()[i][matrix_.diagIndices()[i]] * x[i];
```

```cpp
template<class Type>
class MatrixAnalyzer {
private:
    LDUMatrix<Type>& matrix_;

public:
    MatrixAnalyzer(LDUMatrix<Type>& mat) : matrix_(mat) {}

    // Compute matrix norm (infinity norm)
    Type computeInfinityNorm() {
        Type norm = 0.0;

        for (label i = 0; i < matrix_.size(); ++i) {
            Type row_sum = 0.0;

            // Sum absolute values in row
            for (label j = 0; j < matrix_.rows()[i].size(); ++j) {
                row_sum += std::abs(matrix_.rows()[i][j]);
            }

            norm = std::max(norm, row_sum);
        }

        return norm;
    }

    // Estimate condition number using power method
    Type estimateConditionNumber() {
        // Compute A_inf norm
        Type A_inf = computeInfinityNorm();

        // Compute A^{-1}_inf norm using iterative refinement
        AlignedField<Type> x(matrix_.size(), 1.0);
        AlignedField<Type> b(matrix_.size());

        // Create right-hand side with maximum values
        for (label i = 0; i < b.size(); ++i) {
            b[i] = 1.0;
        }

        // Solve multiple times to estimate inverse norm
        Type max_norm = 0.0;
        const int num_tests = 10;

        for (int test = 0; test < num_tests; ++test) {
            // Create random right-hand side
            std::random_device rd;
            std::mt19937 gen(rd());
            std::uniform_real_distribution<Type> dist(-1.0, 1.0);

            for (label i = 0; i < b.size(); ++i) {
                b[i] = dist(gen);
            }

            // Solve Ax = b
            GaussSeidelSolver<Type> solver(matrix_);
            solver.solve(x, b);

            // Compute norm of solution
            Type x_norm = 0.0;
            for (label i = 0; i < x.size(); ++i) {
                x_norm = std::max(x_norm, std::abs(x[i]));
            }

            max_norm = std::max(max_norm, x_norm);
        }

        return A_inf * max_norm;
    }

    // Analyze eigenvalue distribution
    void analyzeEigenvalues() {

        std::cout << "Size: " << matrix_.size() << "x" << matrix_.size() << "\n";
        std::cout << "Non-zeros: " << matrix_.nnz() << "\n";
        std::cout << "Sparsity: " << static_cast<double>(matrix_.nnz()) / (matrix_.size() * matrix_.size()) << "\n";

        Type condition = estimateConditionNumber();
        std::cout << "Estimated condition number: " << condition << "\n";

        // Estimate spectral radius of iteration matrix
        Type spectral_radius = estimateSpectralRadius();
        std::cout << "Estimated spectral radius of GS iteration matrix: " << spectral_radius << "\n";
        std::cout << "Expected convergence rate: " << -std::log(spectral_radius) << "\n";

private:
    Type estimateSpectralRadius() {
        // Power method for iteration matrix T = -(D+L)^{-1}U
        AlignedField<Type> x(matrix_.size(), 1.0);
        AlignedField<Type> x_new(matrix_.size());

        Type lambda = 0.0;
        const int max_iter = 100;
        const Type tol = 1e-10;

        for (int iter = 0; iter < max_iter; ++iter) {
            // Apply iteration matrix: T*x = -(D+L)^{-1}*U*x
            computeIterationMatrixAction(x, x_new);

            // Compute Rayleigh quotient
            Type numerator = 0.0;
            Type denominator = 0.0;

            for (label i = 0; i < x.size(); ++i) {
                numerator += x[i] * x_new[i];
                denominator += x[i] * x[i];
            }

            lambda = numerator / denominator;

            // Normalize
            Type norm = 0.0;
            for (label i = 0; i < x_new.size(); ++i) {
                norm += x_new[i] * x_new[i];
            }
            norm = std::sqrt(norm);

            for (label i = 0; i < x.size(); ++i) {
                x[i] = x_new[i] / norm;
            }

            // Check convergence
            if (std::abs(lambda) < tol) {
                break;
            }
        }

        return std::abs(lambda);
    }

    void computeIterationMatrixAction(const AlignedField<Type>& x, AlignedField<Type>& result) {
        // Compute y = U*x
        AlignedField<Type> y(matrix_.size(), Type());

        for (label i = 0; i < matrix_.size(); ++i) {
            for (label j = matrix_.diagIndices()[i] + 1; j < matrix_.rows()[i].size(); ++j) {
                label col = i + (j - matrix_.diagIndices()[i]);
                y[i] += matrix_.rows()[i][j] * x[col];
            }
        }

        // Solve (D+L)*z = -y using forward substitution
        for (label i = 0; i < matrix_.size(); ++i) {
            Type sum = y[i];

            // Lower triangular
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label col = i - (matrix_.diagIndices()[i] - j);
                sum -= matrix_.rows()[i][j] * result[col];
            }

            // Diagonal
            result[i] = -sum / matrix_.operator[](i);
        }
    }
```
