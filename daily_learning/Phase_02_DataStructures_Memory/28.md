## 28.1 Iterative Solver Fundamentals

### Introduction to Iterative Methods

Linear systems in CFD applications often involve millions of unknowns, making direct solvers computationally expensive. Iterative methods provide a scalable alternative by progressively improving the solution until convergence.

The fundamental idea behind iterative methods is to rewrite $A\mathbf{x} = \mathbf{b}$ as:

$$
\mathbf{x}^{(k+1)} = G\mathbf{x}^{(k)} + \mathbf{c}
$$

Where $G$ is the iteration matrix and $\mathbf{c}$ is a constant vector. The method converges if the spectral radius $\rho(G) < 1$.

### Iterative Method Classification

**Stationary Methods:**
- Update each component using fixed formulas
- Examples: Jacobi, Gauss-Seidel, SOR
- Simple to implement but slow convergence

**Non-stationary Methods:**
- Use varying iteration matrices
- Examples: Conjugate Gradient, GMRES, BiCGSTAB
- More complex but faster convergence

**Preconditioned Methods:**
- Apply transformation to improve conditioning
- Examples: PCG, GMRES with preconditioning
- Best for ill-conditioned systems

### Convergence Metrics

We monitor convergence using several metrics:

1. **Relative Residual**: $\frac{||\mathbf{b} - A\mathbf{x}^{(k)}||}{||\mathbf{b}||}$
2. **Absolute Residual**: $||\mathbf{b} - A\mathbf{x}^{(k)}||$
3. **Solution Change**: $||\mathbf{x}^{(k+1)} - \mathbf{x}^{(k)}||$
4. **A-norm**: $||\mathbf{x}^{(k+1)} - \mathbf{x}^{(k)}||_A = \sqrt{(\mathbf{x}^{(k+1)} - \mathbf{x}^{(k)})^T A (\mathbf{x}^{(k+1)} - \mathbf{x}^{(k)})}$

### Matrix Properties Affecting Convergence

For symmetric positive definite (SPD) matrices:
- Gauss-Seidel always converges
- Convergence rate depends on condition number: $\kappa(A)$
- Preconditioning can reduce $\kappa(M^{-1}A)$

For general matrices:
- Convergence depends on spectral radius
- May require preconditioning or different methods

## 28.2 Gauss-Seidel Implementation for LDU Format

### Optimized GS Solver Implementation

```cpp
// File: daily_learning/Phase_02_DataStructures_Memory/lduGaussSeidel.H
#ifndef lduGaussSeidel_H
#define lduGaussSeidel_H

#include "lduMatrix.H"
#include "Time.H"

template<class Type>
class lduGaussSeidelSolver
{
private:
    lduMatrix<Type>& matrix_;
    Type tolerance_;
    label maxIterations_;
    bool verbose_;

    // Performance monitoring
    mutable label totalIterations_;
    mutable scalar convergenceTime_;

public:
    lduGaussSeidelSolver(
        lduMatrix<Type>& mat,
        Type tol = 1e-10,
        label maxIter = 1000,
        bool verbose = true
    )
    :
    matrix_(mat),
    tolerance_(tol),
    maxIterations_(maxIter),
    verbose_(verbose),
    totalIterations_(0),
    convergenceTime_(0.0)
    {}

    bool solve(AlignedField<Type>& x, const AlignedField<Type>& b) const;

    // Performance accessors
    label totalIterations() const { return totalIterations_; }
    scalar convergenceTime() const { return convergenceTime_; }
};

template<class Type>
bool lduGaussSeidelSolver<Type>::solve(
    AlignedField<Type>& x,
    const AlignedField<Type>& b
) const {
    const label size = matrix_.size();
    AlignedField<Type> x_old(size, 0.0);

    auto startTime = std::chrono::high_resolution_clock::now();

    Type residualNorm = 0.0;
    label iteration = 0;

    for (iteration = 0; iteration < maxIterations_; ++iteration) {
        // Store current solution
        for (label i = 0; i < size; ++i) {
            x_old[i] = x[i];
        }

        // Gauss-Seidel sweep
        Type maxUpdate = 0.0;

        for (label i = 0; i < size; ++i) {
            Type sum = b[i];
            Type x_new = x[i];

            // Lower triangular part (using updated values)
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label col = i - (matrix_.diagIndices()[i] - j);
                sum -= matrix_.lower()[j] * x[col];
            }

            // Diagonal
            sum -= matrix_.diagonal()[i] * x[i];

            // Upper triangular part (using old values)
            for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
                label col = i + 1 + j;
                sum -= matrix_.upper()[j] * x_old[col];
            }

            // Update solution
            x_new = sum / matrix_.diagonal()[i];
            Type update = std::abs(x_new - x_old[i]);

            if (update > maxUpdate) {
                maxUpdate = update;
            }

            x[i] = x_new;
        }

        // Calculate residual
        residualNorm = calculateResidual(matrix_, x, b);

        if (verbose_) {
            Info<< "GS Iteration " << iteration + 1
                << ", Residual: " << residualNorm
                << ", Max Update: " << maxUpdate << endl;
        }

        // Check convergence
        if (residualNorm < tolerance_) {
            break;
        }

        // Early exit if no progress
        if (maxUpdate < 1e-15) {
            Warning<< "GS solver stalled at iteration " << iteration + 1 << endl;
            break;
        }
    }

    auto endTime = std::chrono::high_resolution_clock::now();
    convergenceTime_ = std::chrono::duration_cast<std::chrono::duration<double>>(
        endTime - startTime
    ).count();

    totalIterations_ = iteration + 1;

    if (verbose_) {
        Info<< "GS solver completed in " << totalIterations_ << " iterations, "
            << convergenceTime_ << " seconds" << endl;
    }

    return iteration < maxIterations_;
}

// Residual calculation utility
template<class Type>
Type calculateResidual(
    const lduMatrix<Type>& matrix,
    const AlignedField<Type>& x,
    const AlignedField<Type>& b
) {
    const label size = matrix.size();
    AlignedField<Type> residual(size, 0.0);

    // Matrix-vector multiplication: residual = b - A*x
    for (label i = 0; i < size; ++i) {
        residual[i] = b[i];

        // Lower triangular
        for (label j = 0; j < matrix.diagIndices()[i]; ++j) {
            label col = i - (matrix.diagIndices()[i] - j);
            residual[i] -= matrix.lower()[j] * x[col];
        }

        // Diagonal
        residual[i] -= matrix.diagonal()[i] * x[i];

        // Upper triangular
        for (label j = 0; j < matrix.upperIndices()[i]; ++j) {
            label col = i + 1 + j;
            residual[i] -= matrix.upper()[j] * x[col];
        }
    }

    // Calculate 2-norm
    Type norm = 0.0;
    for (label i = 0; i < size; ++i) {
        norm += residual[i] * residual[i];
    }

    return std::sqrt(norm);
}
#endif
```

### LDU-Specific Optimizations

The Gauss-Seidel implementation above leverages the LDU structure for efficiency:

1. **Memory Access Pattern**: Cache-friendly traversal by row
2. **Triangular Separation**: Lower part uses updated values, upper part uses old values
3. **No Matrix Assembly**: Direct access to L, D, U components
4. **Early Termination**: Multiple convergence checks for robustness

### Implementation Details

**Key Optimization**: The inner loop separates lower and upper triangular accesses:

```cpp
// Lower triangular (updated values)
for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
    label col = i - (matrix_.diagIndices()[i] - j);
    sum -= matrix_.lower()[j] * x[col];
}

// Upper triangular (old values)
for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
    label col = i + 1 + j;
    sum -= matrix_.upper()[j] * x_old[col];
}
```

This ensures optimal cache utilization while maintaining the Gauss-Seidel algorithm's convergence properties.

## 28.3 Convergence Monitoring and Residual Calculation

### Convergence Criteria Implementation

```cpp
// File: daily_learning/Phase_02_DataStructures_Memory/convergenceCriteria.H
#ifndef convergenceCriteria_H
#define convergenceCriteria_H

#include "lduMatrix.H"
#include "alignedField.H"

template<class Type>
class ConvergenceMonitor
{
private:
    lduMatrix<Type>& matrix_;
    Type tolerance_;
    label maxIterations_;
    Type relativeTolerance_;

    // Convergence history
    mutable std::vector<Type> residualHistory_;
    mutable std::vector<Type> updateHistory_;
    mutable std::vector<scalar> timeHistory_;

public:
    enum class Criterion {
        RESIDUAL,
        SOLUTION_CHANGE,
        RELATIVE_RESIDUAL,
        MIXED
    };

private:
    Criterion criterion_;
    label checkInterval_;

public:
    ConvergenceMonitor(
        lduMatrix<Type>& mat,
        Type absTol = 1e-10,
        Type relTol = 1e-8,
        label maxIter = 1000,
        Criterion crit = Criterion::MIXED,
        label checkInt = 10
    )
    :
    matrix_(mat),
    tolerance_(absTol),
    maxIterations_(maxIter),
    relativeTolerance_(relTol),
    criterion_(crit),
    checkInterval_(checkInt)
    {
        reserveHistory(maxIterations_);
    }

    bool checkConvergence(
        const AlignedField<Type>& x,
        const AlignedField<Type>& x_old,
        const AlignedField<Type>& b,
        label iteration,
        scalar timeElapsed
    ) const;

    const std::vector<Type>& residualHistory() const { return residualHistory_; }
    const std::vector<Type>& updateHistory() const { return updateHistory_; }

    // Advanced convergence analysis
    Type estimateConvergenceRate() const;
    bool detectOscillations() const;
    Type getOptimalDamping() const;

private:
    void reserveHistory(label maxIter) {
        residualHistory_.reserve(maxIter);
        updateHistory_.reserve(maxIter);
        timeHistory_.reserve(maxIter);
    }

    Type calculateResidual(const AlignedField<Type>& x, const AlignedField<Type>& b) const;
    Type calculateSolutionChange(const AlignedField<Type>& x, const AlignedField<Type>& x_old) const;
};

template<class Type>
bool ConvergenceMonitor<Type>::checkConvergence(
    const AlignedField<Type>& x,
    const AlignedField<Type>& x_old,
    const AlignedField<Type>& b,
    label iteration,
    scalar timeElapsed
) const {
    // Update history
    residualHistory_.push_back(calculateResidual(x, b));
    updateHistory_.push_back(calculateSolutionChange(x, x_old));
    timeHistory_.push_back(timeElapsed);

    // Check convergence only at intervals
    if (iteration % checkInterval_ != 0 && iteration < maxIterations_ - 1) {
        return false;
    }

    Type residual = residualHistory_.back();
    Type update = updateHistory_.back();

    switch (criterion_) {
        case Criterion::RESIDUAL:
            return residual < tolerance_;

        case Criterion::SOLUTION_CHANGE:
            return update < tolerance_;

        case Criterion::RELATIVE_RESIDUAL: {
            Type initialResidual = residualHistory_.size() > 1 ?
                                  residualHistory_[0] : 1.0;
            return residual < relativeTolerance_ * initialResidual;
        }

        case Criterion::MIXED:
            // Both residual and solution change must be small
            Type residualCrit = residual < tolerance_;
            Type updateCrit = update < tolerance_ * 1e-3;
            return residualCrit && updateCrit;

        default:
            return false;
    }
}

template<class Type>
Type ConvergenceMonitor<Type>::calculateResidual(
    const AlignedField<Type>& x,
    const AlignedField<Type>& b
) const {
    Type residualNorm = 0.0;
    const label size = matrix_.size();

    // Calculate A*x
    for (label i = 0; i < size; ++i) {
        Type ax = matrix_.diagonal()[i] * x[i];

        // Lower triangular
        for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
            label col = i - (matrix_.diagIndices()[i] - j);
            ax += matrix_.lower()[j] * x[col];
        }

        // Upper triangular
        for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
            label col = i + 1 + j;
            ax += matrix_.upper()[j] * x[col];
        }

        residualNorm += (b[i] - ax) * (b[i] - ax);
    }

    return std::sqrt(residualNorm);
}

template<class Type>
Type ConvergenceMonitor<Type>::calculateSolutionChange(
    const AlignedField<Type>& x,
    const AlignedField<Type>& x_old
) const {
    Type maxChange = 0.0;

    for (label i = 0; i < x.size(); ++i) {
        Type change = std::abs(x[i] - x_old[i]);
        if (change > maxChange) {
            maxChange = change;
        }
    }

    return maxChange;
}

template<class Type>
Type ConvergenceMonitor<Type>::estimateConvergenceRate() const {
    if (residualHistory_.size() < 3) {
        return 1.0;
    }

    // Use last three residuals to estimate rate
    Type r0 = residualHistory_[residualHistory_.size() - 3];
    Type r1 = residualHistory_[residualHistory_.size() - 2];
    Type r2 = residualHistory_.back();

    if (r0 <= 0 || r1 <= 0 || r2 <= 0) {
        return 1.0;
    }

    // Estimate rate as geometric mean
    Type rate1 = r1 / r0;
    Type rate2 = r2 / r1;

    return std::sqrt(rate1 * rate2);
}

template<class Type>
bool ConvergenceMonitor<Type>::detectOscillations() const {
    if (updateHistory_.size() < 5) {
        return false;
    }

    // Check for alternating sign pattern
    for (label i = updateHistory_.size() - 5; i < updateHistory_.size() - 1; ++i) {
        if (updateHistory_[i] * updateHistory_[i + 1] > 0) {
            return false; // Same sign, not oscillating
        }
    }

    return true;
}

template<class Type>
Type ConvergenceMonitor<Type>::getOptimalDamping() const {
    if (updateHistory_.size() < 3) {
        return 1.0;
    }

    // Simple heuristic: reduce oscillations
    Type lastUpdate = updateHistory_.back();
    Type prevUpdate = updateHistory_[updateHistory_.size() - 2];

    if (lastUpdate * prevUpdate > 0) {
        return 1.0; // No oscillation, full step
    } else {
        return 0.5; // Damped step to reduce oscillation
    }
}
#endif
```

### Advanced Convergence Features

**1. Rate Estimation**: The `estimateConvergenceRate()` method calculates the geometric mean of recent convergence rates:

```cpp
Type rate1 = r1 / r0;
Type rate2 = r2 / r1;
return std::sqrt(rate1 * rate2);
```

**2. Oscillation Detection**: Detects when the solution update alternates in sign:

```cpp
for (label i = updateHistory_.size() - 5; i < updateHistory_.size() - 1; ++i) {
    if (updateHistory_[i] * updateHistory_[i + 1] > 0) {
        return false; // Same sign, not oscillating
    }
}
```

**3. Adaptive Damping**: Automatically adjusts the damping factor based on oscillation detection:

```cpp
if (lastUpdate * prevUpdate > 0) {
    return 1.0; // No oscillation, full step
} else {
    return 0.5; // Damped step to reduce oscillation
}
```

```cpp
    mkDir(outputDir);

    Info<< "Starting benchmark suite..." << endl;

    for (label size : problemSizes_) {
        for (double sparsity : sparsityLevels_) {
            for (Type tol : tolerances_) {
                // Generate test problem
                auto testMatrix = generateTestMatrix(size, sparsity);
                auto testRHS = generateTestRHS(size);

                // Configure tolerance
                setGlobalTolerance(tol);

                // Test each solver
                benchmarkJacobi(testRHS);
                benchmarkGaussSeidel(testRHS);
                benchmarkSOR(testRHS);
                benchmarkRedBlackGS(testRHS);

                // Only test analytical for small problems
                if (size <= 1000) {
                    benchmarkAnalytical(testRHS);
                }

                Info<< "Completed: size=" << size << ", sparsity=" << sparsity
                    << ", tol=" << tol << endl;
            }
        }
    }

    // Export results
    exportResults(outputDir);
    generatePlots(outputDir);
    printSummary();
}

template<class Type>

    AlignedField<Type> x(matrix_.size(), 0.0);

    auto startTime = std::chrono::high_resolution_clock::now();
    bool converged = jacobiSolver.solve(x, b);
    auto endTime = std::chrono::high_resolution_clock::now();

    scalar timeElapsed = std::chrono::duration_cast<std::chrono::duration<double>>(
        endTime - startTime
    ).count();

    // Calculate final residual
    Type finalResidual = calculateResidual(matrix_, x, b);

    // Store result
    BenchmarkResult result;
    result.problemSize = matrix_.size();
    result.sparsity = calculateSparsity(matrix_);
    result.tolerance = tolerance_;
    result.solverType = solverType::JACOBI;
    result.iterations = jacobiSolver.totalIterations();
    result.time = timeElapsed;
    result.finalResidual = finalResidual;
    result.memoryUsage = getMemoryUsage();

    results_.push_back(result);

    Info<< "Jacobi: " << jacobiSolver.totalIterations() << " iterations, "
        << timeElapsed << " seconds" << endl;
}
```

```cpp
bool lduSORSolver<Type>::solve(
    AlignedField<Type>& x,
    const AlignedField<Type>& b
) const {
    const label size = matrix_.size();
    AlignedField<Type> x_old(size, 0.0);

    auto startTime = std::chrono::high_resolution_clock::now();

    Type residualNorm = 0.0;
    label iteration = 0;

    for (iteration = 0; iteration < maxIterations_; ++iteration) {
        // Store current solution
        for (label i = 0; i < size; ++i) {
            x_old[i] = x[i];
        }

        // SOR sweep
        Type maxUpdate = 0.0;

        for (label i = 0; i < size; ++i) {
            Type sum = b[i];
            Type x_new = x[i];

            // Gauss-Seidel part
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label col = i - (matrix_.diagIndices()[i] - j);
                sum -= matrix_.lower()[j] * x[col];
            }

            sum -= matrix_.diagonal()[i] * x[i];

            for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
                label col = i + 1 + j;
                sum -= matrix_.upper()[j] * x_old[col];
            }

            // SOR update with relaxation
            x_new = (1 - omega_) * x[i] + omega_ * sum / matrix_.diagonal()[i];
            Type update = std::abs(x_new - x_old[i]);

            if (update > maxUpdate) {
                maxUpdate = update;
            }

            x[i] = x_new;
        }

        // Calculate residual
        residualNorm = calculateResidual(matrix_, x, b);

        if (verbose_) {
            Info<< "SOR Iteration " << iteration + 1
                << ", Residual: " << residualNorm
                << ", Omega: " << omega_ << endl;
        }

        // Check convergence
        if (residualNorm < tolerance_) {
            break;
        }
    }

    auto endTime = std::chrono::high_resolution_clock::now();
    convergenceTime_ = std::chrono::duration_cast<std::chrono::duration<double>>(
        endTime - startTime
    ).count();

    totalIterations_ = iteration + 1;

    if (verbose_) {
        Info<< "SOR solver completed in " << totalIterations_ << " iterations, "
            << convergenceTime_ << " seconds" << endl;
    }

    return iteration < maxIterations_;
}
#endif
```

### Optimal Relaxation Parameter

The optimal ω for SOR can be estimated as:

$$
\omega_{opt} = \frac{2}{1 + \sqrt{1 - \rho^2(GS)}}
$$

Where ρ(GS) is the spectral radius of the Gauss-Seidel iteration matrix.

```cpp
#define preconditioners_H

#include "alignedField.H"

template<class Type>
class DiagonalPreconditioner
{
private:
    std::vector<Type> diagonal_;

public:
    DiagonalPreconditioner(const lduMatrix<Type>& matrix) {
        const label size = matrix.size();
        diagonal_.resize(size);

        for (label i = 0; i < size; ++i) {
            diagonal_[i] = 1.0 / matrix.diagonal()[i];
        }
    }

    void apply(AlignedField<Type>& x) const {
        for (label i = 0; i < x.size(); ++i) {
            x[i] *= diagonal_[i];
        }
    }

    void applyTranspose(AlignedField<Type>& x) const {
        apply(x);  // Diagonal is symmetric
    }
};

template<class Type>
class SSORPreconditioner
{
private:
    lduMatrix<Type>& matrix_;
    Type omega_;
    mutable lduMatrix<Type> lower_;
    mutable lduMatrix<Type> upper_;

public:
    SSORPreconditioner(lduMatrix<Type>& mat, Type omega = 1.0)
    :
    matrix_(mat),
    omega_(omega)
    {}

    void apply(AlignedField<Type>& x) const {
        // Forward sweep: (D + ωL)x = x
        for (label i = 0; i < matrix_.size(); ++i) {
            Type sum = x[i];
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label col = i - (matrix_.diagIndices()[i] - j);
                sum -= omega_ * matrix_.lower()[j] * x[col];
            }
            x[i] = sum / (omega_ * matrix_.diagonal()[i]);
        }

        // Backward sweep: (D + ωU)x = x
        for (label i = matrix_.size() - 1; i >= 0; --i) {
            Type sum = x[i];
            for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
                label col = i + 1 + j;
                sum -= omega_ * matrix_.upper()[j] * x[col];
            }
            x[i] = sum / (omega_ * matrix_.diagonal()[i]);
        }
    }
};
```

```cpp
            matrix.addUpper(i, i + 1, -1.0);
```

```cpp

#ifndef convergenceAnalysis_H
#define convergenceAnalysis_H

#include "lduMatrix.H"
#include "alignedField.H"

template<class Type>
class ConvergenceAnalyzer
{
private:
    lduMatrix<Type>& matrix_;

public:
    ConvergenceAnalyzer(lduMatrix<Type>& mat)
    :
    matrix_(mat)
    {}

    // Estimate spectral radius of iteration matrices
    Type estimateJacobiSpectralRadius();
    Type estimateGSSpectralRadius();

    // Theoretical convergence rates
    Type theoreticalJacobiRate();
    Type theoreticalGSRate();

    // Experimental convergence rate
    Type experimentalConvergenceRate(
        const std::vector<Type>& residualHistory
    );
};

template<class Type>
Type ConvergenceAnalyzer<Type>::estimateJacobiSpectralRadius() {
    // For SPD matrices, bounds are known
    // ρ(J) ≤ ρ(GS)² < 1

    // Simple estimate using diagonal dominance
    Type rho_estimate = 0.0;
    const label size = matrix_.size();

    for (label i = 0; i < size; ++i) {
        Type diag = std::abs(matrix_.diagonal()[i]);
        Type off_diag_sum = 0.0;

        // Lower triangular
        for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
            off_diag_sum += std::abs(matrix_.lower()[j]);
        }

        // Upper triangular
        for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
            off_diag_sum += std::abs(matrix_.upper()[j]);
        }

        // Local spectral estimate
        Type local_rho = off_diag_sum / diag;
        if (local_rho > rho_estimate) {
            rho_estimate = local_rho;
        }
    }

    return rho_estimate;
}

template<class Type>
Type ConvergenceAnalyzer<Type>::theoreticalJacobiRate() {
    // For 2D Poisson: ρ(J) ≈ 1 - π²/(2n²)
    Type rho = estimateJacobiSpectralRadius();
    return rho * rho;  // Conservative bound
}

template<class Type>
Type ConvergenceAnalyzer<Type>::experimentalConvergenceRate(
    const std::vector<Type>& residualHistory
) {
    if (residualHistory.size() < 3) {
        return 1.0;
    }

    // Fit exponential decay: r_k = r_0 * ρ^k
    // Take logarithm: log(r_k) = log(r_0) + k * log(ρ)
    // Linear fit to get log(ρ)

    double sum_k = 0.0, sum_log_r = 0.0;
    double sum_k2 = 0.0, sum_k_log_r = 0.0;
    const label n = residualHistory.size();

    for (label k = 0; k < n; ++k) {
        double log_r = std::log(residualHistory[k]);
        sum_k += k;
        sum_log_r += log_r;
        sum_k2 += k * k;
        sum_k_log_r += k * log_r;
    }

    // Linear regression: log(r) = a + b*k
    // b = log(ρ), so ρ = exp(b)
    double b = (n * sum_k_log_r - sum_k * sum_log_r) /
               (n * sum_k2 - sum_k * sum_k);

    return std::exp(b);
}
```

```cpp
    Type adaptive_tol = baseTolerance_ * tol_factor;

    // Relax tolerance if iterations are high
    Type iter_factor = std::min(
        1.0 + 0.1 * std::log(currentIteration + 1),
        10.0
    );
    adaptive_tol *= iter_factor;

    // Time-based relaxation
    Type time_factor = 1.0;
    if (elapsed > 10.0) {  // If taking too long
        time_factor = 10.0;  // Relax tolerance
    }

    adaptive_tol *= time_factor;

    // Clamp to bounds
    adaptive_tol = std::max(adaptive_tol, minTolerance_);
    adaptive_tol = std::min(adaptive_tol, maxTolerance_);

    return adaptive_tol;
}
```

```cpp
#include "mpi.H"
#include "commSchedule.H"

template<class Type>
class ParallelGaussSeidelSolver
{
private:
    lduMatrix<Type>& matrix_;
    MPI::Intracomm comm_;

    // Domain decomposition
    label localSize_;
    label globalSize_;
    label startLabel_;

    // Communication
    label haloSize_;
    std::vector<label> haloIndices_;
    std::vector<Type> haloValues_;

    // Convergence
    Type tolerance_;
    label maxIterations_;

public:
    ParallelGaussSeidelSolver(
        lduMatrix<Type>& mat,
        MPI::Intracomm comm,
        Type tol = 1e-10,
        label maxIter = 1000
    )
    :
    matrix_(mat),
    comm_(comm),
    tolerance_(tol),
    maxIterations_(maxIter)
    {
        initializeDecomposition();
    }

    bool solve(AlignedField<Type>& x, const AlignedField<Type>& b);

private:
    void initializeDecomposition();
    void exchangeHaloValues(AlignedField<Type>& x);
    Type calculateGlobalResidual(const AlignedField<Type>& x, const AlignedField<Type>& b);
};

template<class Type>
void ParallelGaussSeidelSolver<Type>::initializeDecomposition() {
    int rank, size;
    comm_.rank(rank);
    comm_.size(size);

    globalSize_ = matrix_.size();
    localSize_ = globalSize_ / size;

    if (rank == size - 1) {
        localSize_ += globalSize_ % size;  // Last process gets remainder
    }

    startLabel_ = rank * (globalSize_ / size);

    // Identify halo cells (neighboring domains)
    haloSize_ = 0;
    haloIndices_.clear();

    for (label i = 0; i < localSize_; ++i) {
        label globalI = startLabel_ + i;

        // Check if cell has neighbors in other domains
        for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
            label neighbor = globalI + 1 + j;
            if (neighbor >= startLabel_ + localSize_) {
                haloIndices_.push_back(i);
                haloSize_++;
                break;
            }
        }

        for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
            label neighbor = globalI - (matrix_.diagIndices()[i] - j);
            if (neighbor < startLabel_) {
                haloIndices_.push_back(i);
                haloSize_++;
                break;
            }
        }
    }

    haloValues_.resize(haloSize_);
}

template<class Type>
void ParallelGaussSeidelSolver<Type>::exchangeHaloValues(AlignedField<Type>& x) {
    int rank, size;
    comm_.rank(rank);
    comm_.size(size);

    // Pack halo values
    for (label i = 0; i < haloSize_; ++i) {
        haloValues_[i] = x[haloIndices_[i]];
    }

    // Create communication schedule
    MPI::Request sendRequests[size];
    MPI::Request recvRequests[size];

    // Non-blocking communication
    for (int dest = 0; dest < size; ++dest) {
        if (dest != rank) {
            // Send to destination
            sendRequests[dest] = comm_.isend(
                haloValues_.data(),
                haloSize_,
                MPI::DOUBLE,
                dest,
                0
            );

            // Receive from destination
            recvRequests[dest] = comm_.irecv(
                haloValues_.data(),
                haloSize_,
                MPI::DOUBLE,
                dest,
                0
            );
        }
    }

    // Wait for communication to complete
    for (int dest = 0; dest < size; ++dest) {
        if (dest != rank) {
            sendRequests[dest].wait();
            recvRequests[dest].wait();
        }
    }
}

template<class Type>
bool ParallelGaussSeidelSolver<Type>::solve(
    AlignedField<Type>& x,
    const AlignedField<Type>& b
) {
    AlignedField<Type> x_old(localSize_, 0.0);

    Type globalResidual = 0.0;
    label iteration = 0;

    for (iteration = 0; iteration < maxIterations_; ++iteration) {
        // Store local solution
        for (label i = 0; i < localSize_; ++i) {
            x_old[i] = x[i];
        }

        // Local Gauss-Seidel sweep
        for (label i = 0; i < localSize_; ++i) {
            Type sum = b[i];
            Type x_new = x[i];

            // Lower triangular (local + updated halos)
            for (label j = 0; j < matrix_.diagIndices()[i]; ++j) {
                label col = i - (matrix_.diagIndices()[i] - j);
                if (col >= 0 && col < localSize_) {
                    sum -= matrix_.lower()[j] * x[col];
                }
            }

            sum -= matrix_.diagonal()[i] * x[i];

            // Upper triangular (local + old halos)
            for (label j = 0; j < matrix_.upperIndices()[i]; ++j) {
                label col = i + 1 + j;
                if (col < localSize_) {
                    sum -= matrix_.upper()[j] * x[col];
                }
            }

            x[i] = sum / matrix_.diagonal()[i];
        }

        // Exchange boundary values
        exchangeHaloValues(x);

        // Calculate global residual
        globalResidual = calculateGlobalResidual(x, b);

        // Check convergence
        if (globalResidual < tolerance_) {
            break;
        }
    }

    return iteration < maxIterations_;
}
```
