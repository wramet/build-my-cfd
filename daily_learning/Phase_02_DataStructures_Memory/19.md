# Day 19: Cache Access Patterns - Owner/Neighbour Ordering

## Introduction to Cache-Aware Mesh Renumbering

In computational fluid dynamics, the performance of matrix operations is critically dependent on memory access patterns. The ordering of computational cells and faces significantly impacts cache utilization, with poor ordering leading to excessive cache misses and degraded performance. This day explores the relationship between mesh renumbering strategies and cache performance in OpenFOAM's finite volume implementations.

Matrix-vector multiplication and other sparse matrix operations exhibit dramatic performance variations based on the ordering of computational elements. When cells are numbered randomly, memory access patterns become non-contiguous, causing frequent cache line evictions. Conversely, intelligent renumbering creates locality-preserving orderings that maximize cache line utilization.

## Face Ordering Impact on Memory Access

### Cache Line Analysis

Modern CPUs utilize cache lines typically 64 bytes in size. When accessing matrix elements, the entire cache line containing the requested data is loaded into cache. Efficient mesh ordering ensures that consecutive operations access elements within the same cache line, maximizing reuse.

```cpp
// Cache line analysis for matrix access
void analyzeCacheLineAccess
(
    const fvMesh& mesh,
    const Field<scalar>& matrix_coefficients,
    label cache_line_size = 64
) {
    const scalar bytes_per_element = sizeof(scalar);
    const label elements_per_cache_line = cache_line_size / bytes_per_element;

    // Track cache line utilization
    std::vector<bool> cache_line_hit(elements_per_cache_line, false);
    label cache_misses = 0;
    label cache_hits = 0;

    // Analyze matrix access pattern
    forAll(mesh.faces(), facei) {
        const label own = mesh.faceOwner()[facei];
        const label nei = mesh.faceNeighbour()[facei];

        // Check cache line for owner
        label owner_cache_line = own / elements_per_cache_line;
        if (cache_line_hit[owner_cache_line % elements_per_cache_line]) {
            cache_hits++;
        } else {
            cache_misses++;
            cache_line_hit[owner_cache_line % elements_per_cache_line] = true;
        }

        // Check cache line for neighbour
        label neighbour_cache_line = nei / elements_per_cache_line;
        if (cache_line_hit[neighbour_cache_line % elements_per_cache_line]) {
            cache_hits++;
        } else {
            cache_misses++;
            cache_line_hit[neighbour_cache_line % elements_per_cache_line] = true;
        }
    }

    double hit_rate = (double)cache_hits / (cache_hits + cache_misses);
    std::cout << "Cache hit rate: " << hit_rate * 100 << "%\n";
}
```

### Bandwidth Reduction Strategies

Memory bandwidth represents a critical bottleneck in scientific computing. Efficient mesh ordering strategies can significantly reduce memory bandwidth requirements by:

1. **Spatial Locality**: Consecutive cells are stored contiguously in memory
2. **Temporal Locality**: Repeated access to the same cache lines
3. **Prefetching**: Hardware can predict and preload data patterns

```cpp

    (

        const VectorType& x,

        label n_faces = matrix.upper().size();

        // Each multiply requires:
        // - n_cells diagonal reads
        // - n_cells vector reads
        // - n_cells writes
        // - 2 * n_faces off-diagonal reads
        return 3 * n_cells + 2 * n_faces;
```

### Cuthill-McKee Implementation

Let's implement the Cuthill-McKee renumbering algorithm:

```cpp
// Cuthill-McKee renumbering implementation
std::vector<label> cuthillMcKee
(
    const std::vector<std::vector<label>>& adjacency_matrix,
    bool reverse = false
) {
    const label n = adjacency_matrix.size();

    // Find starting node (minimum degree)
    label start_node = 0;
    label min_degree = adjacency_matrix[0].size();
    for (label i = 0; i < n; i++) {
        if (adjacency_matrix[i].size() < min_degree) {
            min_degree = adjacency_matrix[i].size();
            start_node = i;
        }
    }

    // BFS traversal
    std::vector<label> ordering;
    std::vector<bool> visited(n, false);
    std::queue<label> queue;

    queue.push(start_node);
    visited[start_node] = true;

    while (!queue.empty()) {
        label current = queue.front();
        queue.pop();
        ordering.push_back(current);

        // Get neighbors sorted by degree
        std::vector<label> neighbors = adjacency_matrix[current];
        std::sort(neighbors.begin(), neighbors.end(),
                 [&](label a, label b) {
                     return adjacency_matrix[a].size() < adjacency_matrix[b].size();
                 });

        // Add unvisited neighbors
        for (label neighbor : neighbors) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.push(neighbor);
            }
        }
    }

    // Reverse if requested
    if (reverse) {
        std::reverse(ordering.begin(), ordering.end());
    }

    return ordering;
}

// Generate adjacency matrix from mesh
std::vector<std::vector<label>> createAdjacencyMatrix
(
    const fvMesh& mesh
) {
    const label nCells = mesh.cells().size();
    std::vector<std::vector<label>> adjacency(nCells);

    forAll(mesh.faces(), facei) {
        if (facei < mesh.nInternalFaces()) {
            const label own = mesh.faceOwner()[facei];
            const label nei = mesh.faceNeighbour()[facei];

            // Add bidirectional edges
            adjacency[own].push_back(nei);
            adjacency[nei].push_back(own);
        }
    }

    return adjacency;
}
```

### Bandwidth Calculation

```cpp
// Calculate matrix bandwidth
label calculateMatrixBandwidth
(
    const lduMatrix& matrix,
    const std::vector<label>& ordering
) {
    label bandwidth = 0;
    const label nCells = matrix.diag().size();

    // Map old indices to new ordering
    std::vector<label> new_index(nCells);
    for (label i = 0; i < nCells; i++) {
        new_index[ordering[i]] = i;
    }

    // Calculate bandwidth for each row
    for (label i = 0; i < nCells; i++) {
        label max_col_dist = 0;

        // Check upper triangle
        for (label j = matrix.diag().begin()[i];
             j < matrix.diag().begin()[i+1]; j++) {
            label col = matrix.upperAddr()[j];
            label col_dist = std::abs(new_index[i] - new_index[col]);
            max_col_dist = std::max(max_col_dist, col_dist);
        }

        bandwidth = std::max(bandwidth, max_col_dist);
    }

    return bandwidth;
}

// Apply renumbering to matrix
void applyRenumbering
(
    lduMatrix& matrix,
    const std::vector<label>& ordering
) {
    const label nCells = matrix.diag().size();

    // Create new matrix with renumbered indices
    lduMatrix new_matrix(nCells);

    // Renumber diagonal
    for (label i = 0; i < nCells; i++) {
        new_matrix.diag()[ordering[i]] = matrix.diag()[i];
    }

    // Renumber off-diagonal
    const label nFaces = matrix.upper().size();
    for (label facei = 0; facei < nFaces; facei++) {
        const label old_own = matrix.lduAddr().upperAddr()[facei];
        const label old_nei = matrix.lduAddr().lowerAddr()[facei];

        const label new_own = ordering[old_own];
        const label new_nei = ordering[old_nei];

        new_matrix.upper()[0][facei] = matrix.upper()[0][facei];
        new_matrix.lower()[0][facei] = matrix.lower()[0][facei];
    }

    // Copy back to original matrix
    matrix = new_matrix;
}
```

### Spectral Reordering

Spectral reordering uses eigenvectors of the Laplacian matrix to find optimal orderings:

```cpp
// Spectral reordering using Fiedler vector
std::vector<label> spectralReordering
(
    const std::vector<std::vector<label>>& adjacency
) {
    const label n = adjacency.size();

    // Compute Laplacian matrix
    Eigen::SparseMatrix<scalar> L(n, n);
    for (label i = 0; i < n; i++) {
        L.coeffRef(i, i) = adjacency[i].size();  // Diagonal
        for (label j : adjacency[i]) {
            L.coeffRef(i, j) = -1.0;  // Off-diagonal
        }
    }

    // Compute Fiedler vector (second eigenvector)
    Eigen::SelfAdjointEigenSolver<Eigen::SparseMatrix<scalar>> solver(L);
    Eigen::VectorXd fiedler = solver.eigenvectors().col(1);

    // Sort by Fiedler vector components
    std::vector<label> ordering(n);
    std::iota(ordering.begin(), ordering.end(), 0);
    std::sort(ordering.begin(), ordering.end(),
              [&](label i, label j) {
                  return fiedler(i) < fiedler(j);
              });

    return ordering;
}
```

```cpp
// Exercise 4: Parallel Cuthill-McKee
std::vector<label> exercise4_parallelCuthillMcKee
(
    const std::vector<std::vector<label>>& adjacency,
    int num_processes
) {
    const label n = adjacency.size();

    // Distribute adjacency matrix across processes
    std::vector<std::vector<label>> local_adj;
    distributeMatrix(adjacency, local_adj, num_processes);

    // Each process finds local minimum degree
    label local_min_node = 0;
    label local_min_degree = local_adj[0].size();
    for (label i = 0; i < local_adj.size(); i++) {
        if (local_adj[i].size() < local_min_degree) {
            local_min_degree = local_adj[i].size();
            local_min_node = i;
        }
    }

    // Find global minimum across all processes
    label global_min_node;
    MPI_Allreduce(&local_min_node, &global_min_node, 1,
                  MPI_INT, MPI_MIN, MPI_COMM_WORLD);

    // Parallel BFS
    std::vector<label> ordering;
    std::queue<label> queue;
    std::vector<bool> visited(n, false);

    queue.push(global_min_node);
    visited[global_min_node] = true;

    while (!queue.empty()) {
        label current = queue.front();
        queue.pop();
        ordering.push_back(current);

        // Get neighbors
        std::vector<label> neighbors = getNeighbors(current, adjacency, num_processes);

        // Sort by degree
        std::sort(neighbors.begin(), neighbors.end(),
                 [&](label a, label b) {
                     return getDegree(a, adjacency, num_processes) <
                            getDegree(b, adjacency, num_processes);
                 });

        for (label neighbor : neighbors) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.push(neighbor);
            }
        }
    }

    return ordering;
}
```
