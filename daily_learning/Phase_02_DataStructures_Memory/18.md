# Day 18: Sparse Matrix Assembly - How fvMatrix Populates LDU

## Introduction to Sparse Matrix Assembly in CFD

Sparse matrix assembly represents the critical process of discretizing partial differential equations (PDEs) into algebraic systems. In computational fluid dynamics, this transformation occurs when finite volume methods convert conservation laws into matrix equations. The efficiency and accuracy of this process directly impact both computational performance and solution quality.

The finite volume method discretizes computational domains into control volumes, integrating PDEs over these volumes to produce algebraic equations. Each equation corresponds to a computational cell, with coefficients determined by flux calculations across cell faces. This naturally results in sparse matrices where most elements are zero due to the localized nature of finite volume discretizations.

## Face-Loop Assembly Pattern

The core of matrix assembly in OpenFOAM follows the "face-loop" pattern, where coefficients are calculated by iterating over computational faces rather than cells. This approach optimizes cache utilization and leverages the natural structure of finite volume discretizations.

```cpp
tmp<fvMatrix<Type>>

(
    const surfaceScalarField& Gamma,
    const VolField<Type>& vf,
    const word& name
)
{
    // Create empty matrix
    tmp<fvMatrix<Type>> tfvm
    (
        new fvMatrix<Type>(vf, name)
    );

    fvMatrix<Type>& fvm = tfvm();

    // Get gradient scheme from field
    const surfaceScalarField::GeometricBoundaryField& Gammab =
        Gamma.boundaryField();

    // Main assembly loop - face-based approach
    forAll(Gamma.boundaryField(), patchi)
    {
        const fvPatchScalarField& GammaPatch = Gamma.boundaryField()[patchi];
        const fvPatch<Type>& vfPatch = vf.boundaryField()[patchi];

        // Boundary condition handling
        if (!GammaPatch.coupled())
        {
            // Fixed/gradient boundary conditions
            fvm.setValues
            (
                patchi,
                -GammaPatch.internalField()*vfPatch
            );
        }
        else
        {
            // Coupled boundary conditions
            fvm.insertInternalCoeffs
            (
                patchi,
                -GammaPatch.patchInternalField()
            );

            fvm.insertCoupledCoeffs
            (
                patchi,
                GammaPatch.patchNeighbourField()
            );
        }
    }

    // Complete matrix assembly
    completeAssembly();

    return tfvm;
}
```

### Face-Centered Assembly Algorithm

The face-loop assembly pattern follows this general structure:

```cpp
// Face-based matrix assembly template
template<class Type>
void faceLoopAssembly
(
    fvMatrix<Type>& fvm,
    const VolField<Type>& vf,
    const surfaceScalarField& coeffs
) {
    const fvMesh& mesh = vf.mesh();
    const label nInternalFaces = mesh.nInternalFaces();

    // PASS 1: Internal faces - bulk assembly
    forAll(mesh.faces(), facei)
    {
        if (facei < nInternalFaces)  // Internal face
        {
            const label own = mesh.faceOwner()[facei];
            const label nei = mesh.faceNeighbour()[facei];

            // Calculate face flux
            scalar flux = coeffs[facei];

            // Assemble matrix coefficients
            fvm[own] += flux;      // Add to owner equation
            fvm[nei] -= flux;      // Subtract from neighbour equation
        }
    }

    // PASS 2: Boundary faces - BC handling
    forAll(mesh.boundary(), patchi)
    {
        const fvPatch<Type>& patch = vf.boundaryField()[patchi];
        const labelUList& faceCells = patch.faceCells();

        forAll(faceCells, facei)
        {
            label celli = faceCells[facei];

            // Apply boundary conditions
            applyBoundaryConditions(fvm, patch, facei, celli);
        }
    }
}
```

### Boundary Condition Handling

Finite volume matrices require special handling for boundary conditions, which affect both matrix coefficients and source terms:

```cpp
// Boundary condition implementation
template<class Type>
void applyBoundaryConditions
(
    fvMatrix<Type>& fvm,
    const fvPatch<Type>& patch,
    label facei,
    label celli
) {
    if (patch.fixValue())
    {
        // Dirichlet boundary condition
        Type prescribedValue = patch.dict().lookup<Type>("value");

        // Set diagonal coefficient
        fvm.diag()[celli] += 1.0;

        // Adjust source term
        fvm.source()[celli] += prescribedValue;

        // Zero out off-diagonal contributions
        fvm.upper()[patch.patchIndex()][facei] = 0.0;
        fvm.lower()[patch.patchIndex()][facei] = 0.0;
    }
    else if (patch.gradient())
    {
        // Neumann boundary condition
        Type gradientValue = patch.dict().lookup<Type>("gradient");

        // Adjust source term
        fvm.source()[celli] += gradientValue;
    }
}
```

## 1D Laplacian Implementation

Let's implement a complete 1D Laplacian discretization to demonstrate the matrix assembly process:

### Problem Statement

Consider the 1D heat conduction equation:

$$\frac{d^2T}{dx^2} = 0$$

With boundary conditions:
- $T(0) = T_0$ (Dirichlet)
- $\frac{dT}{dx}(L) = q$ (Neumann)

### Analytical Solution

The analytical solution for this problem is:

$$T(x) = T_0 + q \cdot x$$

### Matrix Assembly Implementation

```cpp
// 1D Laplacian matrix assembly
template<class Type>
fvMatrix<Type> create1DLaplacian
(
    const fvMesh& mesh,
    const scalarField& gamma
) {
    const label nCells = mesh.cells().size();
    fvMatrix<Type> fvm(nCells);

    // Initialize source field
    fvm.source() = Field<Type>(nCells, 0.0);

    // Coefficient assembly
    for (label facei = 0; facei < mesh.nInternalFaces(); facei++)
    {
        const label own = mesh.faceOwner()[facei];
        const label nei = mesh.faceNeighbour()[facei];

        // Face area and distance
        scalar faceArea = mesh.magSf()[facei];
        scalar delta = mesh.magDistance()[facei];

        // Face coefficient
        scalar coeff = gamma[facei] * faceArea / delta;

        // Assemble matrix
        fvm.diag()[own] += coeff;  // Owner equation
        fvm.diag()[nei] += coeff;  // Neighbour equation
        fvm.upper()[0][facei] = -coeff;  // Upper coefficient
        fvm.lower()[0][facei] = -coeff;  // Lower coefficient
    }

    // Apply boundary conditions
    apply1DBoundaryConditions(fvm, mesh);

    return fvm;
}

// Boundary condition application
void apply1DBoundaryConditions
(
    fvMatrix<scalar>& fvm,
    const fvMesh& mesh
) {
    // Left boundary (Dirichlet)
    const label leftPatch = 0;
    const scalarField& leftValues = fvm.psi().boundaryField()[leftPatch];

    forAll(leftValues, i)
    {
        label celli = mesh.boundary()[leftPatch].faceCells()[i];
        fvm.diag()[celli] += 1.0;
        fvm.source()[celli] = leftValues[i];
    }

    // Right boundary (Neumann)
    const label rightPatch = mesh.boundary().size() - 1;
    const scalarField& rightGradient =
        fvm.psi().boundaryField()[rightPatch].gradient();

    forAll(rightGradient, i)
    {
        label celli = mesh.boundary()[rightPatch].faceCells()[i];
        fvm.source()[celli] += rightGradient[i];
    }
}
```

```cpp
void verify1DLaplacian

    const fvMesh& mesh,
    const scalar analyticalTolerance = 1e-6

    // Solve matrix system
    Field<scalar> T = lduMatrix::solver::solve(fvm);

    // Analytical solution
    const label nCells = mesh.cells().size();
    scalar L = mesh.bounds().max().x() - mesh.bounds().min().x();
```

```cpp
void analyzeMatrixMemoryUsage
(
    const fvMatrix<scalar>& fvm,
    const fvMesh& mesh
) {
    // Calculate memory usage
    label diagBytes = fvm.diag().size() * sizeof(scalar);
    label sourceBytes = fvm.source().size() * sizeof(scalar);

    // Off-diagonal storage
    label lowerBytes = 0, upperBytes = 0;
    forAll(fvm.lower(), i)
    {
        lowerBytes += fvm.lower()[i].size() * sizeof(scalar);
        upperBytes += fvm.upper()[i].size() * sizeof(scalar);

    label totalBytes = diagBytes + sourceBytes + lowerBytes + upperBytes;

    // Sparsity ratio
    label nCells = mesh.nCells();
    label nFaces = mesh.nFaces();
    label nnz = nCells + 2 * nFaces;  // Approximate non-zeros
    label denseSize = nCells * nCells;
```

## Advanced Assembly Techniques

### Deferred Assembly

For complex problems, deferred assembly can improve performance by separating coefficient calculation from matrix construction:

```cpp
// Deferred assembly implementation
template<class Type>
class DeferredFvMatrix
{
private:
    // Deferred coefficient storage
    PtrList<FieldField<Field, Type>> deferredLower_;
    PtrList<FieldField<Field, Type>> deferredUpper_;

    // Final assembly method
    void completeAssembly()
    {
        // Transfer deferred coefficients to final storage
        forAll(deferredLower_, i)
        {
            lower_[i] = deferredLower_[i];
            upper_[i] = deferredUpper_[i];
        }

        deferredLower_.clear();
        deferredUpper_.clear();
    }
};

// Usage example
DeferredFvMatrix<scalar> createDeferredLaplacian
(
    const fvMesh& mesh,
    const surfaceScalarField& coeffs
) {
    DeferredFvMatrix<scalar> fvm(mesh);

    // Face-based coefficient calculation
    forAll(mesh.faces(), facei)
    {
        if (facei < mesh.nInternalFaces())
        {
            const label own = mesh.faceOwner()[facei];
            const label nei = mesh.faceNeighbour()[facei];

            // Calculate coefficient
            scalar coeff = coeffs[facei];

            // Store in deferred format
            fvm.deferredLower()[0][facei] = -coeff;
            fvm.deferredUpper()[0][facei] = -coeff;
        }
    }

    // Complete assembly when needed
    fvm.completeAssembly();

    return fvm;
}
```

### Parallel Assembly

For large-scale problems, parallel matrix assembly is essential:

```cpp
// Parallel assembly implementation
void parallelAssembly
(
    fvMatrix<scalar>& fvm,
    const fvMesh& mesh,
    const surfaceScalarField& coeffs
) {
    // Get decomposition information
    const labelList& own = mesh.faceOwner();
    const labelList& nei = mesh.faceNeighbour();

    // Parallel reduce for coefficient summation
    Field<scalar> diagLocal(fvm.diag().size(), 0.0);
    Field<scalar> upperLocal(fvm.upper()[0].size(), 0.0);
    Field<scalar> lowerLocal(fvm.lower()[0].size(), 0.0);

    // Local assembly
    forAll(coeffs, facei)
    {
        if (facei < mesh.nInternalFaces())
        {
            scalar coeff = coeffs[facei];

            // Local contributions
            diagLocal[own[facei]] += coeff;
            diagLocal[nei[facei]] += coeff;
            upperLocal[facei] = -coeff;
            lowerLocal[facei] = -coeff;
        }
    }

    // Parallel communication
    Pstream::listCombineGather(diagLocal, plusEqOp<scalar>());
    Pstream::listCombineGather(upperLocal, plusEqOp<scalar>());
    Pstream::listCombineGather(lowerLocal, plusEqOp<scalar>());

    // Update matrix
    fvm.diag() = diagLocal;
    fvm.upper()[0] = upperLocal;
    fvm.lower()[0] = lowerLocal;
}
```

```cpp
// Exercise 2: Face-based assembly
template<class Type>
void exercise2_faceBasedAssembly
(
    fvMatrix<Type>& fvm,
    const VolField<Type>& T,
    const surfaceScalarField& k
) {
    const fvMesh& mesh = T.mesh();
    const label nInternalFaces = mesh.nInternalFaces();

    // Internal faces
    for (label facei = 0; facei < nInternalFaces; facei++) {
        const label own = mesh.faceOwner()[facei];
        const label nei = mesh.faceNeighbour()[facei];

        // Thermal resistance
        scalar faceConductivity = 0.5 * (k[own] + k[nei]);
        scalar faceArea = mesh.magSf()[facei];
        scalar delta = mesh.magDistance()[facei];

        // Heat transfer coefficient
        scalar coeff = faceConductivity * faceArea / delta;

        // Assemble matrix
        fvm.diag()[own] += coeff;
        fvm.diag()[nei] += coeff;
        fvm.upper()[0][facei] = -coeff;
        fvm.lower()[0][facei] = -coeff;
    }

    // Boundary faces
    forAll(mesh.boundary(), patchi) {
        const fvPatch<Type>& Tpatch = T.boundaryField()[patchi];
        const labelUList& faceCells = Tpatch.faceCells();

        forAll(faceCells, facei) {
            label celli = faceCells[facei];

            if (Tpatch.fixValue()) {
                // Dirichlet BC
                fvm.diag()[celli] += 1.0;
                fvm.source()[celli] = Tpatch[facei];
            }
        }
    }
}
```

### Exercise 4: Analytical Verification

```cpp
// Exercise 4: Analytical verification
void exercise4_analyticalVerification
(
    const fvMatrix<scalar>& fvm,
    const fvMesh& mesh,
    const scalar T0,
    const scalar TL
) {
    // Solve
    Field<scalar> T = lduMatrix::solver::solve(fvm);

    // Analytical solution for 2D steady-state heat conduction
    // T(x,y) = T0 + (TL - T0) * x / L
    const scalar L = mesh.bounds().max().x() - mesh.bounds().min().x();

    scalarField analyticalT(mesh.nCells());
    for (label i = 0; i < mesh.nCells(); i++) {
        scalar x = mesh.cellCentres()[i].x();
        analyticalT[i] = T0 + (TL - T0) * x / L;
    }

    // Calculate errors
    scalarField error = T - analyticalT;
    scalar maxError = gMax(error);
    scalar rmsError = sqrt(sum(error * error) / mesh.nCells());

    // Verify
    Info << "Analytical verification results:" << endl;
    Info << "  Maximum error: " << maxError << endl;
    Info << "  RMS error: " << rmsError << endl;
    Info << "  Tolerance: 1e-6" << endl;

    if (maxError > 1e-6) {
        FatalError << "Solution exceeds analytical tolerance" << endl;
    }
}

This exercise implements analytical verification for 2D heat conduction,
ensuring numerical accuracy by comparing against the analytical
solution for steady-state heat transfer between parallel plates.
```
