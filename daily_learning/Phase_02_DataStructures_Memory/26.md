# Day 26: Field<T> Memory - Alignment, Padding, SIMD Readiness

## 26.1 Memory Alignment Fundamentals (Memory Layout and SIMD Optimization)

### Introduction to Memory Alignment in Scientific Computing

In computational fluid dynamics, memory alignment is not just a performance consideration—it's a fundamental requirement for modern hardware optimization. When we work with large arrays of field data like velocity fields, pressure fields, or temperature fields, how we organize memory in RAM directly impacts computational efficiency.

```cpp
// Example: Naive field implementation (poor alignment)
class UnalignedField {
private:
    scalar* data;  // Pointer to contiguous memory
    label size;    // Number of elements

public:
    UnalignedField(label sz, scalar val, const char* name)
        : size(sz) {
        data = new scalar[size];  // Standard allocation - no guarantees
        for (label i = 0; i < size; ++i)
            data[i] = val;
    }

    scalar& operator[](label i) { return data[i]; }
};
```

**❌ PROBLEM:** This implementation allocates memory using `new[]`, which provides no alignment guarantees. On modern CPUs:

- SSE instructions require 16-byte alignment for optimal performance
- AVX instructions require 32-byte alignment
- AVX-512 instructions require 64-byte alignment

### Understanding Memory Architecture

Modern CPUs access memory through cache lines—typically 64-byte blocks. When we access misaligned data, the CPU may need to:

1. Fetch multiple cache lines
2. Extract the required bytes
3. Perform additional processing

This results in **significant performance penalties**.

```cpp
// Hardware-specific alignment requirements
#include <immintrin.h>

// Intel Architecture Reference
// SSE (Streaming SIMD Extensions): 16-byte alignment
// AVX (Advanced Vector Extensions): 32-byte alignment
// AVX-512: 64-byte alignment

// Performance impact of alignment
void test_alignment_impact() {
    const size_t size = 1000000;
    float* unaligned = new float[size];
    float* aligned = static_cast<float*>(_mm_malloc(size * sizeof(float), 32));

    // Access patterns
    for (size_t i = 0; i < size; i += 4) {
        // Unaligned access (slow)
        float sum1 = unaligned[i] + unaligned[i+1] + unaligned[i+2] + unaligned[i+3];

        // Aligned access (fast)
        __m128 vec = _mm_load_ps(aligned + i);  // Single instruction
        float sum2 = _mm_cvtss_f32(_mm_hadd_ps(vec, vec));
    }

    _mm_free(aligned);
    delete[] unaligned;
}
```

### Mathematical Foundations of SIMD Optimization

SIMD (Single Instruction, Multiple Data) operations enable us to perform the same operation on multiple data elements simultaneously. For field operations, this is particularly valuable.

Consider a vector field update operation:

```cpp
// Traditional scalar approach (slow)
for (label i = 0; i < N; ++i) {
    U[i] = U[i] + dt * ( -U[i] * dUdx[i] );
}

// SIMD approach (4x faster with AVX)
for (label i = 0; i < N; i += 8) {
    __m256 u_vec = _mm256_load_ps(&U[i]);
    __m256 dudx_vec = _mm256_load_ps(&dUdx[i]);
    __m256 dt_vec = _mm256_set1_ps(dt);

    __m256 result = _mm256_fmadd_ps(u_vec, dudx_vec, dt_vec);
    _mm256_store_ps(&U[i], _mm256_fnmadd_ps(result, dt_vec, u_vec));
}
```

The mathematical foundation lies in vectorizing operations:

$$
\mathbf{U}_{new} = \mathbf{U}_{old} - \Delta t \cdot \mathbf{U}_{old} \cdot \frac{\partial \mathbf{U}}{\partial x}
$$

When using AVX, we're effectively computing:

$$
\begin{bmatrix} U_0 \\ U_1 \\ U_2 \\ U_3 \\ U_4 \\ U_5 \\ U_6 \\ U_7 \end{bmatrix}_{new} =
\begin{bmatrix} U_0 \\ U_1 \\ U_2 \\ U_3 \\ U_4 \\ U_5 \\ U_6 \\ U_7 \end{bmatrix}_{old} -
\Delta t \cdot
\begin{bmatrix} U_0 \\ U_1 \\ U_2 \\ U_3 \\ U_4 \\ U_5 \\ U_6 \\ U_7 \end{bmatrix}_{old} \cdot
\begin{bmatrix} \frac{\partial U_0}{\partial x} \\ \frac{\partial U_1}{\partial x} \\ \frac{\partial U_2}{\partial x} \\ \frac{\partial U_3}{\partial x} \\ \frac{\partial U_4}{\partial x} \\ \frac{\partial U_5}{\partial x} \\ \frac{\partial U_6}{\partial x} \\ \frac{\partial U_7}{\partial x} \end{bmatrix}
$$

### Cache-Aware Memory Access Patterns

Memory alignment affects not just SIMD performance but also cache efficiency. Consider the memory access patterns:

```cpp
// Poor: Strided access (poor cache utilization)
for (label i = 0; i < N; ++i) {
    for (label j = 0; j < M; ++j) {
        field[i][j] = ...;  // Non-contiguous memory access
    }
}

// Good: Contiguous access (excellent cache utilization)
for (label j = 0; j < M; ++j) {
    for (label i = 0; i < N; ++i) {
        field[i][j] = ...;  // Contiguous memory access
    }
}
```

### Memory Bandwidth Considerations

Modern CPUs have limited memory bandwidth. Proper alignment helps maximize utilization:

```cpp
// Memory bandwidth calculation
void calculate_bandwidth_impact() {
    const size_t size = 1000000;  // 1 million elements
    const size_t iterations = 1000;

    // Aligned allocation
    float* aligned = static_cast<float*>(_mm_malloc(size * sizeof(float), 32));

    // Time aligned access
    clock_t start = clock();
    for (size_t iter = 0; iter < iterations; ++iter) {
        for (size_t i = 0; i < size; i += 8) {
            __m256 vec = _mm256_load_ps(&aligned[i]);
            _mm256_store_ps(&aligned[i], vec);  // SIMD operation
        }
    }
    clock_t aligned_time = clock() - start;

    // Calculate effective bandwidth
    size_t total_bytes = size * sizeof(float) * iterations;
    double aligned_bandwidth = static_cast<double>(total_bytes) /
                              (aligned_time * 1e-6 * 1024 * 1024);

    printf("Aligned bandwidth: %.2f MB/s\n", aligned_bandwidth);

    _mm_free(aligned);
}
```

### Alignment and Thread Safety

When working with multithreaded applications, alignment becomes even more critical:

```cpp
// Thread-safe aligned field implementation
template<class Type>
class ThreadSafeAlignedField {
private:
    std::atomic<Type*> dataPtr_;
    std::atomic<label> size_;
    const label alignment_;

public:
    ThreadSafeAlignedField(label size, const Type& val, const char* name, label alignment = 32)
        : alignment_(alignment) {
        size_.store(size);
        dataPtr_.store(static_cast<Type*>(
            ::aligned_alloc(alignment, size * sizeof(Type))
        ));

        // Initialize values
        Type* data = dataPtr_.load();
        for (label i = 0; i < size; ++i) {
            data[i] = val;
        }
    }

    // Thread-safe access
    Type& operator[](label i) {
        return dataPtr_.load()[i];
    }
};
```

### Enhanced Field Class with Alignment

```cpp

#ifndef ALIGNED_FIELD_H
#define ALIGNED_FIELD_H

#include <iostream>
#include <cstring>
#include <memory>
#include <immintrin.h>

template<class Type>
class AlignedField {
private:
    Type* dataPtr_;
    label size_;
    label alignment_;

    // Memory allocation with alignment
    static void* aligned_allocate(size_t size, size_t alignment) {
        void* ptr = nullptr;
        int result = posix_memalign(&ptr, alignment, size);
        if (result != 0) {
            throw std::runtime_error("Failed to allocate aligned memory");
        }
        return ptr;
    }

    // Memory deallocation
    static void aligned_deallocate(void* ptr) {
        free(ptr);
    }

public:
    // Constructor pattern: ClassName(size, value, "name")
    AlignedField(label size, const Type& val = Type(), const char* name = "field", label alignment = 32)
        : size_(size), alignment_(alignment) {

        if (size_ > 0) {
            // Allocate aligned memory
            dataPtr_ = static_cast<Type*>(aligned_allocate(size_ * sizeof(Type), alignment_));

            // Initialize values
            for (label i = 0; i < size_; ++i) {
                dataPtr_[i] = val;
            }
        } else {
            dataPtr_ = nullptr;
        }
    }

    // Destructor
    ~AlignedField() {
        if (dataPtr_) {
            aligned_deallocate(dataPtr_);
        }
    }

    // Copy constructor
    AlignedField(const AlignedField& other)
        : size_(other.size_), alignment_(other.alignment_) {
        if (size_ > 0) {
            dataPtr_ = static_cast<Type*>(aligned_allocate(size_ * sizeof(Type), alignment_));
            std::memcpy(dataPtr_, other.dataPtr_, size_ * sizeof(Type));
        } else {
            dataPtr_ = nullptr;
        }
    }

    // Move constructor
    AlignedField(AlignedField&& other) noexcept
        : dataPtr_(other.dataPtr_), size_(other.size_), alignment_(other.alignment_) {
        other.dataPtr_ = nullptr;
        other.size_ = 0;
    }

    // Copy assignment
    AlignedField& operator=(const AlignedField& other) {
        if (this != &other) {
            if (dataPtr_) {
                aligned_deallocate(dataPtr_);
            }

            size_ = other.size_;
            alignment_ = other.alignment_;

            if (size_ > 0) {
                dataPtr_ = static_cast<Type*>(aligned_allocate(size_ * sizeof(Type), alignment_));
                std::memcpy(dataPtr_, other.dataPtr_, size_ * sizeof(Type));
            } else {
                dataPtr_ = nullptr;
            }
        }
        return *this;
    }

    // Move assignment
    AlignedField& operator=(AlignedField&& other) noexcept {
        if (this != &other) {
            if (dataPtr_) {
                aligned_deallocate(dataPtr_);
            }

            dataPtr_ = other.dataPtr_;
            size_ = other.size_;
            alignment_ = other.alignment_;

            other.dataPtr_ = nullptr;
            other.size_ = 0;
        }
        return *this;
    }

    // Access operators
    Type& operator[](const label i) {
        if (i >= size_) {
            throw std::out_of_range("Index out of range");
        }
        return dataPtr_[i];
    }

    const Type& operator[](const label i) const {
        if (i >= size_) {
            throw std::out_of_range("Index out of range");
        }
        return dataPtr_[i];
    }

    // Raw pointer access (for SIMD operations)
    Type* data() { return dataPtr_; }
    const Type* data() const { return dataPtr_; }

    // Size information
    label size() const { return size_; }

    // Iterator support
    Type* begin() { return dataPtr_; }
    Type* end() { return dataPtr_ + size_; }
    const Type* begin() const { return dataPtr_; }
    const Type* end() const { return dataPtr_ + size_; }

    // SIMD operations
    void simd_add(const AlignedField& other) {
        if (size_ != other.size_) {
            throw std::invalid_argument("Field sizes must match");
        }

        // SIMD-accelerated addition
        for (label i = 0; i < size_; i += 8) {
            const label remaining = size_ - i;
            if (remaining >= 8) {
                // Process 8 elements at once
                __m256 a = _mm256_load_ps(dataPtr_ + i);
                __m256 b = _mm256_load_ps(other.dataPtr_ + i);
                __m256 result = _mm256_add_ps(a, b);
                _mm256_store_ps(dataPtr_ + i, result);
            } else {
                // Process remaining elements
                for (label j = 0; j < remaining; ++j) {
                    dataPtr_[i + j] += other.dataPtr_[i + j];
                }
            }
        }
    }

    void simd_multiply(const AlignedField& other) {
        if (size_ != other.size_) {
            throw std::invalid_argument("Field sizes must match");
        }

        // SIMD-accelerated multiplication
        for (label i = 0; i < size_; i += 8) {
            const label remaining = size_ - i;
            if (remaining >= 8) {
                __m256 a = _mm256_load_ps(dataPtr_ + i);
                __m256 b = _mm256_load_ps(other.dataPtr_ + i);
                __m256 result = _mm256_mul_ps(a, b);
                _mm256_store_ps(dataPtr_ + i, result);
            } else {
                for (label j = 0; j < remaining; ++j) {
                    dataPtr_[i + j] *= other.dataPtr_[i + j];
                }
            }
        }
    }

    // Memory alignment utilities
    bool is_aligned() const {
        return (reinterpret_cast<uintptr_t>(dataPtr_) % alignment_) == 0;
    }

    label get_alignment() const {
        return alignment_;
    }
```

```cpp
#define SIMD_FIELD_OPS_H

// SIMD operations for scalar fields (float)
template<>
void AlignedField<float>::simd_multiply(const AlignedField<float>& other) {
    if (size_ != other.size_) {
        throw std::invalid_argument("Field sizes must match");
    }

    for (label i = 0; i < size_; i += 8) {
        const label remaining = size_ - i;
        if (remaining >= 8) {
            // Load 8 floats
            __m256 a = _mm256_load_ps(dataPtr_ + i);
            __m256 b = _mm256_load_ps(other.dataPtr_ + i);

            // Multiply
            __m256 result = _mm256_mul_ps(a, b);

            // Store result
            _mm256_store_ps(dataPtr_ + i, result);
        } else {
            // Handle remaining elements
            for (label j = 0; j < remaining; ++j) {
                dataPtr_[i + j] *= other.dataPtr_[i + j];
            }
        }
    }
}

// Vector operations for Vector3 fields
#include <OpenFOAM/Vector/Vector.H>

template<>
void AlignedField<OpenFOAM::Vector>::simd_multiply(const AlignedField<OpenFOAM::Vector>& other) {
    if (size_ != other.size_) {
        throw std::invalid_argument("Field sizes must match");
    }

    for (label i = 0; i < size_; i += 4) {  // 4 vectors * 3 components = 12 floats
        const label remaining = size_ - i;
        if (remaining >= 4) {
            // Load vectors (3 components each)
            __m128 a_x = _mm_load_ps(&dataPtr_[i].x());
            __m128 a_y = _mm_load_ps(&dataPtr_[i].y());
            __m128 a_z = _mm_load_ps(&dataPtr_[i].z());

            __m128 b_x = _mm_load_ps(&other.dataPtr_[i].x());
            __m128 b_y = _mm_load_ps(&other.dataPtr_[i].y());
            __m128 b_z = _mm_load_ps(&other.dataPtr_[i].z());

            // Multiply component-wise
            __m128 r_x = _mm_mul_ps(a_x, b_x);
            __m128 r_y = _mm_mul_ps(a_y, b_y);
            __m128 r_z = _mm_mul_ps(a_z, b_z);

            // Store results
            _mm_store_ps(&dataPtr_[i].x(), r_x);
            _mm_store_ps(&dataPtr_[i].y(), r_y);
            _mm_store_ps(&dataPtr_[i].z(), r_z);
        } else {
            // Handle remaining elements
            for (label j = 0; j < remaining; ++j) {
                dataPtr_[i + j] *= other.dataPtr_[i + j];
            }
        }
    }

// Mathematical operations
template<class Type>
class FieldOperations {
public:
    // Dot product of two fields
    static float dot_product(const AlignedField<Type>& a, const AlignedField<Type>& b) {
        if (a.size() != b.size()) {
            throw std::invalid_argument("Field sizes must match");
        }

        float sum = 0.0f;

        // SIMD-accelerated dot product
        if constexpr (std::is_same_v<Type, float>) {
            for (label i = 0; i < a.size(); i += 8) {
                const label remaining = a.size() - i;
                if (remaining >= 8) {
                    __m256 vec_a = _mm256_load_ps(a.data() + i);
                    __m256 vec_b = _mm256_load_ps(b.data() + i);
                    __m256 product = _mm256_mul_ps(vec_a, vec_b);
                    __m256 sum_vec = _mm256_hadd_ps(product, product);
                    sum += _mm_cvtss_f32(_mm256_extractf128_ps(sum_vec, 0));
                } else {
                    for (label j = 0; j < remaining; ++j) {
                        sum += a[i + j] * b[i + j];
                    }
                }
            }
        } else {
            // For vector types
            for (label i = 0; i < a.size(); ++i) {
                sum += dot(a[i], b[i]);
            }
        }

        return sum;
    }

    // L2 norm of a field
    static float norm(const AlignedField<Type>& field) {
        return std::sqrt(dot_product(field, field));
    }

    // Normalize field to unit vector
    static void normalize(AlignedField<Type>& field) {
        float n = norm(field);
        if (n > 1e-10f) {
            field.simd_multiply(AlignedField<Type>(field.size(), 1.0f / n));
        }
    }
};
```

### Exercise 1: Memory Alignment Fundamentals

**Question:** Explain why memory alignment is critical for SIMD operations in CFD simulations. Calculate the performance penalty for accessing misaligned data on a modern CPU.

**Solution:**

Memory alignment is critical for SIMD operations because:

1. **Hardware Requirements**: SSE instructions require 16-byte alignment, AVX requires 32-byte alignment, and AVX-512 requires 64-byte alignment.

2. **Performance Penalty**: When accessing misaligned data, the CPU must:
   - Fetch multiple cache lines (typically 64 bytes each)
   - Extract the required bytes
   - Perform additional processing

The performance penalty can be calculated as follows:

```cpp
// Performance penalty calculation
void calculate_alignment_penalty() {
    const size_t size = 1000000;
    float* unaligned = new float[size];
    float* aligned = static_cast<float*>(_mm_malloc(size * sizeof(float), 32));

    // Time unaligned access
    clock_t start = clock();
    float sum1 = 0.0f;
    for (size_t i = 0; i < size; ++i) {
        sum1 += unaligned[i] * unaligned[i];
    }
    clock_t unaligned_time = clock() - start;

    // Time aligned access
    start = clock();
    float sum2 = 0.0f;
    for (size_t i = 0; i < size; i += 8) {
        __m256 vec = _mm256_load_ps(aligned + i);
        __m256 squares = _mm256_mul_ps(vec, vec);
        __m128 sum128 = _mm256_extractf128_ps(squares, 0);
        sum2 += _mm_cvtss_f32(sum128) + _mm_cvtss_f32(_mm_shuffle_ps(sum128, sum128, 0xB1));
    }
    clock_t aligned_time = clock() - start;

    double penalty = static_cast<double>(unaligned_time) / aligned_time;
    printf("Performance penalty: %.2fx\n", penalty);

    _mm_free(aligned);
    delete[] unaligned;
}
```

**Expected Result:** Performance penalty typically ranges from 2-10x depending on the CPU architecture and memory access pattern.

```cpp
void vectorized_heat_solver() {

    const float alpha = 0.01f;
    const float dt = 0.0001f;

    AlignedField<float> T(N, 0.0f);  // Temperature field
    AlignedField<float> T_new(N, 0.0f);

    // Set initial conditions
    for (label i = N/4; i < 3*N/4; ++i) {
        T[i] = 100.0f;  // Hot region
    }

    // Time stepping with SIMD
    auto start = std::chrono::high_resolution_clock::now();
    for (int step = 0; step < 1000; ++step) {
        // Vectorized update: T_new[i] = T[i] + alpha*dt*(T[i+1] - 2*T[i] + T[i-1])
        for (label i = 1; i < N-1; i += 8) {
            const label remaining = (N-1) - i;
            if (remaining >= 8) {
                __m256 t = _mm256_load_ps(T.data() + i);
                __m256 t_prev = _mm256_load_ps(T.data() + i - 1);
                __m256 t_next = _mm256_load_ps(T.data() + i + 1);

                __m256 laplacian = _mm256_sub_ps(
                    _mm256_add_ps(t_next, t_prev),
                    _mm256_mul_ps(_mm256_set1_ps(2.0f), t)
                );

                __m256 dt_alpha = _mm256_set1_ps(alpha * dt);
                __m256 update = _mm256_mul_ps(dt_alpha, laplacian);

                _mm256_store_ps(T_new.data() + i, _mm256_add_ps(t, update));
            } else {
                for (label j = 0; j < remaining; ++j) {
                    T_new[i + j] = T[i + j] + alpha * dt *
                                 (T[i + j + 1] - 2 * T[i + j] + T[i + j - 1]);
                }
            }
        }

        // Swap fields
        std::swap(T, T_new);
    }
    auto end = std::chrono::high_resolution_clock::now();

    auto vectorized_time = std::chrono::duration_cast<std::chrono::milliseconds>(end - start);
    double vectorized_bandwidth = (static_cast<double>(N) * sizeof(float) * 1000) /
                                (vectorized_time.count() * 1e-6 * 1024 * 1024);

    printf("Vectorized time: %lld ms\n", vectorized_time.count());
    printf("Vectorized bandwidth: %.2f MB/s\n", vectorized_bandwidth);
```

```cpp
#include <cpuid.h>

class SIMDSelector {
public:
    enum class SIMDLevel {
        NONE,
        SSE,
        SSE2,
        SSE3,
        SSSE3,
        SSE41,
        SSE42,
        AVX,
        AVX2,
        AVX512
    };

    static SIMDLevel detect_capabilities() {
        SIMDLevel level = SIMDLevel::NONE;

        unsigned int eax, ebx, ecx, edx;

        // Check for SSE
        if (__get_cpuid(1, &eax, &ebx, &ecx, &edx)) {
            if (edx & (1 << 25)) level = SIMDLevel::SSE;
            if (edx & (1 << 26)) level = SIMDLevel::SSE2;
            if (ecx & (1 << 0)) level = SIMDLevel::SSE3;
            if (ecx & (1 << 9)) level = SIMDLevel::SSSE3;
            if (ecx & (1 << 19)) level = SIMDLevel::SSE41;
            if (ecx & (1 << 20)) level = SIMDLevel::SSE42;

            // Check for AVX
            if (ecx & (1 << 28)) level = SIMDLevel::AVX;

            // Check for AVX2
            if (__get_cpuid(7, &eax, &ebx, &ecx, &edx)) {
                if (ebx & (1 << 5)) level = SIMDLevel::AVX2;
            }

            // Check for AVX512
            if (__get_cpuid(7, &eax, &ebx, &ecx, &edx)) {
                if (ebx & (1 << 16)) level = SIMDLevel::AVX512;
            }
        }

        return level;
    }

    static void print_capabilities() {
        auto level = detect_capabilities();
        printf("Detected SIMD capabilities: ");

        switch (level) {
            case SIMDLevel::NONE: printf("None\n"); break;
            case SIMDLevel::SSE: printf("SSE\n"); break;
            case SIMDLevel::SSE2: printf("SSE2\n"); break;
            case SIMDLevel::SSE3: printf("SSE3\n"); break;
            case SIMDLevel::SSSE3: printf("SSSE3\n"); break;
            case SIMDLevel::SSE41: printf("SSE4.1\n"); break;
            case SIMDLevel::SSE42: printf("SSE4.2\n"); break;
            case SIMDLevel::AVX: printf("AVX\n"); break;
            case SIMDLevel::AVX2: printf("AVX2\n"); break;
            case SIMDLevel::AVX512: printf("AVX512\n"); break;
        }
    }

    // Optimized field operation based on detected capabilities
    template<class Type>
    static void optimized_multiply(AlignedField<Type>& a, const AlignedField<Type>& b) {
        auto level = detect_capabilities();

        switch (level) {
            case SIMDLevel::AVX512:
                // AVX512-optimized implementation
                for (label i = 0; i < a.size(); i += 16) {
                    const label remaining = a.size() - i;
                    if (remaining >= 16) {
                        __m512 va = _mm512_load_ps(a.data() + i);
                        __m512 vb = _mm512_load_ps(b.data() + i);
                        _mm512_store_ps(a.data() + i, _mm512_mul_ps(va, vb));
                    } else {
                        // Fallback to AVX
                        for (label j = 0; j < remaining; j += 8) {
                            __m256 va = _mm256_load_ps(a.data() + i + j);
                            __m256 vb = _mm256_load_ps(b.data() + i + j);
                            _mm256_store_ps(a.data() + i + j, _mm256_mul_ps(va, vb));
                        }
                    }
                }
                break;

            case SIMDLevel::AVX2:
                // AVX-optimized implementation
                for (label i = 0; i < a.size(); i += 8) {
                    const label remaining = a.size() - i;
                    if (remaining >= 8) {
                        __m256 va = _mm256_load_ps(a.data() + i);
                        __m256 vb = _mm256_load_ps(b.data() + i);
                        _mm256_store_ps(a.data() + i, _mm256_mul_ps(va, vb));
                    } else {
                        // Fallback to SSE
                        for (label j = 0; j < remaining; ++j) {
                            a[i + j] *= b[i + j];
                        }
                    }
                }
                break;

            default:
                // Scalar fallback
                for (label i = 0; i < a.size(); ++i) {
                    a[i] *= b[i];
                }
                break;
        }
    }
```

```cpp
#include <unordered_map>
#include <chrono>

class MemoryProfiler {
private:
    struct AllocationInfo {
        size_t size;
        std::chrono::time_point<std::chrono::high_resolution_clock> alloc_time;
        std::chrono::time_point<std::chrono::high_resolution_clock> last_access;
        int access_count;
    };

    std::unordered_map<void*, AllocationInfo> allocations_;
    std::unordered_map<std::string, size_t> type_allocations_;

public:
    void* allocate(size_t size, const char* type) {
        void* ptr = malloc(size);
        if (ptr) {
            allocations_[ptr] = {size, std::chrono::high_resolution_clock::now(),
                               std::chrono::high_resolution_clock::now(), 0};
            type_allocations_[type] += size;
        }
        return ptr;
    }

    void deallocate(void* ptr) {
        auto it = allocations_.find(ptr);
        if (it != allocations_.end()) {
            size_t size = it->second.size;
            const char* type = "unknown";  // Could enhance to track types

            type_allocations_[type] -= size;
            allocations_.erase(it);
        }
        free(ptr);
    }

    void record_access(void* ptr) {
        auto it = allocations_.find(ptr);
        if (it != allocations_.end()) {
            it->second.last_access = std::chrono::high_resolution_clock::now();
            it->second.access_count++;
        }
    }

    void print_report() {
        printf("=== Memory Allocation Report ===\n");
        printf("Total allocations: %zu\n", allocations_.size());

        // Type-based allocation summary
        printf("\nType-based allocations:\n");
        for (const auto& [type, size] : type_allocations_) {
            printf("%s: %zu bytes\n", type.c_str(), size);
        }

        // Hot allocations (frequently accessed)
        printf("\nHot allocations (access count > 100):\n");
        for (const auto& [ptr, info] : allocations_) {
            if (info.access_count > 100) {
                auto now = std::chrono::high_resolution_clock::now();
                auto duration = std::chrono::duration_cast<std::chrono::seconds>(
                    now - info.alloc_time).count();

                printf("Ptr: %p, Size: %zu bytes, Accesses: %d, Age: %ld seconds\n",
                       ptr, info.size, info.access_count, duration);
            }
        }
    }
};

// Usage example
void profile_cfd_simulation() {
    MemoryProfiler profiler;

    // Profile field allocations
    void* velocity_field = profiler.allocate(1000000 * sizeof(float), "velocity");
    void* pressure_field = profiler.allocate(1000000 * sizeof(float), "pressure");

    // Simulate field operations
    for (int iter = 0; iter < 1000; ++iter) {
        // Access velocity field frequently
        profiler.record_access(velocity_field);

        // Access pressure field less frequently
        if (iter % 10 == 0) {
            profiler.record_access(pressure_field);
        }
    }

    profiler.print_report();

    // Cleanup
    profiler.deallocate(velocity_field);
    profiler.deallocate(pressure_field);
}
```
