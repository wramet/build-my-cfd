# Day 38: Modern Error Handling — Exceptions and `std::system_error`

## Part 1: Pattern Identification

### The Error Handling Problem

CFD code can fail in many ways:
- File I/O errors (missing mesh, can't write output)
- Solver convergence failure
- Invalid configuration
- Memory allocation failures
- MPI communication errors

**Challenge:** How to handle errors gracefully while maintaining code correctness?

### Error Handling Approaches

```cpp
// Approach 1: Return codes (C-style)
int solve() {
    if (!converged()) return -1;  // Error code
    return 0;  // Success
}

// Problems:
// - Easy to ignore return values
// - No error information propagation
// - Cleanup manual and error-prone
```

```cpp
// Approach 2: Exceptions (C++ style)
void solve() {
    if (!converged()) {
        throw std::runtime_error("Solver failed to converge");
    }
}

// Benefits:
// - Can't be ignored (must catch or terminate)
// - Automatic cleanup (RAII)
// - Rich error information
```

### Exception Safety

```cpp
// Exception-safe code guarantees:
// 1. No resource leaks
// 2. Objects remain valid
// 3. Program state consistent

class Solver {
    std::vector<double> matrix_;

public:
    void solve() {
        std::vector<double> temp(matrix_.size());
        // If exception thrown here:
        // - matrix_ still valid (not modified)
        // - temp cleaned up automatically
        // - stack unwinding destroys temp
    }
};
```

## Part 2: Theory — Exception Hierarchy

### Standard Exception Hierarchy

```cpp
std::exception
├── std::runtime_error
│   ├── std::overflow_error
│   ├── std::range_error
│   └── std::system_error  // For OS/errno errors
├── std::logic_error
│   ├── std::invalid_argument
│   ├── std::out_of_range
│   └── std::domain_error
└── std::bad_alloc
```

### When to Use Which

| Exception Type | Use Case |
|----------------|----------|
| `std::invalid_argument` | Wrong parameter value |
| `std::out_of_range` | Index out of bounds |
| `std::runtime_error` | Runtime failure (solver divergence) |
| `std::system_error` | OS/library call failure |
| `std::bad_alloc` | Memory allocation failure |

```cpp
void solve(double tolerance, int maxIter) {
    if (tolerance <= 0) {
        throw std::invalid_argument("Tolerance must be positive");
    }

    if (maxIter <= 0) {
        throw std::invalid_argument("Max iterations must be positive");
    }

    bool converged = false;

        std::cerr << "Invalid argument: " << e.what() << std::endl;
        return 1;

        std::cerr << "Runtime error: " << e.what() << std::endl;
        return 2;
```

### System Error Handling

```cpp
#include <system_error>
#include <fstream>

void writeFile(const std::string& filename, const std::vector<double>& data) {
    std::ofstream out(filename, std::ios::binary);

    if (!out) {
        // Preserve errno for detailed error info
        throw std::system_error(
            errno,
            std::generic_category(),
            "Cannot open file: " + filename
        );
    }

    out.write(reinterpret_cast<const char*>(data.data()),
             data.size() * sizeof(double));

    if (!out) {
        throw std::system_error(
            errno,
            std::generic_category(),
            "Write failed for: " + filename
        );
    }
}

int main() {
    try {
        std::vector<double> data(100, 1.0);
        writeFile("output.bin", data);
    } catch (const std::system_error& e) {
        std::cerr << "System error: " << e.what() << std::endl;
        std::cerr << "Code: " << e.code() << std::endl;
        std::cerr << "Message: " << e.code().message() << std::endl;
        return 1;
    }

    return 0;
}
```

### Comprehensive Error System

```cpp
// ErrorCodes.H
#pragma once
#include <system_error>
#include <string>

namespace CFDErrors {
    enum class SolverErrc {
        Success = 0,
        NotConverged,
        SingularMatrix,
        InvalidTolerance,
        MaxIterationsExceeded
    };

    enum class MeshErrc {
        Success = 0,
        FileNotFound,
        InvalidFormat,
        NonManifoldGeometry,
        InvalidBoundary
    };

    enum class IOErrc {
        Success = 0,
        FileOpenFailed,
        FileWriteFailed,
        FileReadFailed,
        InvalidFormat
    };
}

// Define error categories for std::system_error
template<typename Errc>
struct CFDErrorCategory : std::error_category {
    const char* name() const noexcept override {
        return "CFD error";
    }

    std::string message(int ev) const override {
        switch (static_cast<Errc>(ev)) {
            case Errc::Success: return "Success";
            default: return "Unknown error";
        }
    }
};

// Specialize for each error type
template<>
std::string CFDErrorCategory<CFDErrors::SolverErrc>::message(int ev) const {
    switch (static_cast<CFDErrors::SolverErrc>(ev)) {
        case CFDErrors::SolverErrc::Success:
            return "Success";
        case CFDErrors::SolverErrc::NotConverged:
            return "Solver failed to converge";
        case CFDErrors::SolverErrc::SingularMatrix:
            return "Singular matrix encountered";
        case CFDErrors::SolverErrc::InvalidTolerance:
            return "Invalid tolerance value";
        case CFDErrors::SolverErrc::MaxIterationsExceeded:
            return "Maximum iterations exceeded";
        default:
            return "Unknown solver error";
    }
}

template<>
std::string CFDErrorCategory<CFDErrors::MeshErrc>::message(int ev) const {
    switch (static_cast<CFDErrors::MeshErrc>(ev)) {
        case CFDErrors::MeshErrc::FileNotFound:
            return "Mesh file not found";
        case CFDErrors::MeshErrc::InvalidFormat:
            return "Invalid mesh format";
        case CFDErrors::MeshErrc::NonManifoldGeometry:
            return "Non-manifold geometry detected";
        case CFDErrors::MeshErrc::InvalidBoundary:
            return "Invalid boundary definition";
        default:
            return "Unknown mesh error";
    }
}

// Get error category
inline const CFDErrorCategory<CFDErrors::SolverErrc>& solver_category() {
    static CFDErrorCategory<CFDErrors::SolverErrc> category;
    return category;
}

inline const CFDErrorCategory<CFDErrors::MeshErrc>& mesh_category() {
    static CFDErrorCategory<CFDErrors::MeshErrc> category;
    return category;
}

inline std::system_error make_sys_error(int err, const std::error_category& cat) {
```

### Solver with Error Handling

```cpp
// LinearSolver.H
#pragma once
#include "ErrorCodes.H"
#include <vector>
#include <stdexcept>

class LinearSolver {
protected:
    const std::vector<double>& matrix_;
    std::vector<double>& solution_;
    const std::vector<double>& rhs_;
    double tolerance_;
    int maxIterations_;

    // Check if matrix is singular
    bool isSingular() const {
        int n = solution_.size();
        for (int i = 0; i < n; ++i) {
            if (std::abs(matrix_[i * n + i]) < 1e-12) {
                return true;
            }
        }
        return false;
    }

public:
    LinearSolver(const std::vector<double>& matrix,
                 std::vector<double>& solution,
                 const std::vector<double>& rhs,
                 double tolerance = 1e-6,
                 int maxIterations = 1000)
        : matrix_(matrix), solution_(solution), rhs_(rhs)
        , tolerance_(tolerance), maxIterations_(maxIterations)
    {
        if (tolerance_ <= 0) {
            throw make_sys_error(
                static_cast<int>(CFDErrors::SolverErrc::InvalidTolerance),
                solver_category()
            );
        }

        if (maxIterations_ <= 0) {
            throw std::invalid_argument("Max iterations must be positive");
        }
    }

    virtual ~LinearSolver() = default;

    virtual void solve() = 0;

    virtual std::string name() const = 0;
};
```

```cpp
// GaussSeidelSolver.C
#include "LinearSolver.H"
#include <iostream>

class GaussSeidelSolver : public LinearSolver {
public:
    using LinearSolver::LinearSolver;

    void solve() override {
        if (isSingular()) {
            throw make_sys_error(
                static_cast<int>(CFDErrors::SolverErrc::SingularMatrix),
                solver_category()
            );
        }

        std::cout << "Solving with " << name() << std::endl;
        std::cout << "  Tolerance: " << tolerance_ << std::endl;
        std::cout << "  Max iterations: " << maxIterations_ << std::endl;

        int n = solution_.size();

        for (int iter = 0; iter < maxIterations_; ++iter) {
            double maxError = 0.0;

            for (int i = 0; i < n; ++i) {
                double oldValue = solution_[i];
                double sum = rhs_[i];

                for (int j = 0; j < i; ++j) {
                    sum -= matrix_[i * n + j] * solution_[j];
                }
                for (int j = i + 1; j < n; ++j) {
                    sum -= matrix_[i * n + j] * solution_[j];
                }

                solution_[i] = sum / matrix_[i * n + i];
                maxError = std::max(maxError, std::abs(solution_[i] - oldValue));
            }

            if (iter % 100 == 0) {
                std::cout << "  Iter " << iter
                          << ", residual = " << maxError << std::endl;
            }

            if (maxError < tolerance_) {
                std::cout << "Converged in " << iter << " iterations" << std::endl;
                return;
            }
        }

        throw make_sys_error(
            static_cast<int>(CFDErrors::SolverErrc::NotConverged),
            solver_category()
        );
    }

    std::string name() const override {
        return "Gauss-Seidel";
    }
};
```

```cpp
// main.C
#include "LinearSolver.H"

#include <iostream>

int main() {

        // Create test problem
        const int n = 100;
        std::vector<double> matrix(n * n, 0.0);
        std::vector<double> solution(n, 0.0);
        std::vector<double> rhs(n, 1.0);

        // Tridiagonal matrix
        for (int i = 0; i < n; ++i) {
            matrix[i * n + i] = 2.0;
            if (i > 0) matrix[i * n + (i - 1)] = -1.0;
            if (i < n - 1) matrix[i * n + (i + 1)] = -1.0;

        std::cerr << "Error code: " << e.code() << std::endl;
        return e.code().value();
    } catch (const std::exception& e) {
        std::cerr << "Error: " << e.what() << std::endl;
        return 1;
    } catch (...) {
        std::cerr << "Unknown error" << std::endl;
        return 2;
```

## Part 5: Trade-offs

### Exceptions vs Return Codes

| Aspect | Exceptions | Return Codes |
|--------|------------|-------------|
| **Performance** | Slight overhead when thrown | Minimal overhead |
| **Safety** | Can't be ignored | Easy to ignore |
| **Information** | Rich (type, message, stack) | Limited (integer code) |
| **Cleanup** | Automatic (RAII) | Manual |
| **Best for** | Exceptional conditions | Expected failures |

### When to Use Exceptions

**Use exceptions for:**
- Constructor failures
- Invalid arguments
- Resource acquisition failures
- Solver divergence
- File I/O errors

**Use return codes for:**
- Expected conditions (not found is OK)
- Performance-critical inner loops
- Optional operations

### Best Practices

1. **Throw by value, catch by reference**
   ```cpp
   throw std::runtime_error("error");
   catch (const std::runtime_error& e) { }
   ```

2. **Provide context in error messages**
   ```cpp
   throw std::runtime_error("Solver failed to converge at t=" +
                               std::to_string(time) + ", residual=" +
                               std::to_string(residual));
   ```

3. **Use RAII for cleanup**
   ```cpp
   class FileWrapper {
       std::ofstream file_;
   public:
       FileWrapper(const std::string& name) : file_(name) {
           if (!file_) throw std::runtime_error("Cannot open " + name);
       }
       ~FileWrapper() { /* automatic close */ }
   };
   ```
