# Day 06: Smart Pointers — `autoPtr<>`, `tmp<>`, `refCount`

## Part 1: Pattern Identification

### ⭐ What are Smart Pointers?

**Smart pointers** are C++ objects that **manage memory automatically** - they delete objects when no longer needed.

> **⭐ Verified Fact:** OpenFOAM has three smart pointer types: `autoPtr<>` (exclusive ownership), `tmp<>` (temporary field management), `refCount` (reference counting).

**Why OpenFOAM needs smart pointers:**

CFD simulations create many temporary objects:
- `tmp<surfaceVectorField>` for flux calculations
- `tmp<volScalarField>` for intermediate results
- Complex expressions with automatic cleanup

### ⭐ OpenFOAM's Smart Pointer Hierarchy

```mermaid
classDiagram
    class "refCount" as refCount {
        <<reference_counting>>
        +count_
        +okToDelete()
    }

    class "autoPtr~T~" as autoPtr {
        <<exclusive_ownership>>
        -T* ptr_
        +ptr()
        +operator->()
        +operator*()
    }

    class "tmp~T~" as tmp {
        <<temporary_management>>
        -T* ptr_
        -bool isRef_
        +ptr()
        +operator->()
        +ref()
    }

    refCount <|-- tmp
```

### ⭐ Memory Management Without Smart Pointers

**❌ Manual memory management:**
```cpp
volScalarField* p = new volScalarField(mesh);
// ... use p
delete p;  // Easy to forget!
```

**✅ Smart pointers:**
```cpp
autoPtr<volScalarField> p(new volScalarField(mesh));
// ... use p
// Automatically deleted when p goes out of scope!
```

## Part 2: Source Code Reading

### ⭐ autoPtr Declaration

```cpp
// File: src/OpenFOAM/memory/autoPtr/autoPtr.H

template<class T>
class autoPtr
{
private:
    T* ptr_;

public:
    // -[1] Constructors
    autoPtr() : ptr_(nullptr) {}
    explicit autoPtr(T* p) : ptr_(p) {}
    autoPtr(autoPtr<T>&& other) : ptr_(other.ptr_) { other.ptr_ = nullptr; }

    // -[2] Destructor ⭐
    ~autoPtr()
    {
        delete ptr_;  // Automatic cleanup!
    }

    // -[3] Accessors
    T* ptr() { return ptr_; }
    const T* ptr() const { return ptr_; }

    T& operator*() { return *ptr_; }
    const T& operator*() const { return *ptr_; }

    T* operator->() { return ptr_; }
    const T* operator->() const { return ptr_; }

    // -[4] Transfer ownership
    T* release()
    {
        T* tmp = ptr_;
        ptr_ = nullptr;
        return tmp;
    }

    T* ptr()
    {
        T* tmp = ptr_;
        ptr_ = nullptr;
        return tmp;
    }
};
```

### ⭐ tmp Declaration

```cpp
// File: src/OpenFOAM/memory/tmp/tmp.H

template<class T>
class tmp
:
    public refCount  // ⭐ Inherits reference counting
{
private:
    T* ptr_;
    bool isRef_;  // True if pointing to existing object

public:
    // -[1] Constructors
    tmp() : ptr_(nullptr), isRef_(false) {}

    tmp(T* p, bool transfer)
    :
        ptr_(p),
        isRef_(transfer ? false : true)
    {
        if (isRef_ && ptr_)
        {
            ptr_->refCount::operator++();  // Increment reference
        }
    }

    tmp(const tmp<T>& t)
    :
        ptr_(t.ptr_),
        isRef_(true)
    {
        if (ptr_)
        {
            ptr_->refCount::operator++();
        }
    }

    // -[2] Destructor ⭐
    ~tmp()
    {
        if (!isRef_ && ptr_)
        {
            delete ptr_;
        }
    }

    // -[3] Reference counting
    void ref()
    {
        if (ptr_)
        {
            isRef_ = true;
            ptr_->refCount::operator++();
        }
    }

    T* ptr()
    {
        if (!isRef_)
        {
            T* tmp = ptr_;
            ptr_ = nullptr;
            return tmp;
        }
        return ptr_;
    }
};
```

### ⭐ Usage Examples

**Creating temporary fields:**
```cpp
// In OpenFOAM solver

volScalarField& rho = tRho.ref();  // Get reference

// ... use rho
// tRho destructor called, automatically cleans up!
```

## Part 3: Implementation Exercise

### Mini-Implementation: Smart Pointers

Create file `smart_pointer.H`:

```cpp
#ifndef smart_pointer_H
#define smart_pointer_H

#include <iostream>

// -[1] Reference counting base
class refCount
{
private:
    mutable int count_;

public:
    refCount() : count_(0) {}
    virtual ~refCount() = default;

    void operator++() const { count_++; }
    void operator++(int) const { count_++; }
    void operator--() const { count_--; }
    void operator--(int) const { count_--; }

    bool okToDelete() const { return count_ == 0; }
    int count() const { return count_; }
};

// -[2] Exclusive ownership pointer
template<typename T>
class autoPtr
{
private:
    T* ptr_;

public:
    autoPtr() : ptr_(nullptr) {}
    explicit autoPtr(T* p) : ptr_(p) {}

    ~autoPtr()
    {
        delete ptr_;
        std::cout << "    [autoPtr] Deleted object\n";
    }

    // Move constructor (transfer ownership)
    autoPtr(autoPtr<T>&& other) : ptr_(other.ptr_)
    {
        other.ptr_ = nullptr;
    }

    // Access
    T& operator*() { return *ptr_; }
    T* operator->() { return ptr_; }
    T* get() { return ptr_; }

    // Release ownership
    T* release()
    {
        T* tmp = ptr_;
        ptr_ = nullptr;
        return tmp;
    }
};

// -[3] Reference-counted pointer
template<typename T>
class tmpPtr
{
private:
    T* ptr_;
    bool isRef_;  // True if we're just referencing

public:
    tmpPtr() : ptr_(nullptr), isRef_(false) {}

    explicit tmpPtr(T* p) : ptr_(p), isRef_(false)
    {
        if (ptr_) ptr_->operator++();
    }

    tmpPtr(const tmpPtr<T>& other) : ptr_(other.ptr_), isRef_(true)
    {
        if (ptr_) ptr_->operator++();
    }

    ~tmpPtr()
    {
        if (ptr_)
        {
            ptr_->operator--();
            if (ptr_->okToDelete())
            {
                delete ptr_;
                std::cout << "    [tmpPtr] Deleted object (ref count = 0)\n";
            }
        }
    }

    T& operator*() { return *ptr_; }
    T* operator->() { return ptr_; }

    T* ptr()
    {
        if (!isRef_)
        {
            T* tmp = ptr_;
            ptr_ = nullptr;
            return tmp;
        }
        return ptr_;
    }
};

#endif
```

### Test Program

Create file `test_smart_ptr.C`:

```cpp
#include "smart_pointer.H"
#include <iostream>

class MyData : public refCount
{
    std::string name_;
public:
    MyData(const std::string& name) : name_(name)
    {
        std::cout << "  [MyData] Created: " << name_ << "\n";
    }

    ~MyData()
    {
        std::cout << "  [MyData] Destroyed: " << name_ << "\n";
    }

    void hello() const
    {
        std::cout << "  Hello from " << name_ << "\n";
    }
};

int main()
{
    std::cout << "=== Testing Smart Pointers ===\n\n";

    // -[1] Test autoPtr
    std::cout << "Test 1: autoPtr (exclusive ownership)\n";
    {
        autoPtr<MyData> p1(new MyData("data1"));
        p1->hello();

        autoPtr<MyData> p2(std::move(p1));  // Transfer ownership
        // p1 no longer owns the object
        std::cout << "  After move:\n";
        if (p1.get() == nullptr)
            std::cout << "    p1 is null\n";
        p2->hello();
    }
    std::cout << "  (p2 destroyed here)\n\n";

    // -[2] Test tmpPtr (reference counting)
    std::cout << "Test 2: tmpPtr (reference counting)\n";
    {
        tmpPtr<MyData> r1(new MyData("shared_data"));
        std::cout << "  Ref count after creation: " << r1->count() << "\n";

        {
            tmpPtr<MyData> r2 = r1;  // Share ownership
            std::cout << "  Ref count after copy: " << r1->count() << "\n";
            r2->hello();
        }
        std::cout << "  Ref count after r2 destroyed: " << r1->count() << "\n";

        r1->hello();
    }
    std::cout << "  (object deleted when last reference destroyed)\n\n";

    // -[3] Demonstrate automatic cleanup
    std::cout << "Test 3: Automatic cleanup\n";
    std::cout << "  ✅ No manual delete needed\n";
    std::cout << "  ✅ No memory leaks\n";
    std::cout << "  ✅ Exception-safe\n";

    return 0;
}
```

## Part 4: Design Trade-offs

### Smart Pointer Comparison

| Type | Ownership | Copyable | Use Case |
|------|-----------|----------|----------|
| `autoPtr<T>` | Exclusive | No (move only) | Factory functions, returns |
| `tmp<T>` | Shared | Yes | Expression templates, temporaries |
| `std::unique_ptr` | Exclusive | No (move only) | Modern C++ equivalent to autoPtr |
| `std::shared_ptr` | Shared | Yes | Modern C++ equivalent to tmp |

### When OpenFOAM Uses Each

**`autoPtr`:**
- Factory methods returning new objects
- Transferring ownership between classes

**`tmp<>`:**
- Temporary fields in expressions
- Return values from field operations
- Reference-counted fields shared between multiple expressions

### Performance Considerations

**Reference counting overhead:**
- Increment/decrement is **atomic** (thread-safe)
- Cost: ~5-10 ns per operation
- For CFD: negligible compared to field operations

**Alternative: Move semantics**
- Zero overhead
- Requires C++11 (OpenFOAM uses custom `tmp` for C++98/03 compatibility)

## Summary

**⭐ Key Takeaways:**

1. **Smart pointers** manage memory automatically
2. **`autoPtr`** = exclusive ownership (move-only)
3. **`tmp<>`** = reference-counted shared ownership
4. **`refCount`** enables sharing via reference counting
5. **OpenFOAM uses `tmp<>`** for temporary fields in expressions
