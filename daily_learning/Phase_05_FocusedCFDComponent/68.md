## Part 1 — Diffusion Operator — Gradient of Flux (ตัวดำเนินการการแพร่ — ขั้นตอนการไล่ระดับของส่วนกลับ)

The laplacian operator represents the divergence of a gradient, making it essential for diffusion processes.

### Mathematical Foundation (พื้นฐานทางคณิตศาสตร์)

For a scalar field $\phi$, the laplacian is defined as:

$$
\nabla^2 \phi = \nabla \cdot (\nabla \phi)
$$

This represents the net flux per unit volume, which is the fundamental physics behind:
- Heat conduction: $\nabla \cdot (k \nabla T)$
- Momentum diffusion: $\nabla \cdot (\mu \nabla \mathbf{U})$
- Species diffusion: $\nabla \cdot (D \nabla Y)$

### Finite Volume Discretization (การแบ่งช่วงแบบแยกช่วงเชิงพื้นที่)

Using the divergence theorem on the gradient flux:

$$
\int_V \nabla \cdot (\Gamma \nabla \phi) \, dV = \oint_{\partial V} (\Gamma \nabla \phi) \cdot \mathbf{n} \, dS
$$

For cell $P$, this becomes:

$$
(\nabla \cdot (\Gamma \nabla \phi))_P = \sum_f (\Gamma \nabla \phi)_f \cdot \mathbf{n}_f \, A_f
$$

### Face Gradient Calculation (การคำนวณอนุพันธ์ในแต่ละหน้าจัว)

The key challenge is calculating $(\nabla \phi)_f$ at faces. Common approaches:

1. **Orthogonal correction:**
   $$(\nabla \phi)_f = \frac{\phi_N - \phi_P}{|\mathbf{r}_{PN}|}$$

2. **Non-orthogonal correction:**
   $$(\nabla \phi)_f = (\nabla \phi)_f^{orth} + (\nabla \phi)_f^{non-orth}$$

### Diffusion Flux Formulation (สูตรการคำนวณส่วนกลับแบบแพร่)

The general diffusion flux at face $f$:

$$
F_f = \Gamma_f (\nabla \phi)_f \cdot \mathbf{n}_f \, A_f
$$

Where:
- $\Gamma_f$ = diffusion coefficient at face
- $(\nabla \phi)_f$ = gradient of $\phi$ at face
- $\mathbf{n}_f$ = face normal vector
- $A_f$ = face area

### Matrix Assembly (การรวมเข้ากับเมทริกซ์)

The laplacian operator contributes to the matrix coefficients:

$$
a_P \phi_P - \sum_N a_N \phi_N = b
$$

Where:
- $a_P = \sum_N a_N + a_P^0 + \Delta t \sum_f F_f$
- $a_N = -F_{PN}$ (negative of face flux between P and N)
- $b$ = source term

### Implementation Strategy (กลยุทธ์การนำไปใช้)

Our implementation follows the systematic approach:

1. **Gradient calculation** - Compute $(\nabla \phi)_f$ using chosen scheme
2. **Flux assembly** - Calculate diffusion flux at each face
3. **Matrix construction** - Add contributions to coefficient matrix
4. **Boundary treatment** - Apply boundary conditions correctly

This ensures accuracy for both orthogonal and non-orthogonal meshes.

```cpp
// Main implementation
template<class FieldType, class GammaFieldType>
tmp<GeometricField<typename FieldType::Type, fvPatchField, volMesh>>
fvmLaplacian<FieldType, GammaFieldType>::laplacian() const
{
    const fvMesh& mesh = vf_.mesh();

    // Create result field
    tmp<GeometricField<typename FieldType::Type, fvPatchField, volMesh>> laplacianField
    (
        GeometricField<typename FieldType::Type, fvPatchField, volMesh>::New
        (
            "laplacian(" + vf_.name() + ')',
            mesh,
            dimensioned<typename FieldType::Type>
            (
                vf_.dimensions()/(sqr(dimLength)),
                typename FieldType::Type::zero
            ),
            fvPatchField<typename FieldType::Type>::boundaryTypes()
        )
    );

    GeometricField<typename FieldType::Type, fvPatchField, volMesh>& result =
        laplacianField();

    // Calculate gradients at cell centers
    tmp<GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>> tgradVf =
        fvc::grad(vf_);
    const GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>& gradVf =
        tgradVf();

    // Interpolate gradients to faces
    tmp<GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>> tgradVf_f =
        scheme_.interpolate(gradVf);
    const GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>& gradVf_f =
        tgradVf_f();

    // Get face areas and normals
    const vectorField& Sf = mesh.Sf();
    const scalarField& magSf = mesh.magSf();

    // Calculate face diffusion fluxes
    surfaceScalarField diffusionFlux = gamma_ * (gradVf_f & Sf);

    // Volume integral of divergence
    result = mesh.V0() * (diffusionFlux & mesh.Sf());

    // Apply boundary conditions
    forAll(result.boundaryField(), patchi)
    {
        result.boundaryField()[patchi] =
            gamma_.boundaryField()[patchi] *
            vf_.boundaryField()[patchi].snGrad() *
            magSf.boundaryField()[patchi];
    }

    return laplacianField;
}
```

### Key Implementation Details (รายละเอียดการนำไปใช้งานสำคัญ)

1. **Gradient Calculation:** Uses `fvc::grad()` to compute cell-centered gradients
2. **Surface Interpolation:** Applies scheme to interpolate gradients to faces
3. **Flux Assembly:** Combines diffusion coefficient with gradient normal component
4. **Volume Integration:** Applies divergence theorem to get volume-averaged laplacian

### Memory Management (การจัดการหน่วยความจำ)

The implementation efficiently manages memory by:
- Using `tmp<>` for lazy evaluation of intermediate fields
- Reusing surface fields where possible
- Minimizing temporary field creation

## Part 3 — Orthogonal Mesh Correction (การแก้ไขความไม่ตรงกันเชิงมุมสำหรับกริดแบบตรงมุม)

For orthogonal meshes, the gradient calculation simplifies significantly.

### Orthogonal Gradient Calculation (การคำนวณอนุพันธ์แบบตรงมุม)

On orthogonal meshes, the gradient at face $f$ between cells $P$ and $N$:

$$
(\nabla \phi)_f = \frac{\phi_N - \phi_P}{|\mathbf{r}_{PN}|}
$$

Where $|\mathbf{r}_{PN}|$ is the distance between cell centers.

```cpp
// File: daily_learning/Phase_05_FocusedCFDComponent/laplacian/orthogonalLaplacian.C
// Lines: 1-100

#include "orthogonalLaplacian.H"
#include "fvMatrix.H"
#include "surfaceFields.H"

namespace Foam
{

template<class FieldType>
tmp<GeometricField<typename FieldType::Type, fvPatchField, volMesh>>
orthogonalLaplacian<FieldType>::laplacianOrthogonal() const
{
    const fvMesh& mesh = vf_.mesh();

    // Create result field
    tmp<GeometricField<typename FieldType::Type, fvPatchField, volMesh>> result
    (
        GeometricField<typename FieldType::Type, fvPatchField, volMesh>::New
        (
            "laplacianOrthogonal(" + vf_.name() + ')',
            mesh,
            dimensioned<typename FieldType::Type>
            (
                vf_.dimensions()/(sqr(dimLength)),
                typename FieldType::Type::zero
            ),
            fvPatchField<typename FieldType::Type>::boundaryTypes()
        )
    );

    GeometricField<typename FieldType::Type, fvPatchField, volMesh>& laplacianField =
        result();

    // Get cell centers and face distances
    const pointField& C = mesh.C();
    const vectorField& Cf = mesh.Cf();
    const labelListList& cellFaces = mesh.cells();
    const labelListList& faceNeighbour = mesh.faceNeighbour();

    // Create diffusion coefficient field
    tmp<GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>> tgamma_f =
        scheme_.interpolate(typename FieldType::Type::one());
    const GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>& gamma_f =
        tgamma_f();

    // Optimized loop for internal faces
    #pragma omp parallel for
    forAll(mesh.faces(), facei)
    {
        if (facei < mesh.nInternalFaces())
        {
            label own = mesh.faceOwner()[facei];
            label nei = faceNeighbour[facei];

            // Distance between cell centers
            vector delta = C[nei] - C[own];
            scalar magDelta = mag(delta);

            // Diffusion coefficient at face
            typename FieldType::Type gammaFace = gamma_f[facei];

            // Face area
            scalar magSf = mesh.magSf()[facei];

            // Coefficient magnitude
            typename FieldType::Type coeff = gammaFace * magSf / magDelta;

            // Add to matrix coefficients (simplified)
            laplacianField[own] += coeff;
            laplacianField[nei] += coeff;
        }
    }

    // Boundary conditions
    forAll(laplacianField.boundaryField(), patchi)
    {
        const fvPatch& patch = mesh.boundary()[patchi];
        const fvsPatchField<typename FieldType::Type>& gammaPatch =
            gamma_f.boundaryField()[patchi];

        forAll(patch, facei)
        {
            label facei = patch.start()+facei;
            label own = mesh.faceOwner()[facei];

            // Distance to boundary
            scalar magDelta =
                mag(patch.Cf()[facei] - mesh.C()[own]);

            // Coefficient
            typename FieldType::Type coeff =
                gammaPatch[facei] * mesh.magSf()[facei] / magDelta;

            laplacianField[own] += coeff;
        }
    }

    return result;
}

} // namespace Foam
```

### Performance Optimization (การเพิ่มประสิทธิภาพ)

The orthogonal implementation is significantly faster because:
1. **No gradient calculation needed** - direct face interpolation
2. **Vectorizable operations** - simple arithmetic on distances
3. **Reduced memory access** - no intermediate gradient fields

## Part 4 — Non-Orthogonal Correction (Deferred Correction) (การแก้ไขความไม่ตรงกันเชิงมุม (แบบล่าช้า))

For non-orthogonal meshes, we need to account for the misalignment between face normal and line connecting cell centers.

### Non-Orthogonality Measure (มาตรการของความไม่ตรงกันเชิงมุม)

The non-orthogonality angle $\theta$:

$$
\cos\theta = \frac{(\mathbf{r}_{PN} \cdot \mathbf{n}_f)}{|\mathbf{r}_{PN}| \cdot |\mathbf{n}_f|}
$$

Where:
- $\mathbf{r}_{PN}$ = vector from P to N
- $\mathbf{n}_f$ = face normal vector

### Deferred Correction Approach (แนวทางการแก้ไขแบบล่าช้า)

The deferred correction method splits the gradient calculation:

$$
(\nabla \phi)_f = (\nabla \phi)_f^{orthogonal} + (\nabla \phi)_f^{non-orthogonal}
$$

Where:
- **Orthogonal part:** $(\nabla \phi)_f^{orth} = \frac{\phi_N - \phi_P}{|\mathbf{r}_{PN}| \cos\theta}$
- **Non-orthogonal part:** Calculated explicitly in post-processing

```cpp
// File: daily_learning/Phase_05_FocusedCFDComponent/laplacian/nonOrthogonalLaplacian.C
// Lines: 1-180

#include "nonOrthogonalLaplacian.H"
#include "fvMatrices.H"
#include "surfaceFields.H"
#include "fvcGrad.H"
#include "fvcInterpolate.H"
#include "fvcSurfaceIntegrate.H"

namespace Foam
{

template<class FieldType>
tmp<fvMatrix<typename FieldType::Type>>
nonOrthogonalLaplacian<FieldType>::laplacianWithCorrection()
{
    const fvMesh& mesh = vf_.mesh();

    // Create correction matrix
    tmp<fvMatrix<typename FieldType::Type>> laplacianMatrix
    (
        new fvMatrix<typename FieldType::Type>
        (
            vf_,
            vf_.dimensions()/(sqr(dimLength)*dimTime)
        )
    );

    fvMatrix<typename FieldType::Type>& matrix = laplacianMatrix();

    // Calculate orthogonal part (main diagonal)
    #pragma omp parallel for
    forAll(matrix.diag(), celli)
    {
        scalar sumCoeffs = 0.0;
        const labelList& ownFaces = mesh.cells()[celli];

        forAll(ownFaces, facei)
        {
            label facei = ownFaces[facei];
            label own = mesh.faceOwner()[facei];
            label nei = mesh.faceNeighbour()[facei];

            if (nei != -1) // Internal face
            {
                // Cell center distance
                vector delta = mesh.C()[nei] - mesh.C()[own];
                scalar magDelta = mag(delta);

                // Face area and normal
                vector Sf = mesh.Sf()[facei];
                scalar magSf = mesh.magSf()[facei];

                // Orthogonal correction factor
                scalar cosAngle = (delta & Sf) / (magDelta * magSf);

                // Diffusion coefficient
                scalar gamma = 1.0; // Simplified

                // Coefficient
                scalar coeff = gamma * magSf / (magDelta * cosAngle);

                // Add to matrix
                matrix[own][nei] = -coeff;
                sumCoeffs += coeff;
            }
        }

        matrix.diag()[celli] += sumCoeffs;
    }

    // Calculate non-orthogonal correction explicitly
    updateCorrections();

    // Add non-orthogonal correction to source
    forAll(matrix.source(), celli)
    {
        matrix.source()[celli] -= nonOrthCorr_[celli];
    }

    // Apply boundary conditions
    forAll(matrix.internalField(), celli)
    {
        forAllRow(matrix.upper(), celli, j)
        {
            label jCell = matrix.mupper()[j];
            matrix.lower()[celli][j] = matrix.upper()[celli][j];
        }
    }

    matrix.correctBoundaryConditions();

    return laplacianMatrix;
}

template<class FieldType>
void nonOrthogonalLaplacian<FieldType>::updateCorrections()
{
    const fvMesh& mesh = vf_.mesh();

    // Calculate gradients
    tmp<GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>> tgradVf =
        fvc::grad(vf_);
    const GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>& gradVf =
        tgradVf();

    // Interpolate gradients to faces
    tmp<GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>> tgradVf_f =
        scheme_.interpolate(gradVf);
    const GeometricField<typename FieldType::Type, fvsPatchField, surfaceMesh>& gradVf_f =
        tgradVf_f();

    // Calculate corrections
    #pragma omp parallel for
    forAll(mesh.faces(), facei)
    {
        label own = mesh.faceOwner()[facei];
        label nei = mesh.faceNeighbour()[facei];

        if (nei != -1) // Internal face
        {
            // Cell center vector
            vector delta = mesh.C()[nei] - mesh.C()[own];

            // Face normal
            vector normal = mesh.Sf()[facei] / mesh.magSf()[facei];

            // Orthogonal correction
            orthCorr_[facei] = (gradVf_f[facei] & normal) * mesh.magSf()[facei];

            // Non-orthogonal correction
            vector nonOrthGrad = gradVf_f[facei] - (gradVf_f[facei] & normal) * normal;
            nonOrthCorr_[facei] = (nonOrthGrad & mesh.Sf()[facei]);
        }
    }

    // Sum corrections to cell centers
    #pragma omp parallel for
    forAll(mesh.cells(), celli)
    {
        orthCorr_[celli] = 0.0;
        nonOrthCorr_[celli] = 0.0;
        const labelList& ownFaces = mesh.cells()[celli];

        forAll(ownFaces, facei)
        {
            label facei = ownFaces[facei];
            orthCorr_[celli] += orthCorr_[facei];
            nonOrthCorr_[celli] += nonOrthCorr_[facei];
        }

        // Normalize by cell volume
        orthCorr_[celli] /= mesh.V0()[celli];
        nonOrthCorr_[celli] /= mesh.V0()[celli];
    }
}

} // namespace Foam
```

### Performance Analysis of Deferred Correction (การวิเคราะห์ประสิทธิภาพของการแก้ไขแบบล่าช้า)

| Method | Accuracy | Speed | Memory |
|--------|----------|-------|--------|
| Orthogonal only | Low for non-orthogonal meshes | Fastest | Low |
| Non-orthogonal explicit | High | Slow | High |
| Deferred correction | High | Medium | Medium |

The deferred correction approach provides the best balance for production CFD codes.

```cpp
// Matrix assembly
void diffusionSolver::assembleMatrix(fvScalarMatrix& matrix)
{
    Info<< "Assembling matrix..." << endl;

    // Time derivative term
    matrix = fvm::ddt(T_);

    // Diffusion term
    matrix += fvm::laplacian(gamma_, T_);

    // Source terms (if any)
    fvOptions& fvOptions = mesh_.lookupObject<fvOptions>("fvOptions");
    fvOptions.constrain(matrix);

    // Normalize
    matrix.relax();

    Info<< "Matrix assembled: "
        << matrix.diag().size() << " x "
        << matrix.upper().size() << endl;
}

// Apply boundary conditions
void diffusionSolver::applyBoundaryConditions(fvScalarMatrix& matrix)
{
    Info<< "Applying boundary conditions..." << endl;

    // Fixed temperature at walls
    forAll(T_.boundaryField(), patchi)
    {
        if (mesh_.boundary()[patchi].type() == "fixedTemperature")
        {
            matrix.setBoundaryField
            (
                patchi,
                new fixedValueFvPatchScalarField
                (
                    T_.boundaryField()[patchi]
                )
            );
        }
        else if (mesh_.boundary()[patchi].type() == "insulated")
        {
            matrix.setBoundaryField
            (
                patchi,
                new zeroGradientFvPatchScalarField
                (
                    T_.boundaryField()[patchi]
                )
            );
        }
    }
}

}
```
