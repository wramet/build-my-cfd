## Part 1 — Normalized Variable Diagram (NVD) (แผนภูมิตัวแปรที่ปรับปริมาณแล้ว)

### Concept of Normalized Variables

The Normalized Variable Diagram (NVD) provides a systematic way to analyze and design high-resolution schemes. It normalizes variables relative to the local extrema:

$$
\phi^* = \frac{\phi - \phi_{\text{min}}}{\phi_{\text{max}} - \phi_{\text{min}}}
$$

where:
- $\phi_{\text{min}} = \min(\phi_U, \phi_P, \phi_N)$
- $\phi_{\text{max}} = \max(\phi_U, \phi_P, \phi_N)$

This transformation maps values to [0,1], making it easier to analyze local behavior.

### NVD Construction

Consider the stencil:
```
U → P → N → F
```

The NVD plots the possible values of $\phi_f$ based on $\phi_U, \phi_P, \phi_N$:

```
    φ_max
     1.0 +---------------------*
          /|                   |
         / |                   |
        /  |                   |
       /   |                   |
      /    |                   |
     /     |                   |
    /      |                   |
   /       |                   |
 0.0 +-----+-----+-----+-----+--→ φ*
      U     P     N     F
```

### Linear Reconstruction

The most basic NVD scheme is linear interpolation:

$$
\phi_f^* = (1 - \alpha) \phi_P^* + \alpha \phi_N^*
$$

where $\alpha$ is the interpolation weight.

### Boundedness Constraints

For reconstruction to be bounded:
$$
\phi_{\text{min}} \leq \phi_f \leq \phi_{\text{max}}
$$

In normalized coordinates:
$$
0 \leq \phi_f^* \leq 1
$$

```mermaid
flowchart LR
    subgraph NVD_Diagram
        A[φ_min] --> B[φ_U]
        B --> C[φ_P]
        C --> D[φ_N]
        D --> E[φ_max]

        H[φ_f < 0 or φ_f > 1] --> I[❌ Unbounded]

```



This creates the "boundedness triangle" in NVD.

### Higher-Order Schemes on NVD

**QUICK Scheme:**
$$
\phi_f = \frac{6\phi_P + 3\phi_N - \phi_U}{8}
$$

In normalized coordinates:
$$
\phi_f^* = \frac{6\phi_P^* + 3\phi_N^* - \phi_U^*}{8}
$$

**CDS Scheme:**
$$
\phi_f = \frac{\phi_P + \phi_N}{2}
$$

In normalized coordinates:
$$
\phi_f^* = \frac{\phi_P^* + \phi_N^*}{2}
$$

### NVD Analysis Framework

The NVD allows systematic analysis of any reconstruction scheme:

1. **Plot the scheme** on the normalized coordinates
2. **Check boundedness** - does it stay within [0,1]?
3. **Check monotonicity** - does it preserve existing extrema?
4. **Check accuracy** - what is the order of accuracy?

### Example: Different Schemes on NVD

```
    1.0 |        QUICK
        |       /\
        |      /  \
        |     /    \
        |    /      \
        |   /        \
        |  /          \
        | /            \
    0.5 +---------------+
        |      CDS
        |     / \
        |    /   \
        |   /     \
        |  /       \
        | /         \
    0.0 +-------------+
        U    P    N
```

QUICK maintains higher accuracy but can violate boundedness near extrema, while CDS is second-order but oscillates.

```cpp
#ifndef NVDAnalyzer_H
#define NVDAnalyzer_H

#include "volFields.H"

class NVDAnalyzer
{
    const fvMesh& mesh_;

public:
    NVDAnalyzer(const fvMesh& mesh)
    :
        mesh_(mesh)
    {}

    // Analyze a cell face for boundedness
    bool isBounded
    (
        scalar phiU,
        scalar phiP,
        scalar phiN,
        scalar phiF
    ) const
    {
        scalar phiMin = min(min(phiU, phiP), phiN);
        scalar phiMax = max(max(phiU, phiP), phiN);

        return (phiF >= phiMin - SMALL) && (phiF <= phiMax + SMALL);
    }

    // Calculate normalized coordinates
    void normalizedCoordinates
    (
        scalar phiU,
        scalar phiP,
        scalar phiN,
        scalar& phiMin,
        scalar& phiMax,
        scalar& phiPStar,
        scalar& phiNStar,
        scalar& phiFStar
    ) const
    {
        phiMin = min(min(phiU, phiP), phiN);
        phiMax = max(max(phiU, phiP), phiN);

        scalar range = phiMax - phiMin;
        if (range < SMALL)
        {
            // Constant value case
            phiPStar = 0.5;
            phiNStar = 0.5;
            phiFStar = 0.5;
        }
        else
        {
            phiPStar = (phiP - phiMin) / range;
            phiNStar = (phiN - phiMin) / range;
            phiFStar = (phiF - phiMin) / range;
        }
    }

    // Evaluate QUICK scheme
    scalar QUICK(scalar phiU, scalar phiP, scalar phiN) const
    {
        return (6*phiP + 3*phiN - phiU) / 8;
    }

    // Evaluate CDS scheme
    scalar CDS(scalar phiP, scalar phiN) const
    {
        return 0.5*(phiP + phiN);
    }

    // Evaluate upwind scheme
    scalar upwind(scalar phiU, scalar phiP, scalar phiN, scalar F) const
    {
        return (F > 0) ? phiU : phiN;
    }
};

#endif
```

## Part 2 — Gamma and QUICK Schemes (รูปแบบแกมมาและ QUICK)

### The Gamma Scheme

The Gamma scheme introduces a free parameter γ to interpolate between schemes:

$$
\phi_f = \phi_P + \gamma \frac{\phi_N - \phi_P}{2} + (1 - \gamma) \frac{\phi_P - \phi_U}{2}
$$

Where:
- γ = 1: Central Differencing Scheme (CDS)
- γ = 0: Upwind Scheme
- γ = 0.5: Linear interpolation

**Boundedness Condition:**
$$
\text{max}(0, 2r/(1+r)) \leq \gamma \leq \text{min}(1, 2r)
$$

where $r = (\phi_P - \phi_U)/(\phi_N - \phi_P)$.

### QUICK Scheme Details

The Quadratic Upstream Interpolation for Convective Kinematics (QUICK) scheme:

$$
\phi_f = \frac{3\phi_P + 3\phi_N - \phi_U}{4} \quad \text{(backward QUICK)}
$$

$$
\phi_f = \frac{3\phi_P + 3\phi_U - \phi_N}{4} \quad \text{(forward QUICK)}
$$

**Accuracy Analysis:**
- Truncation error: $O(\Delta x^3)$
- Third-order accurate for smooth solutions
- Better resolution of boundary layers

**Boundedness Issues:**
```cpp
// Quick boundedness check
bool isQUICKBounded(scalar phiU, scalar phiP, scalar phiN)
{
    scalar phiF = (6*phiP + 3*phiN - phiU)/8;
    scalar phiMin = min(min(phiU, phiP), phiN);
    scalar phiMax = max(max(phiU, phiP), phiN);
    return (phiF >= phiMin - SMALL) && (phiF <= phiMax + SMALL);
}
```

### Hybrid Gamma-QUICK Scheme

Combining the strengths of both approaches:

$$
\phi_f = \phi_P + \text{QUICKCorrection}(\phi_U, \phi_P, \phi_N) \cdot \Gamma(\nabla \phi)
$$

Where $\Gamma$ is a limiter function.

### Implementation of High-Resolution Schemes

**File: `src/schemes/highResolution/highResolutionSchemes.H`**

```cpp
#ifndef highResolutionSchemes_H
#define highResolutionSchemes_H

#include "volFields.H"
#include "surfaceFields.H"

// Base class for high-resolution schemes
class HighResolutionScheme
{
    const fvMesh& mesh_;

protected:
    scalar CFL_;
    scalar gamma_;

public:
    HighResolutionScheme(const fvMesh& mesh, scalar gamma = 0.5)
    :
        mesh_(mesh),
        gamma_(gamma)
    {}

    virtual ~HighResolutionScheme() = default;

    virtual tmp<surfaceScalarField> interpolate
    (
        const volScalarField& phi
    ) = 0;

    virtual word name() const = 0;

    void setCFL(scalar CFL) { CFL_ = CFL; }
};

// Gamma scheme
class GammaScheme : public HighResolutionScheme
{
public:
    GammaScheme(const fvMesh& mesh, scalar gamma = 0.5)
    :
        HighResolutionScheme(mesh, gamma)
    {}

    tmp<surfaceScalarField> interpolate
    (
        const volScalarField& phi
    ) override
    {
        tmp<surfaceScalarField> tphiInterp
        (
            new surfaceScalarField
            (
                IOobject::groupName("phi", phi.group()),
                phi.mesh(),
                dimensionedScalar("phi", phi.dimensions(), 0)
            )
        );

        surfaceScalarField& phiInterp = tphiInterp.ref();

        forAll(phiInterp, faceI)
        {
            label own = mesh_.faceOwner()[faceI];
            label nei = mesh_.faceNeighbour()[faceI];

            scalar phiU = phi[own];
            scalar phiP = phi[nei];

            // Smoothness parameter
            scalar r = (phiU - phi[nei])/(phiP - phiU + SMALL);

            // Boundedness condition
            scalar gammaLower = max(0, 2*r/(1 + r));
            scalar gammaUpper = min(1, 2*r);

            if (gamma_ < gammaLower || gamma_ > gammaUpper)
            {
                // Use limiter to ensure boundedness
                scalar psi = limiter(r);
                gamma_ = gammaLower + psi*(gammaUpper - gammaLower);
            }

            // Gamma interpolation
            phiInterp[faceI] = phiU + gamma_*(phiP - phiU);
        }

        return tphiInterp;
    }

    word name() const override { return "gamma"; }

private:
    scalar limiter(scalar r)
    {
        if (r <= 0) return 0;
        if (r >= 1) return 1;
        return r;  // Minmod-like
    }
};

// QUICK scheme
class QUICKScheme : public HighResolutionScheme
{
public:
    QUICKScheme(const fvMesh& mesh)
    :
        HighResolutionScheme(mesh, 0)
    {}

    tmp<surfaceScalarField> interpolate
    (
        const volScalarField& phi
    ) override
    {
        tmp<surfaceScalarField> tphiInterp
        (
            new surfaceScalarField
            (
                IOobject::groupName("phi", phi.group()),
                phi.mesh(),
                dimensionedScalar("phi", phi.dimensions(), 0)
            )
        );

        surfaceScalarField& phiInterp = tphiInterp.ref();

        forAll(phiInterp, faceI)
        {
            label own = mesh_.faceOwner()[faceI];
            label nei = mesh_.faceNeighbour()[faceI];

            scalar F = mesh_.magSf()[faceI] & mesh_.U()[nei];

            if (mag(F) < SMALL)
            {
                phiInterp[faceI] = 0.5*(phi[own] + phi[nei]);
            }
            else
            {
                label upwind = (F > 0) ? own : nei;
                label downwind = (F > 0) ? nei : own;

                scalar phiUpwind = phi[upwind];
                scalar phiDownwind = phi[downwind];

                // Find the far cell
                label farCell = (F > 0) ? nei : own;

                scalar phiF = (6*phiDownwind + 3*phiUpwind - phi[farCell])/8;

                // Check boundedness
                scalar phiMin = min(min(phiUpwind, phi[farCell]), phiDownwind);
                scalar phiMax = max(max(phiUpwind, phi[farCell]), phiDownwind);

                if (phiF < phiMin || phiF > phiMax)
                {
                    // Fall back to limiter
                    scalar r = (phiUpwind - phi[farCell])/(phiDownwind - phiUpwind + SMALL);
                    scalar psi = vanLeerLimiter(r);
                    phiF = phiUpwind + psi*(0.5*(phiUpwind + phiDownwind) - phiUpwind);
                }

                phiInterp[faceI] = phiF;
            }
        }

        return tphiInterp;
    }

    word name() const override { return "QUICK"; }
};

// Hybrid scheme combining multiple approaches
class HybridScheme : public HighResolutionScheme
{
    autoPtr<HighResolutionScheme> gammaScheme_;
    autoPtr<HighResolutionScheme> quickScheme_;
    scalar threshold_;

public:
    HybridScheme(const fvMesh& mesh, scalar gamma = 0.5, scalar threshold = 0.1)
    :
        HighResolutionScheme(mesh, gamma),
        threshold_(threshold)
    {
        gammaScheme_.set(new GammaScheme(mesh, gamma));
        quickScheme_.set(new QUICKScheme(mesh));
    }

    tmp<surfaceScalarField> interpolate
    (
        const volScalarField& phi
    ) override
    {
        tmp<surfaceScalarField> tgamma = gammaScheme_->interpolate(phi);
        tmp<surfaceScalarField> tquick = quickScheme_->interpolate(phi);

        surfaceScalarField& gammaResult = tgamma.ref();
        const surfaceScalarField& quickResult = tquick.ref();

        // Detect gradients and choose scheme
        forAll(gammaResult, faceI)
        {
            label own = mesh_.faceOwner()[faceI];
            label nei = mesh_.faceNeighbour()[faceI];

            scalar phiU = phi[own];
            scalar phiP = phi[nei];
            scalar gradient = abs(phiP - phiU);

            if (gradient > threshold_)
            {
                // Strong gradient, use bounded QUICK
                gammaResult[faceI] = quickResult[faceI];
            }
            else
            {
                // Smooth region, use higher-order gamma
                gammaResult[faceI] = gammaResult[faceI];
            }
        }

        return tgamma;
    }

    word name() const override { return "hybrid"; }
};

#endif
```

### Advanced Limiter Functions

**File: `src/schemes/limiters/advancedLimiters.H`**

```cpp
#ifndef advancedLimiters_H
#define advancedLimiters_H

// Sweby limiter - optimized for specific flow characteristics
class SwebyLimiter
{
public:
    scalar operator()(scalar r) const
    {
        if (r <= 0) return 0;
        if (r >= 1) return 1;
        return min(2*r, 1);  // Sweby's original
    }
};

// MUSCL limiter - Monotonic Upstream Scheme for Conservation Laws
class MUSCLLimiter
{
public:
    scalar operator()(scalar r) const
    {
        if (r <= 0) return 0;
        if (r >= 1) return 1;
        return (3*r + r*r)/(3 + 3*r + r*r);  // HCUS-like
    }
};

// Koren limiter - good for compressible flows
class KorenLimiter
{
public:
    scalar operator()(scalar r) const
    {
        if (r <= 0) return 0;
        if (r >= 1) return 1;
        return max(0, min(2*r, 0.5*(1 + r), 2*r/(1 + r)));
    }
};
```

## Part 3 — Implementation Comparison (การเปรียบเทียบการนำไปใช้งาน)

### Comparative Study Setup

Let's implement a comprehensive comparison of different schemes:

**File: `src/schemes/comparison/schemeComparison.H`**

```cpp
#ifndef schemeComparison_H
#define schemeComparison_H

#include "fvCFD.H"
#include "limitedSchemes.H"
#include "highResolutionSchemes.H"
#include "NVD/NVDAnalyzer.H"

class SchemeComparison
{
    const fvMesh& mesh_;
    autoPtr<LimiterFunction> minmod_;
    autoPtr<LimiterFunction> vanLeer_;
    autoPtr<LimiterFunction> SUPERBEE_;
    autoPtr<GammaScheme> gamma05_;
    autoPtr<GammaScheme> gamma10_;
    autoPtr<QUICKScheme> quick_;
    autoPtr<HybridScheme> hybrid_;

public:
    SchemeComparison(const fvMesh& mesh)
    :
        mesh_(mesh)
    {
        // Initialize limiters
        minmod_.set(new MinmodLimiter);
        vanLeer_.set(new vanLeerLimiter);
        SUPERBEE_.set(new SUPERBEElimiter);

        // Initialize schemes
        gamma05_.set(new GammaScheme(mesh, 0.5));
        gamma10_.set(new GammaScheme(mesh, 1.0));
        quick_.set(new QUICKScheme(mesh));
        hybrid_.set(new HybridScheme(mesh, 0.7, 0.1));
    }

    // Run comparison test
    void runComparison(const volScalarField& initialPhi, const volVectorField& U)
    {
        List<word> schemeNames = {
            "minmod", "vanLeer", "SUPERBEE",
            "gamma0.5", "gamma1.0", "QUICK", "hybrid"
        };

        List<volScalarField> results(schemeNames.size());
        List<List<scalar>> histories(schemeNames.size());

        // Time integration
        scalar dt = 0.001;
        scalar totalTime = 1.0;
        label nSteps = totalTime/dt;

        volScalarField referencePhi = initialPhi;

        // Initialize all fields
        forAll(schemeNames, i)
        {
            results[i].setReference(initialPhi);
            results[i] = initialPhi;
        }

        // Time stepping
        for (label step = 0; step < nSteps; ++step)
        {
            scalar time = step * dt;

            // Update each scheme
            forAll(schemeNames, schemeI)
            {
                volScalarField& phi = results[schemeI];

                // Choose interpolation scheme based on name
                tmp<surfaceScalarField> phiInterp;
                if (schemeNames[schemeI] == "minmod")
                {
                    limitedSurfaceInterpolation<scalar> limiter(mesh_, "minmod");
                    phiInterp = limiter.interpolate(phi);
                }
                else if (schemeNames[schemeI] == "vanLeer")
                {
                    limitedSurfaceInterpolation<scalar> limiter(mesh_, "vanLeer");
                    phiInterp = limiter.interpolate(phi);
                }
                else if (schemeNames[schemeI] == "SUPERBEE")
                {
                    limitedSurfaceInterpolation<scalar> limiter(mesh_, "SUPERBEE");
                    phiInterp = limiter.interpolate(phi);
                }
                else if (schemeNames[schemeI] == "gamma0.5")
                {
                    phiInterp = gamma05_->interpolate(phi);
                }
                else if (schemeNames[schemeI] == "gamma1.0")
                {
                    phiInterp = gamma10_->interpolate(phi);
                }
                else if (schemeNames[schemeI] == "QUICK")
                {
                    phiInterp = quick_->interpolate(phi);
                }
                else if (schemeNames[schemeI] == "hybrid")
                {
                    phiInterp = hybrid_->interpolate(phi);
                }

                // Update with convective flux
                forAll(phi, cellI)
                {
                    scalar flux = 0.0;
                    label own = cellI;

                    forAll(mesh_.cells()[cellI], faceI)
                    {
                        label f = mesh_.cells()[cellI][faceI];
                        label nei = mesh_.faceNeighbour()[f];

                        if (mesh_.isInternalFace(f))
                        {
                            scalar F = mesh_.magSf()[f] & U[nei];
                            flux += F * phiInterp()[f];
                        }
                    }

                    phi[cellI] -= dt * flux / mesh_.V()[cellI];
                }

                // Store history
                if (step % 100 == 0)
                {
                    histories[schemeI].append(max(phi).value());
                }
            }
        }

        // Write comparison results
        writeComparisonResults(schemeNames, results, histories);
        writeAccuracyMetrics(schemeNames, results, referencePhi);
    }

private:
    void writeComparisonResults
    (
        const List<word>& schemeNames,
        const List<volScalarField>& results,
        const List<List<scalar>>& histories
    )
    {
        // Write final profiles
        OFstream profileFile("scheme_profiles.dat");
        profileFile << "# x";

        forAll(schemeNames, i)
        {
            profileFile << " " << schemeNames[i];
        }
        profileFile << endl;

        forAll(mesh_.C(), cellI)
        {
            profileFile << mesh_.C()[cellI].x();

            forAll(schemeNames, i)
            {
                profileFile << " " << results[i][cellI];
            }
            profileFile << endl;
        }

        // Write time history
        OFstream historyFile("scheme_history.dat");
        historyFile << "# time";
        forAll(schemeNames, i)
        {
            historyFile << " " << schemeNames[i];
        }
        historyFile << endl;

        forAll(histories[0], step)
        {
            historyFile << step * 0.1;

            forAll(schemeNames, i)
            {
                historyFile << " " << histories[i][step];
            }
            historyFile << endl;
        }
    }

    void writeAccuracyMetrics
    (
        const List<word>& schemeNames,
        const List<volScalarField>& results,
        const volScalarField& reference
    )
    {
        OFstream metricsFile("scheme_metrics.dat");
        metricsFile << "# Scheme Max Min L2Error MaxOvershoot MaxUndershoot CPU_time(ms)" << endl;

        forAll(schemeNames, i)
        {
            const volScalarField& phi = results[i];

            // Calculate metrics
            scalar maxVal = max(phi).value();
            scalar minVal = min(phi).value();
            scalar l2Error = calculateL2Error(phi, reference);
            scalar maxOvershoot = calculateMaxOvershoot(phi, reference);
            scalar maxUndershoot = calculateMaxUndershoot(phi, reference);

            metricsFile << schemeNames[i] << " "
                       << maxVal << " "
                       << minVal << " "
                       << l2Error << " "
                       << maxOvershoot << " "
                       << maxUndershoot << " "
                       << 0.0  // Placeholder for CPU time
                       << endl;
        }
    }

    scalar calculateL2Error
    (
        const volScalarField& phi,
        const volScalarField& reference
    )
    {
        scalar error = 0;
        forAll(phi, cellI)
        {
            error += pow(phi[cellI] - reference[cellI], 2);
        }
        return sqrt(error / phi.size());
    }

    scalar calculateMaxOvershoot
    (
        const volScalarField& phi,
        const volScalarField& reference
    )
    {
        scalar overshoot = 0;
        forAll(phi, cellI)
        {
            scalar over = phi[cellI] - reference[cellI];
            if (over > 0)
            {
                overshoot = max(overshoot, over);
            }
        }
        return overshoot;
    }

    scalar calculateMaxUndershoot
    (
        const volScalarField& phi,
        const volScalarField& reference
    )
    {
        scalar undershoot = 0;
        forAll(phi, cellI)
        {
            scalar under = reference[cellI] - phi[cellI];
            if (under > 0)
            {
                undershoot = max(undershoot, under);
            }
        }
        return undershoot;
    }
};

#endif
```

## Part 4 — Accuracy vs. Boundedness Trade-off (การแลกเปลี่ยนระหว่างความแม่นยำและการจำกัดขอบเขต)

### Trade-off Analysis Framework

Let's implement a systematic analysis of the accuracy-boundedness trade-off:

**File: `src/schemes/tradeoff/TradeoffAnalyzer.H`**

```cpp
#ifndef TradeoffAnalyzer_H
#define TradeoffAnalyzer_H

#include "volFields.H"

class TradeoffAnalyzer
{
    const fvMesh& mesh_;

public:
    TradeoffAnalyzer(const fvMesh& mesh)
    :
        mesh_(mesh)
    {}

    // Calculate various accuracy metrics
    struct AccuracyMetrics
    {
        scalar l2Norm;
        scalar lInfNorm;
        scalar conservationError;
        scalar gradientResolution;
    };

    // Calculate boundedness metrics
    struct BoundednessMetrics
    {
        scalar maxOvershoot;
        scalar maxUndershoot;
        scalar violationCount;
        scalar boundednessRatio;
    };

    // Run tradeoff analysis
    List<Dictionary> analyzeTradeoff
    (
        const List<word>& schemeNames,
        const List<volScalarField>& solutions,
        const volScalarField& exact
    )
    {
        List<Dictionary> results(schemeNames.size());

        forAll(schemeNames, i)
        {
            const volScalarField& solution = solutions[i];

            // Calculate metrics
            AccuracyMetrics accuracy = calculateAccuracyMetrics(solution, exact);
            BoundednessMetrics boundedness = calculateBoundednessMetrics(solution, exact);

            // Store results
            Dictionary result;
            result.set("scheme", schemeNames[i]);
            result.set("accuracy", accuracy);
            result.set("boundedness", boundedness);
            result.set("efficiency", calculateEfficiency(accuracy, boundedness));

            results[i] = result;
        }

        return results;
    }

private:
    AccuracyMetrics calculateAccuracyMetrics
    (
        const volScalarField& solution,
        const volScalarField& exact
    )
    {
        AccuracyMetrics metrics;

        // L2 norm error
        scalar error2 = 0;
        forAll(solution, cellI)
        {
            error2 += pow(solution[cellI] - exact[cellI], 2);
        }
        metrics.l2Norm = sqrt(error2 / solution.size());

        // L∞ norm error
        scalar maxError = 0;
        forAll(solution, cellI)
        {
            scalar error = abs(solution[cellI] - exact[cellI]);
            maxError = max(maxError, error);
        }
        metrics.lInfNorm = maxError;

        // Conservation error
        scalar totalSolution = 0;
        scalar totalExact = 0;
        forAll(solution, cellI)
        {
            totalSolution += solution[cellI] * mesh_.V()[cellI];
            totalExact += exact[cellI] * mesh_.V()[cellI];
        }
        metrics.conservationError = abs(totalSolution - totalExact) / max(totalExact, SMALL);

        // Gradient resolution (how well sharp gradients are preserved)
        scalar gradientResolution = 0;
        forAll(mesh_.internalFaces(), faceI)
        {
            label own = mesh_.faceOwner()[faceI];
            label nei = mesh_.faceNeighbour()[faceI];
            scalar gradient = abs(solution[own] - solution[nei]);
            gradientResolution = max(gradientResolution, gradient);
        }
        metrics.gradientResolution = gradientResolution;

        return metrics;
    }

    BoundednessMetrics calculateBoundednessMetrics
    (
        const volScalarField& solution,
        const volScalarField& exact
    )
    {
        BoundednessMetrics metrics;

        // Find exact bounds
        scalar exactMin = min(exact).value();
        scalar exactMax = max(exact).value();

        // Check violations
        scalar overshoot = 0;
        scalar undershoot = 0;
        label violationCount = 0;

        forAll(solution, cellI)
        {
            scalar val = solution[cellI];

            if (val > exactMax)
            {
                overshoot = max(overshoot, val - exactMax);
                violationCount++;
            }
            if (val < exactMin)
            {
                undershoot = max(undershoot, exactMin - val);
                violationCount++;
            }
        }

        metrics.maxOvershoot = overshoot;
        metrics.maxUndershoot = undershoot;
        metrics.violationCount = violationCount;

        // Boundedness ratio (fraction of cells within bounds)
        label boundedCount = solution.size() - violationCount;
        metrics.boundednessRatio = scalar(boundedCount) / solution.size();

        return metrics;
    }

    scalar calculateEfficiency
    (
        const AccuracyMetrics& accuracy,
        const BoundednessMetrics& boundedness
    )
    {
        // Efficiency metric combining accuracy and robustness
        scalar accuracyScore = 1.0 / (1.0 + accuracy.l2Norm);
        scalar boundednessScore = boundedness.boundednessRatio;

        return 0.5 * (accuracyScore + boundednessScore);
    }
};
```
