# การไหลที่ขับเคลื่อนด้วยแรงลอยตัว (Buoyancy-Driven Flows)

## 📖 บทนำ (Introduction) 

การพาความร้อนตามธรรมชาติ (**Natural Convection**) เกิดขึ้นเมื่อความแตกต่างของอุณหภูมิทำให้เกิดความหนาแน่นที่ไม่สม่ำเสมอ จนเกิดแรงลอยตัวที่ขับเคลื่อนการไหลในสนามแรงโน้มถ่วง

---

## 🔄 1. การประมาณแบบบูสสิเนสก์ (Boussinesq Approximation)

ในกรณีที่ความแตกต่างของอุณหภูมิไม่มากนัก ($\Delta T$ น้อย) เราสามารถใช้สมมติฐานเพื่อประหยัดเวลาคำนวณ โดยถือว่าความหนาแน่นคงที่ในทุกเทอม **ยกเว้นในเทอมแรงลอยตัว**

### 1.1 สมการความหนาแน่น
$$\rho = \rho_0 [1 - \beta (T - T_0)]$$
โดยที่ $\beta$ คือ สัมประสิทธิ์การขยายตัวทางความร้อน (Thermal expansion coefficient)

### 1.2 แรงลอยตัวในสมการโมเมนตัม
$$\mathbf{F}_b = \rho_0 \mathbf{g} \beta (T - T_0)$$

---

## 🔬 2. จำนวนไร้มิติที่สำคัญ

พฤติกรรมการไหลตามธรรมชาติถูกควบคุมโดย:

### 2.1 จำนวนกราสชอฟ (Grashof Number, Gr)
อัตราส่วนของแรงลอยตัวต่อแรงหนืด:
$$Gr = \frac{g \beta \Delta T L^3}{\nu^2}$$

### 2.2 จำนวนเรย์ลี (Rayleigh Number, Ra)
ตัวชี้วัดความแข็งแกร่งของการพาความร้อนตามธรรมชาติ:
$$Ra = Gr \cdot Pr = \frac{g \beta \Delta T L^3}{\nu \alpha}$$
- **$Ra < 10^3$**: การนำความร้อนเป็นหลัก
- **$10^3 < Ra < 10^9$**: การพาความร้อนแบบลามินาร์
- **$Ra > 10^9$**: การพาความร้อนแบบปั่นป่วน

---

## 💻 3. การนำไปใช้ใน OpenFOAM

Solver หลักคือ **`buoyantBoussinesqSimpleFoam`** (Steady) หรือ **`buoyantBoussinesqPimpleFoam`** (Transient)

### 3.1 การระบุแรงโน้มถ่วง
กำหนดในไฟล์ `constant/g`:
```cpp
dimensions      [0 1 -2 0 0 0 0];
value           (0 -9.81 0);
```

### 3.2 การกำหนดตัวแปร Beta
ระบุใน `constant/transportProperties`:
```cpp
beta            [0 0 0 -1 0 0 0] 0.003; // ค่าสำหรับอากาศโดยประมาณ
```

## ⚠️ ข้อควรพิจารณา (Stability and Convergence)

1. **Mesh Resolution**: บริเวณชั้นขอบเขตความร้อน (Thermal Boundary Layer) มักจะบางมาก ต้องการ Mesh ที่ละเอียดใกล้ผนังสูง
2. **Under-Relaxation**: สำหรับปัญหาแรงลอยตัวสูง อาจต้องลด relaxation factors สำหรับอุณหภูมิลงเหลือ 0.5 - 0.7 เพื่อป้องกันการแกว่ง
