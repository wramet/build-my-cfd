# การเปรียบเทียบอัลกอริทึมการเชื่อมโยงความดัน-ความเร็ว

## 📊 1. ตารางเปรียบเทียบคุณลักษณะหลัก

| คุณสมบัติ | SIMPLE | PISO | PIMPLE |
|-----------|--------|------|---------|
| **ชนิดของปัญหา** | Steady-state | Transient | Transient (Strong) |
| **ความแม่นยำเชิงเวลา** | ไม่สำคัญ (Pseudo-time) | อันดับ 2 | อันดับ 1-2 |
| **Under-relaxation** | จำเป็น ($\alpha < 1$) | ไม่ใช้ ($\alpha = 1$) | ใช้ใน Outer loops |
| **ขีดจำกัด Courant ($Co$)** | ไม่มี | แนะนำ $Co < 1$ | อนุญาต $Co > 1$ |
| **ต้นทุนการคำนวณ** | ต่ำที่สุด | ปานกลาง | สูงที่สุดต่อก้าวเวลา |
| **ความแข็งแกร่ง (Robustness)** | สูง | ต่ำ (ถ้าก้าวเวลาใหญ่) | สูงมาก |

## 🎯 2. แนวทางการเลือกใช้งาน (Selection Guide)

### 2.1 เมื่อใดควรใช้ SIMPLE?
- ปัญหาที่หัวใจสำคัญอยู่ที่สถานะสุดท้ายที่คงที่ (Steady equilibrium)
- การหาค่าสัมประสิทธิ์แรงต้าน (Drag) หรือแรงยก (Lift) เฉลี่ย
- **Solver**: `simpleFoam`, `buoyantSimpleFoam`

### 2.2 เมื่อใดควรใช้ PISO?
- ปัญหาที่ต้องการจับความถูกต้องของการเปลี่ยนแปลงตามเวลาอย่างละเอียด
- การจำลองคลื่นเสียง (Aeroacoustics) หรือโครงสร้างความปั่นป่วนขนาดเล็ก
- ต้องมั่นใจว่าทรัพยากรคอมพิวเตอร์เพียงพอที่จะรันด้วย $\Delta t$ ขนาดเล็ก ($Co < 1$)
- **Solver**: `icoFoam`, `pisoFoam`

### 2.3 เมื่อใดควรใช้ PIMPLE?
- ปัญหาที่มีความซับซ้อนของ Mesh เช่น Mesh เคลื่อนที่ (Moving Mesh) หรือรูปทรงเปลี่ยนรูป
- ปัญหา Multiphase (เช่น `interFoam`) ที่อินเทอร์เฟซมีความไม่เสถียรสูง
- ต้องการรันการจำลองให้จบเร็วขึ้นโดยใช้ก้าวเวลาขนาดใหญ่
- **Solver**: `pimpleFoam`, `buoyantPimpleFoam`, `interFoam`

## 🛠️ 3. การปรับแต่งพารามิเตอร์เพื่อประสิทธิภาพสูงสุด

### 3.1 สำหรับ SIMPLE
- **Relaxation**: หากลู่เข้ายาก ให้ลด $\alpha_p$ เหลือ 0.2 และ $\alpha_U$ เหลือ 0.5
- **Non-orthogonal**: หาก Mesh คุณภาพต่ำ ให้เพิ่ม `nNonOrthogonalCorrectors` เป็น 1 หรือ 2

### 3.2 สำหรับ PIMPLE
- **Outer Loops**: โดยทั่วไปใช้ `nOuterCorrectors` 2-5 รอบ หากฟิสิกส์ซับซ้อนมากอาจต้องเพิ่มเป็น 10+
- **Residual Control**: ใช้เพื่อหยุด Outer loop เมื่อค่าคำตอบในก้าวเวลานั้นลู่เข้าแล้ว ช่วยประหยัดเวลาคำนวณ:
```cpp
PIMPLE
{
    residualControl
    {
        "(U|p)" { tolerance 1e-5; relTol 0; }
    }
}
```

## 📝 4. บทสรุป (Conclusion)

การเลือกอัลกอริทึมที่ถูกต้องคือการสร้างสมดุลระหว่าง **ความแม่นยำ (Accuracy)**, **เสถียรภาพ (Stability)** และ **ต้นทุนทรัพยากร (Computational Cost)**

- **SIMPLE** คือทางเลือกที่ดีที่สุดสำหรับงานออกแบบเชิงวิศวกรรมที่เน้นสภาวะคงที่
- **PISO** คือมาตรฐานสำหรับงานวิจัยที่ต้องการ Temporal Accuracy สูง
- **PIMPLE** คือเครื่องมืออเนกประสงค์ที่แข็งแกร่งที่สุดสำหรับการใช้งานในโลกแห่งความเป็นจริง
