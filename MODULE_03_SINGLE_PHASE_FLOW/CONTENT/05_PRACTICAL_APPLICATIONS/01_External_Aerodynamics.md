# พลศาสตร์อากาศภายนอก (External Aerodynamics)

## 📖 บทนำ (Introduction)

พลศาสตร์อากาศภายนอกเป็นการศึกษาการไหลรอบวัตถุ ซึ่งมีความสำคัญในการออกแบบยานพาหนะ เครื่องบิน และโครงสร้างพื้นฐาน OpenFOAM มีเครื่องมือที่ทรงพลังสำหรับการวิเคราะห์แรงต้าน แรงยก และลักษณะกระแสลมท้าย (Wake)

---

## 🔍 1. ตัวชี้วัดอากาศพลศาสตร์ที่สำคัญ

### 1.1 สัมประสิทธิ์แรงต้านและแรงยก (Drag and Lift Coefficients)
แรงที่กระทำต่อวัตถุถูกทำให้ไร้มิติเพื่อการเปรียบเทียบ:

**Drag Coefficient ($C_D$):**
$$C_D = \frac{F_D}{0.5 \rho U_\infty^2 A}$$ 

**Lift Coefficient ($C_L$):**
$$C_L = \frac{F_L}{0.5 \rho U_\infty^2 A}$$ 

โดยที่:
- $F_D, F_L$: แรงต้านและแรงยก [N]
- $A$: พื้นที่อ้างอิง (Projected area สำหรับแรงต้าน, Planform area สำหรับแรงยก)
- $U_\infty$: ความเร็วกระแสอิสระ [m/s]

### 1.2 ตัวเลขสเตราฮัล (Strouhal Number, St)
ใช้ระบุความถี่ของการแยกตัวของกระแสวน (Vortex Shedding):
$$St = \frac{f_s D}{U_\infty}$$ 
โดยที่ $f_s$ คือความถี่ของการสลัดกระแสวน และ $D$ คือขนาดลักษณะเฉพาะ

---

## 🏗️ 2. การสร้าง Mesh สำหรับการไหลภายนอก

การจำลองการไหลภายนอกต้องการโดเมนขนาดใหญ่เพื่อป้องกัน "ผลกระทบจากขอบเขต" (Boundary effects):
- **ทางเข้า (Upstream)**: 5-10 เท่าของความยาววัตถุ
- **ทางออก (Downstream)**: 15-20 เท่าของความยาววัตถุ (เพื่อจับภาพ Wake ที่สมบูรณ์)
- **บริเวณใกล้ผิว**: ต้องมีการใส่ **Prism Layers** เพื่อจับชั้น Boundary Layer ($y^+ \approx 1$ หรือ $30-300$ ตามโมเดลที่เลือก)

## 💻 3. การนำไปใช้ใน OpenFOAM

เราใช้ Function Object `forces` ในไฟล์ `system/controlDict` เพื่อดึงค่าสัมประสิทธิ์ออกมาโดยอัตโนมัติ:

```cpp
functions
{
    forcesCoeffs
    {
        type            forces;
        libs            (fieldFunctionObjects);
        patches         (car_body);
        rho             rhoInf;
        rhoInf          1.225;
        CofR            (0 0 0);    // จุดศูนย์กลางการหมุน
        dragDir         (1 0 0);    // ทิศทางแรงต้าน (แกน X)
        liftDir         (0 0 1);    // ทิศทางแรงยก (แกน Z)
        magUInf         20.0;       // ความเร็วอ้างอิง
        lRef            4.5;        // ความยาวอ้างอิง
        Aref            2.2;        // พื้นที่อ้างอิง
        writeControl    timeStep;
        writeInterval   1;
    }
}
```

## 🌪️ 4. การเลือกแบบจำลองความปั่นป่วน

- **Steady-state (simpleFoam)**: ใช้ **k-ω SST** เป็นมาตรฐานสำหรับอากาศพลศาสตร์ยานยนต์
- **Transient (pimpleFoam)**: ใช้ **DES (Detached Eddy Simulation)** สำหรับปัญหาที่มีการแยกตัวของการไหลที่ซับซ้อนและโครงสร้าง Wake ที่ไม่คงที่
