# แบบจำลอง RANS (Reynolds-Averaged Navier-Stokes)

แบบจำลอง RANS เป็นมาตรฐานในการจำลองการไหลระดับอุตสาหกรรมเนื่องจากความสมดุลระหว่างความแม่นยำและต้นทุนการคำนวณ ใน OpenFOAM แบบจำลองเหล่านี้ถูกแบ่งตามจำนวนสมการขนส่งเพิ่มเติมที่ใช้

--- 

## 🌪️ 1. แบบจำลอง k-ε (Two-Equation Model)

เป็นแบบจำลองที่นิยมที่สุด ใช้สมการขนส่งสองสมการสำหรับ $k$ และ $\epsilon$:

### 1.1 สมการขนส่ง (Transport Equations)
- **Turbulent Kinetic Energy ($k$):** วัดพลังงานในความปั่นป่วน
- **Dissipation Rate ($\epsilon$):** วัดอัตราการสลายตัวของพลังงานเป็นความร้อน

$$\nu_t = C_\mu \frac{k^2}{\epsilon} \tag{1.1}$$ 

### 1.2 ข้อดีและข้อเสีย
- ✅ **ข้อดี**: เสถียรสูง, ให้ผลดีสำหรับการไหลในที่โล่ง (Free-shear flows)
- ❌ **ข้อเสีย**: ทำนายการแยกตัวของไหล (Separation) ได้ไม่ดี, ไม่แม่นยำในบริเวณที่มีแรงดันไล่ระดับสูง

--- 

## 🏔️ 2. แบบจำลอง k-ω SST (Shear Stress Transport)

พัฒนาโดย Menter (1994) เพื่อเชื่อมโยงข้อดีของ k-ω (แม่นยำใกล้ผนัง) และ k-ε (เสถียรในกระแสอิสระ)

### 2.1 กลไก Blending Function
SST model ใช้ฟังก์ชัน $F_1$ เพื่อสลับการทำงาน:
- **ใกล้ผนัง**: ทำงานเหมือน k-ω
- **ห่างจากผนัง**: สลับไปทำงานเหมือน k-ε

### 2.2 จุดเด่นทางเทคนิค
มีการจำกัดความหนืดปั่นป่วนเพื่อป้องกันการทำนายที่สูงเกินไปในบริเวณที่มีการแยกตัว ทำให้เป็นที่นิยมที่สุดสำหรับงานอากาศพลศาสตร์ (Aerodynamics)

## 🏛️ 3. สถาปัตยกรรมคลาสใน OpenFOAM

OpenFOAM ใช้ Template Metaprogramming เพื่อจัดการโมเดลความปั่นป่วน:

```cpp
// ลำดับชั้นการสืบทอด
turbulenceModel
    └── RASModel
        └── eddyViscosity
            ├── kEpsilon
            └── kOmegaSST
```

### ตัวอย่างการคำนวณเทอมการผลิต (Production Term G) ใน `kEpsilon.C`:
```cpp
volScalarField::Internal G
(
    this->GName(),
    nut()*(dev(twoSymm(tgradU().v())) && tgradU().v())
);
```

## 🚀 5. การระบุค่าที่ Inlet (Turbulence Specification)

เพื่อให้ผลการจำลองแม่นยำ ต้องกำหนดค่า $k$ และ $\epsilon$ หรือ $\omega$ ที่ Inlet อย่างเหมาะสม:

- **Intensity ($I$):** สัดส่วนความเร็วผันผวน ($1\%$ สำหรับ low turbulence, $5-10\%$ สำหรับ high)
- **Length Scale ($L$):** ขนาดลักษณะเฉพาะ (เช่น $0.07 \times D$ สำหรับท่อ)

**สูตรคำนวณ:**
$$k = \frac{3}{2} (U I)^2$$
$$\epsilon = \frac{C_\mu^{3/4} k^{3/2}}{L}$$
